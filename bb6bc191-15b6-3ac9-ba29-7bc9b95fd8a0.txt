
<br/>　　<br/>　　    抚仙湖的水底仿佛另外一个世界，颜色只有一种，四周一片寂静，到处都充满了黑暗，人类可以征服最高的山峰，但却未必能征服最深的水域，哪怕就连一片湖泊想要探寻，也不是那么容易的。<br/>　　<br/>　　    下潜的十几道人影，逐渐接近了水下古城，从视频的影像中已经隐约能看见一座城市的轮廓了，电脑前的人都绷住了呼吸。<br/>　　<br/>　　    六个考古队的人，在临近古城的时候开始各自分开，拉开距离，四个蛙人间隔着穿插在他们中间，谨慎的盯着几道身影，他们不负责发掘工作，只需要保证这些人在出现意外的时候，迅速接近然后施以救援，力求不出现任何事故。<br/>　　<br/>　　    开始的时候进展的都很顺利，六个人各自负责自己的区域，方位，对古城进行测绘，他们的主要工作就是绘制出地图一类的图景，用于以后将整座古城都体现在三维图像上。<br/>　　<br/>　　    时间一点一滴的过去，此时距离他们下水已经有一个小时的时间了，氧气瓶的含氧量时间，差不多是在两个小时左右，当探测工作进行到一小时四十分左右时，就该开始往水面返回了，避免出现来不及回返然后缺氧的现象。<br/>　　<br/>　　    电脑前面原本绷着一口气的考古队这时也逐渐的放松了心态，从现在来看一切进行的如此顺利，差不多再有半个小时的时间，水下的人就能上来了。<br/>　　<br/>　　    李菲儿扭头，眼神穿过人群落在了靠在栏杆上面无表情的王惊蛰身上，对方似乎感觉有人在看自己，就跟她一双眼睛接触了下，四目相对中王惊蛰伸手指了指抚仙湖的水面，意味深长的摇了摇头。<br/>　　<br/>　　    李菲儿略一皱眉，然后回过脑袋，看着视频影像。<br/>　　<br/>　　    古城东南角落边缘，有道身影扶着一面城墙的中间，另外一只手拿着摄影机，正在缓慢的向前游着记录画面，当他游了能有二十多米的距离后，此时他头顶有一处类似于岗楼的地方，水流明显发生了变化，一道漩涡从下往上开始盘旋着升了上去。<br/>　　<br/>　　    这就是水下的复杂状况了，你根本意料不到会发生什么事，打个比方，原本水下是很平静的，就像是个静止的世界，流水缓慢仿佛一动都没动，但这时当有一个体积稍大一点的物体忽然来回游动的时候，就有可能对水流产生变化，这个水流的变化是可以被成倍放大的，就像平静的水面你扔下一颗石子，带起的水波会一圈一圈的往外扩散，距离可能会达到几十米，而在水下水流动的变化更是如此，甚至要是碰上暗流的话，那状况就更危险了。<br/>　　<br/>　　    当这道身影从城墙下游过的时候，一道暗流逐渐升起，带起一道漩涡，城墙上的岗楼顶，有一块城砖抖动了两下，随即晃了晃，就从顶部跌落，直接奔着下方的人落去。<br/>　　<br/>　　    由于此人手里的强光手电还有头顶的视频摄像头是朝着前方的，不光是他自己，电脑前面的人谁也没有看见这块石头从上方跌落，一直到这块石头无声无息的落在他的头顶，画面剧烈的抖动了几下后，电脑前的人才看见画面忽然翻转了过来，正在朝着水底落去。<br/>　　<br/>　　    顿时，船上的人全都惊了，明显没有想到会出现这种变故，李菲儿的反应还算是快的，她连忙瞄向另外一处画面，此时其他的几个考古队员还有四个蛙人暂时谁也没有看见这个意外，船上的人当即都懵了。<br/>　　<br/>　　    画面静止了，那个叫秋雨的人仰面躺在湖底的沙石上，谁也不知道那一块突然掉落的砖头砸在了他的头部，将他的脑袋都给划出血了，这块用来搭砌城墙的砖，如果拿到岸上来的话，重量差不多有将近五十斤左右了，从城墙岗楼上掉下来的距离差不多有十来米左右，要不是有水承担了大部分的力道，就这个距离和重量都有可能将人当场给砸死了。<br/>　　<br/>　　    船上的人都知道出现意外了，但他们再急也没有用，因为你根本联系不上，只能期望着有人能够发现这场突如其来的变故了。<br/>　　<br/>　　    “怎么回事，秋雨怎么突然就倒了？”文峰惊愕不解的问道。<br/>　　<br/>　　    李菲儿摇头说道：“不清楚，估计是出现了什么意外，他没发现……”<br/>　　<br/>　　    黄詹“嚯”的一下就站了起来，跟黄九郎说道：“黄主任，马上派人下去救援”<br/>　　<br/>　　    黄九郎当即朝着几个早就准备接应的蛙人打了个手势，随即两人带上阳气面罩就跃入了湖中，急速朝着水下潜去，然后黄九郎皱眉看了眼王惊蛰，他耸了耸肩膀，那意思是不好意思，我已经提醒过了。<br/>　　<br/>　　    当这人脑袋上的视频画面静止了将近两分钟左右，突然间有一道人影就从上方快速的接近了过来，那是一个离他最近的蛙人在发现这边城墙下的人影不见了时就赶紧朝着这边游了过来，这得幸亏他们下水之前李菲儿慎重的提醒了一句，所以这才有个蛙人离秋雨比较近，也时刻关注着这边，要不然在耽搁一会的话，人肯定没救了。<br/>　　<br/>　　    这名蛙人落入湖底，将秋雨扶了起来，发现人已经昏迷了，但幸好他嘴里还咬着氧气嘴能够自主呼吸，蛙人将秋雨扶起，一手穿插在对方的胳膊下，双脚连续快速摆动，奋力的朝着水面升去，同时他们这边一出现意外的动作，另外一个蛙人就赶紧迎了过来，跟他从一左一右就夹着人加速上升。<br/>　　<br/>　　    当视频画面一直朝着水上升去的时候，船上的人揪着的心才略微放下了一点，如果这人运气好的话，还是来得及救治的。<br/>　　<br/>　　    李菲儿“唰”的一下转过头，神色复杂的看着王惊蛰，对方眯了眯眼睛，淡淡的靠在了栏杆上。<br/>　　<br/>　　    片刻后，水面涌动，几道人影同时钻了出来，四个蛙人将昏迷了的秋雨送上了船，然后随行的医护人员赶紧检查，人此时是昏迷的，呼吸微弱，脸色发青，明显状况堪忧。
