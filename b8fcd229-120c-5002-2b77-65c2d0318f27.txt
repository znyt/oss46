　　边关，一直是异域与本土修士之间的战场，当年异域便是从此处进攻星空的，同时，此处也是王者试炼的终极关，亦是宇宙界壁所在之地。
<br/>　　这日，元始一身红衣的来到边关之上，看着这堆积慢尸骨的边关，颇有些感慨，战争永远是最残酷的，真不知道洪荒是否也会有发动界域战争的一天，若真是有那么一天，他又该如何？
<br/>　　他一步步地向前走去，全然不顾其他人的目光。
<br/>　　“看，那人想要干什么？”远处，有人发现元始，惊呼道。
<br/>　　“什么？”
<br/>　　“快看，那人不见了。”一人激动道。
<br/>　　顺着那人指的方向看去，只看到一道巨大的裂缝正在缓缓愈合着，周边无尽混沌之气涌动过来，将四周破坏，若不是此地有大阵守护，恐怕就要遭殃了。
<br/>　　“刚才那人竟然徒手撕开界壁！我没看错吧？”男子声音中带着不可思议的表情。
<br/>　　“不错，的确是徒手，想不到竟然看见了至尊强者？”
<br/>　　“……”
<br/>　　已经消失的元始却并没有管这件事，他来到混沌之中，已经熟悉了不能再熟悉了，混沌空间还是一样的单调枯燥，无边无际。元始在混沌之中，看到一条巨大的道路通向远方。他立刻便知道这是大能布置的道路，能够让那些境界低下的修士进入混沌。
<br/>　　混沌空间并不是什么人都能进的，若是修为低，挡不住混沌的侵蚀，便会身死道消，这道路便可以解决这样的问题。异域为了进攻星空，还真是花费不少心思。
<br/>　　元始随着道路飞了过去，不久便看到另一个宇宙的界壁了。
<br/>　　“什么人？”
<br/>　　就在此时，一声大喝响起，伴随着一股强大的至尊威压向着元始压迫过来。元始寻声看去，却看到一个头生双角，背负双翼的大汉正在看守界壁。在他的身后，还有数十人，修为都有王者级别，此刻正紧紧地盯着元始。
<br/>　　“空间禁锢。”
<br/>　　元始没有废话，直接将周边空间禁锢，元气化作大手将为首那个至尊抓在手里，其他人顺势一掌拍碎，对于异域强者，元始没有兴趣。
<br/>　　“搜魂！”
<br/>　　问也不问，元始直接开始强制的搜魂，他想要了解异域的情况，这人身为至尊，应该知道的有很多吧。
<br/>　　“啊——”
<br/>　　那人惨叫，被直接搜魂，就如同是被撕裂神魂，让他痛不欲生。良久，元始一掌拍死这人，他已经给到了自己想要的消息。
<br/>　　他直接撕开界壁，进入异域之中。这异域之人所处的世界，与之前的世界并不相同，这个世界整个宇宙只有一块生命大陆，像是一片独立的世界一般，不似之前那般星空中有无数生命星球，而且这个宇宙也比之前的宇宙要小无数倍。
<br/>　　怪不得要发动界域战争，相比此处，星空简直就像是一块大蛋糕，无数的生存之地，无尽资源，想不让人疯狂都不行。
<br/>　　元始懒得仔细查看，他直接将自己强大的神念肆无忌惮的扫了出去，瞬间便扫视了整片大陆，整个大陆的却情况都出现在他的“视野”中。
<br/>　　“什么人，竟敢如此放肆！”
<br/>　　“大胆！”
<br/>　　“……”
<br/>　　数个地方都升起一股股强烈无比的至尊威压，他们感受到元始那肆无忌惮的扫视，顿时大怒，想要出来阻止。
<br/>　　“死！”
<br/>　　元始淡淡吐出一个字，伸出一个指头，缓缓点向一个至尊。
<br/>　　“仙，怎么会？这个世界上怎么还会有仙？”那人惊骇道，他在那指头面前，连抵挡的勇气都升不起来，实在是那气息太伟岸太浩瀚了。
<br/>　　杀了那人，元始不停，继续弹出数道混沌剑气，向着剩下的几位至尊灭杀过去。
<br/>　　“你真让本座意外啊，没想到此界竟然还有你这样的高手。”
<br/>　　就在这时候，这片宇宙中，突然升起一股超出至尊的强大气息，将元始随手打出的混沌剑气毁灭，同时，一道身影从虚空之中走出，这是一个中年男子，一身赤色锦袍，满头赤发，头生独角，背后更有一对燃烧着熊熊烈火的真凤火羽。
<br/>　　“你的出现，让本座很感兴趣，我有一样东西在你身上。”
<br/>　　男子眯着眼睛，看向元始，淡淡道。他虽然心中疑惑自己看不透这人的修为，但他自信自己不会输。心中只是以为是这人有什么秘法或者宝贝，遮掩了气息。他也不会在意，连道祖都被他杀了，更何况其他人了。
<br/>　　“你说的是它吗？”元始眼神平淡，慢慢从袖中取出一口青铜棺。
<br/>　　“往生棺，想不到我当年留下的东西，被你拿走了。”中年男子开口道。
<br/>　　“往生棺吗？”元始低语一句，看着眼前的男子，淡淡开口道：“真主，吾问你一个问题，你知道此界为何不能有仙，渡劫时会出现斩仙台？”
<br/>　　对于对方能够喊出自己的名字，真主一点也不觉得意外，反倒是对方问的问题，让他双眼一缩，仿佛是才想到了什么，他开口冷笑道：“没有仙，那本座是什么？至于之前为何无仙，那是因为他们死了。”
<br/>　　“死在斩仙台上？”
<br/>　　“不错，都死在了斩仙台上，至于成仙时为何会出现斩仙台，那是因为天要猎食了。”真主相信自己的实力，所以对这些也没有隐瞒，说了出来。
<br/>　　“猎食？仙吗？”元始若有所思。
<br/>　　“不只是仙，有时是至尊。”
<br/>　　“原来如此。”元始仿佛是明白了什么似的。
<br/>　　“小子，哪有那么多问题，还是给本座乖乖死来！”说着，真主伸出大手向着元始狠狠抓去，大手之中带着剧烈无比的力量，至尊刚刚被触碰到，便炸成了血雾，空间都有些扭曲。显然，这位真主也不是省油的灯，一出手就是全力，非死即伤。
<br/>　　“听说此棺叫往生棺，那我便用此棺送你去往生吧。”元始脸色淡漠，把玩着手中的青铜棺，淡淡道。
<br/>　　“去！”
<br/>　　元始屈指一弹，手中的青铜棺快速旋转着飞了出去，同时也在快速变大着，瞬间便击破真主的大手印，向着真主镇压过去。
<br/>　　“哼，还有两下子，当年我连道祖都能杀，就凭你还想送我去往生？”真主不屑一笑道，接着背后真凤火羽张开，遮天蔽日，向着元始杀去。
<br/>　　元始笑了笑，淡淡道：“哦？你确定你杀了道祖吗？”
<br/>　　看着真主杀过来，元始不急不忙做出反应，真主的修为在大罗圆满，但毕竟没有突破准圣，而元始早已是准圣圆满了，对付真主，元始随手可以镇压，但他并没有那样做。
<br/>　　“道祖当然被我杀了。”真主不明所以，他也不去解释，挥动手中长枪，杀向元始。
<br/>　　“真主，别来无恙！”正在此时，一道突兀的声音响起。
<br/>　　“道祖！”真主脸色大变，不过还不等他看到对方，一只巨大的手掌便从他头顶压了下来，这只手看上去很普通，但却带着无穷威能，直接击在真主背后。
<br/>　　“噗——”
<br/>　　真主整个后背都坍塌下去了，他回头，就见一个身穿道袍，满头白发的老者从虚空中走了出来。
<br/>　　“你没死！”真主脸色难看，脸上惊疑不定。
<br/>　　“你都没死，贫道又怎会死去？”道祖开口，面色淡然道。
<br/>　　“怎么可能，你竟然没死？你竟然能从那里逃出来，不可能！”真主脸上满是不可置信。
<br/>　　“往生棺正好送你往生，你废话太多了。”正在此时，一道巨大青铜棺从天而降，将真主的身体碾压，同时棺口打开，一道青光闪烁，将他的身体拖入里面，棺盖自动合上。
<br/>　　“啊！怎么可能？”
<br/>　　真主大叫，到现在都不敢相信，自己最后会死在自己的武器上，可惜，最后连元神都被磨灭了。
<br/>　　“见过道友，不知道友道号？”一旁道祖深深看了眼前这人一眼，只觉得他深不可测，见其收起往生棺，便开口问道。
<br/>　　“本座元始，道祖之名，早已如雷贯耳，没曾想今日得见真人，幸会！”
<br/>　　“……”
<br/>(本章完)
