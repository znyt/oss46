公司高层是站在一边恭敬的目送陆寒霆离开的，夏夕绾突然杀了出来，往商务豪车跑去，他们迅速阻止道，“这位小姐，请你留步！”
<br/>
<br/>        现在的女孩子真是不像话，竟然公然往自家总裁身上扑了。
<br/>
<br/>        但是他们没能拦住，夏夕绾已经跑了过去。
<br/>
<br/>        夏夕绾跑到车边，澄亮的翦瞳落在商务豪车的车窗上，车窗并没有滑下，昂贵的黑色车膜让她看不到里面，她伸出小手拍了拍车窗，“陆总，能不能耽误你几分钟，我有一件事想要跟你说。”
<br/>
<br/>        驾驶座上的崇文透过后视镜看了一眼自家少主，女孩出现的时候，自家少主的目光已经看了过去。
<br/>
<br/>        内心小激动的崇文现在好想召唤尚武八卦啊武，快看，有故事。
<br/>
<br/>        陆寒霆抿了一下薄唇，他看着窗外突然跑过来的女孩，她来拦他的车！
<br/>
<br/>        她拦他车干什么？
<br/>
<br/>        她不是应该跟那个篮球队队长王聪在一起吗？
<br/>
<br/>        “这位小姐，请你立刻离开，我们总裁不是你想见就能见到的，请你先预约，然后等通知。”一个高层走了过来，伸手就去抓夏夕绾的纤臂。
<br/>
<br/>        但是手刚伸出去，那个高层就觉得有一道鹰隼般锋冷的目光盯在了他的手上，好像要将他的手给盯出一个洞来了。
<br/>
<br/>        那个高层手一僵，这时后座的车窗缓缓滑了下来，陆寒霆那张精致英俊的面容闯入了视线里。
<br/>
<br/>        陆寒霆深邃的狭眸淡淡的扫了一下高层僵住的手，然后嗓音低沉磁性的开腔道，“下去吧。”
<br/>
<br/>        那个高层一头的汗，迅速下去了。
<br/>
<br/>        陆寒霆的目光这才落在夏夕绾巴掌大的绝丽小脸上，清冷矜贵道，“你不是在里面看篮球赛么，找我干什么？”
<br/>
<br/>        “陆总，有一件事我需要你的帮忙。”夏夕绾道。
<br/>
<br/>        找他帮忙？
<br/>
<br/>        陆寒霆两条大长腿优雅的叠加在了一起，薄唇勾出一道弧线，看着心情很不错，“说说看。”
<br/>
<br/>        “刚才中场休息，我们A大的王聪队长被人给打了，我现在需要调出监控，查出那个黑手是谁，陆总能不能把那段监控交给我？”
<br/>
<br/>        陆寒霆刚才那点好心情荡然无存，深邃的狭眸倏然眯了起来，变得森寒冷冽，“你找我就为了这件事，为了那个王聪？”
<br/>
<br/>        “陆总，王聪队长真的被打了，右臂还骨折了，不能参加下半场的篮球赛了，我需要…”
<br/>
<br/>        陆寒霆根本就不想听她说话，他还以为她拦他车，找他帮什么忙，她竟然为了那个男生！
<br/>
<br/>        “开车！”他从薄唇里溢出两个字眼。
<br/>
<br/>        崇文本来还在看戏，突然接受到指令，他只能踩下油门，开动车。
<br/>
<br/>        他怎么走了？
<br/>
<br/>        “陆总！陆寒霆！”
<br/>
<br/>        夏夕绾追着商务豪车跑了几步，但是她脚一崴，直接摔在了地上。
<br/>
<br/>        这已经是她第二次追他车摔在地上了，夏夕绾羽捷轻颤，白皙的眼眶变得红红的。
<br/>
<br/>        这时耳畔突然响起了刺耳的刹车声，夏夕绾抬眸，只见前方的莱斯劳斯商务豪车已经停了下来。
<br/>
<br/>        她以为他走了。
<br/>
<br/>        但是，他停了下来！
