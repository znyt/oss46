　　陨星山脉内的矿脉，除了大型矿脉受三大家族控制外，其他小型矿脉都是无主之物，谁都能去争抢。
<br/>　　所以哪怕慕容家的子弟抢了八座小型矿脉，萧家的长辈也不好说什么，只能怪自家子弟技不如人。
<br/>　　萧家这次出来历练的子弟，全都在凡境四重以上，人数大概在百人左右，分别守护在十座小型矿脉中。
<br/>　　开采出的紫云铁，一部分上交家族，一部分归自己所有。
<br/>　　当然了，现在只剩下两座了，其余八座被慕容家抢了去。
<br/>　　而且，这两座，分别由萧壮和萧子木以及其他子弟守护。其余八座原先的萧家子弟，早就被打成重伤，到了萧家的大型矿脉中疗伤了。
<br/>　　萧逸带着还有战力的二十个子弟，快速赶往原本属于他们的八座矿脉中。
<br/>　　“慕容家的混蛋，滚出来。”
<br/>　　萧壮迫不及待地大骂着。
<br/>　　小型矿脉内，慕容家子弟正奋力开采着紫云铁，听到一声大骂，顿时个个大怒，趾高气扬地走了出来。
<br/>　　“哟，原来是萧家的杂碎，竟然还敢回来，找死不成？”
<br/>　　“咦，那不是萧逸那个小废物吗，他怎么来陨星山脉了？”
<br/>　　慕容家子弟语气中夹杂着不屑与嘲讽。
<br/>　　萧逸脸色一冷，瞬间出手。
<br/>　　嘭、嘭、嘭、、、
<br/>　　一阵阵嘭声出现，十多个慕容家子弟顷刻间被吐血击飞，直接晕厥。
<br/>　　“直接绑起来带走。”萧逸喝了一声。
<br/>　　萧家子弟顿时个个摩拳擦掌，他们已经准备好报复慕容家的子弟。
<br/>　　慕容家的子弟也并不是太多，所以分部到各个小型矿脉后，也是一座矿脉十人左右守护罢了。
<br/>　　大半天之后，萧逸带着萧壮等人夺回了原本的八座小型矿脉，俘虏了八十多个慕容家子弟。
<br/>　　八十多人，全都被捆绑起来，场面还是比较壮观的。
<br/>　　这应该是慕容家子弟有史以来最丢脸的一次了。
<br/>　　“萧家的杂碎，识相地快放了我们，否则，有你们好瞧的。”几个嘴硬的慕容家子弟愤愤地骂着。
<br/>　　萧壮等人抡起手掌就是一巴掌过去，直扇得他们鼻青脸肿。
<br/>　　想起其他受伤的萧家子弟，萧壮等人丝毫没有手软。
<br/>　　一旁的萧逸则在皱眉，因为，这些慕容家子弟中，竟然不见慕容轩等人，还有慕容千军也不在，估计是在慕容家的矿脉中吧。
<br/>　　“走吧，去慕容家的矿脉。”萧逸说了一声。
<br/>　　萧壮等人点点头，一脚踹向慕容家子弟，怒喝道，“听不到我们少家主的话吗，都给老子站起来，否则，鞭子侍候。”
<br/>　　萧家子弟完全是赶牛般，赶着慕容家的人走。
<br/>　　.....
<br/>　　大半个时辰后，萧逸等人来到慕容家的矿脉。
<br/>　　果然，之前逃跑的慕容轩等人在这里。
<br/>　　慕容轩看着眼前浩浩荡荡的人群，被捆绑起来的同族子弟，顿时倒吸了一口凉气。
<br/>　　“萧、萧逸，你干什么，快放了我的族人。”慕容轩怒喝道。
<br/>　　萧逸懒得跟他废话，吩咐萧壮道，“把他们也绑了。”
<br/>　　慕容轩等人之前早就被萧逸打伤了，如今根本不是萧壮他们的对手。
<br/>　　“走，去下一个矿脉。”
<br/>　　.....
<br/>　　到了慕容家的第二个小型矿脉，毫无疑问，萧逸出手，将守护矿脉的十多个慕容家子弟全部干翻，而后萧壮把他们绑了起来。
<br/>　　看着这浩浩荡荡的百多个慕容家子弟，萧逸皱起了眉头，走向慕容轩，冷声问道，“你们慕容家到底有多少子弟在陨星山脉？”
<br/>　　“哼。”慕容轩冷眼瞥了萧逸一眼，不屑地转过了头。
<br/>　　‘啪’的一声，萧壮一巴掌抽了过去，“聋子吗？没听到我们少家主在问你话？”
<br/>　　“混蛋。”慕容轩感受到脸颊上那火辣辣的疼痛，顿时大骂一声。
<br/>　　他慕容轩在紫云城内也是颇有名气，如今竟被当众掌掴，他不愤怒才怪。
<br/>　　“萧壮，你给我等着，这一巴掌，他日我慕容轩必当十倍奉还。”慕容轩恶毒地看着萧壮。
<br/>　　‘啪’，又是一巴掌抽过去。
<br/>　　但这次出手的是萧逸，而且出手更重。
<br/>　　慕容轩原本也算颇为帅气，油头满面的，现在帅气的脸庞肿得跟个猪头似的。
<br/>　　“你...”慕容轩顿时双目冒火，“萧逸，不要欺人太甚。对慕容娇儿小姐做了那等过分下流的事情，如今还武力相对，试图报复我们慕容家的子弟。”
<br/>　　“哼，等我们解开绳子后，这百多个慕容家子弟，必将你挫骨扬灰。”
<br/>　　“对...杀了萧逸。”
<br/>　　“将那小废物挫骨扬灰。”
<br/>　　慕容轩真的恶毒，竟用言语激起了其他慕容家子弟的怒气，导致全场百多人纷纷对萧逸大骂了起来。
<br/>　　“呵。”对此，萧逸充耳不闻，淡淡地笑了一声，“将我挫骨扬灰？明明是你们散播谣言，陷我不义，还恶人先告状，也罢。”
<br/>　　“老子现在就跟你们明说了，慕容娇儿就是老子玷污的，老子现在就是要废了你们，你们能奈我何？”
<br/>　　萧逸借着真气，将声音扩大了数十倍，响遍了全场。
<br/>　　特别是那句‘你们能奈我何？’更是犹如一记响亮的巴掌，让慕容家子弟无话可说。
<br/>　　“哼，反正带着你们一百多号人也是累赘。萧壮，不听话的刚脆就地给他埋了。”萧逸的语气中毫不掩饰杀意。
<br/>　　“好。”萧壮也不是个怕事的人。
<br/>　　“你...你们...胆敢杀我们？”慕容轩瞪大了眼睛，仿佛不敢相信。
<br/>　　“你看我敢是不敢。”萧逸淡淡一笑。
<br/>　　那淡然的笑容很平常，但看在慕容轩等人眼里，却比恶魔的笑意更加狰狞，更加冰冷阴森。
<br/>　　“不...不要...”慕容轩当即慌了。
<br/>　　“我再问你一次，你们慕容家到底有多少子弟在陨星山脉。”萧逸冷声问道。
<br/>　　“也就一百来人，跟你们萧家差不多。”慕容轩这次立马老实回答。
<br/>　　“嗯？”萧逸皱起了眉头，道，“也就是说，你们慕容家现在所有子弟都被我捆了？”
<br/>　　“额，是得。”慕容轩虽然觉得羞辱，自己家族所有子弟都被绑了，但还是老实回答。
<br/>　　“那慕容千军呢？”萧逸问道。
<br/>　　萧逸一直在奇怪怎么不见慕容千军。
<br/>　　“额，那个...我也不知道。”慕容轩道。
<br/>　　“还敢说谎，哼，萧壮，埋了他。”萧逸怒喝道。
<br/>　　“好勒。”萧壮恶狠狠地走了过来，拿起铲子，准备挖坑。
<br/>　　“我...我没有说谎阿。”慕容轩顿时急了，“千军大哥...他...他前几天还在这里，现在不知道去哪了，我真的不知道阿。”
<br/>　　萧逸观慕容轩的表情，不似说谎，暗暗疑惑。
