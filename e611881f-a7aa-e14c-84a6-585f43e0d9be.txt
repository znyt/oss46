　　第477章
<br/>　　黑夜里，连只鸟叫都没有，南飞的燕子这个季节还没有北归，春寒入骨。
<br/>　　程大雷一离开大营，就像换了一个人，本来是笨重的蛤蟆，此刻却轻巧的像只麻雀。身无彩凤双飞翼施展开来，很快便消失得无影无踪，至于喜字队定然是跟不上他的步伐。
<br/>　　眼前是一片湖泊，荒草萋萋，倒是没有窥见什么人影。手里一直揣着连击驽，但程大雷对此不报什么希望，他主要的依仗还是腰间的匹夫剑。
<br/>　　前方荒草堆里，一道亮光一闪，这并不是湖水的反光，而是兵器的光芒。
<br/>　　程大雷窝在地上，悄悄举起了手中的连击驽，二十步内，这连击驽可以造成致命伤。程大雷还是第一次实战使用，却也可以试一试效果如何。
<br/>　　他试图瞄准，扣动扳机，嗖地一声，弩箭射出，射在了枯草中。
<br/>　　程大雷撇撇嘴，威力是有的，但自己这准头实在差点。
<br/>　　但这弩箭落地的声音已经惊动了敌人，对方站起身，准备查明情况。紧跟着，程大雷飞身赶到，匹夫剑出鞘，刺入了对方的喉咙。
<br/>　　临死之前，对方没来得及发出任何声音。
<br/>　　程大雷将剑拔出，把剑上的血在对方衣服上擦干净。看了看周围，没有被任何人发现，程大雷又走回原来的位置，将连击驽从地上捡起来，看了看又丢在地上，一脚踩碎了。
<br/>　　这种东西，还真是不适合自己使用呐。自己最擅长的，除了斧头，就是手上这柄匹夫剑。
<br/>　　自己要是白马宝剑走江湖，何尝不能闯出个侠客名声，和那些小姐女侠逗闷子。可此时此刻，自己却为何在这里做这些事情。
<br/>　　他收拾起心情，开始寻找下一个目标，执行下一次击杀行动。
<br/>　　于此同时，喜字队的击杀行动，也在有条不紊的进行着。
<br/>　　五人一组，有人负责寻找目标，有人负责警戒，有人负责击杀。他们都经过训练，都是从江湖上来得，有资格执行这样的行动，本身武艺都是过得去的。
<br/>　　他们配合默契，杀戮干净利落，将敌军的斥候，一个个拔掉。这次行动，他们同样也希望能够有个完美结果，也好在其他队伍前露露脸。
<br/>　　在顺利执行任务时，他们心里也有个担心，就是程大雷现在究竟在什么地方。
<br/>　　因为纪律的原因，他们执行任务时都不能开口说话，不过每个人心里却都悬着一块石头。
<br/>　　在黎明时分，各小组分头撤回，返回了大本营。天就要亮了，夜里悄悄拔掉那么多眼睛，帝国军反应再慢，这个时候也该有个察觉，再不走，怕就走不成了。
<br/>　　每个小组都有收获，所击杀的斥候，他们的腰牌都被带了回来。到时候山寨论功行赏，这些都是凭证。
<br/>　　少的三四枚，多的七八枚，每个小组都算是出色的完成了任务。不过大家的心情都不算太好，关键是大当家现在回来没有，如果大当家出事折在帝国军里，那可是杀多少人都没办法弥补的过失。
<br/>　　心事重重的走进大帐，还没准备好怎么开口，就见程大雷圈着双手坐在大帐里。
<br/>　　“都回来啦？”程大雷道。
<br/>　　“大当家，您，您什么时候回来的？”一人愣愣道。
<br/>　　“喔，天太冷了，我就提前回来了。”程大雷随口道：“你们呢，行动还顺利么？”
<br/>　　“禀报大当家，喜字队五十人，全员安然无恙，共击杀敌人七十三人。”
<br/>　　“蛮好，蛮好。”程大雷把一堆腰牌堆在桌子上：“能够平安回来就好。这算起来有近一百人了。”
<br/>　　诸人眼睛都瞪直了，即便谈得上顺利，想要不知不觉击杀一名斥候都不是一件容易的事情。而程大雷轻描淡写，一个人所杀的斥候，便抵得上三五个小组的击杀数量。
<br/>　　“看什么看，看什么看！大当家当年闯荡江湖的时候，你们还在家里撒尿和泥呢。”阿喜及时道：“有件事你们怕是不知道，当年大当家孤身一人，潜入戎族王城，亲手刺杀了戎族王。现在杀几个斥候，还不是手到擒来。”
<br/>　　“好啦，好啦。”程大雷道：“这种事，完全可以业余再讲么，我们现在聊正事，聊正事。”
<br/>　　阿喜陪着笑，道：“大当家，这些令牌怎么处理？”
<br/>　　“找人给帝国军送去呀，现在，正是打击敌人嚣张气焰的最好时机。”
<br/>　　于此同时，帝国军的大营中，李星已经气得暴跳如雷。
<br/>　　“九十七名斥候，一夜之间被人杀干净，最近的距离大营只有五十步。就算是猪，也该知道哼唧两声，可你们竟然一点反应都没有！”
<br/>　　在李星面前，摆着将近一百具尸体，都是发出去，散落在大营四周的斥候。而就在昨夜，他们全部被杀死了。他们的死法各不相同，但身上都只有一处致命伤，这证明他们杀人的手法十分干净。
<br/>　　李星后怕的是，他们能杀得了这些斥候，难道就杀不了自己么。如果他们昨夜想杀死自己，怕自己的脑袋也已经被割掉了。
<br/>　　军师近前道：“大人，请您先不要动怒，还是身体要紧。”
<br/>　　“呵，身体要紧，我要是被杀死了，也就不用保重什么身体了。”李星的精神濒临崩溃。
<br/>　　“大人，大人……”一手下过来，跪倒在地上：“刚从对面传来的消息。”
<br/>　　“说罢，又有什么坏消息了。”李星觉得自己现在已经百毒不侵。
<br/>　　“那个，对面差人送过来一个盒子，说是从外面见到的，该是我们的，现在要物归原主。”
<br/>　　“盒子里面是什么？”
<br/>　　“手下还没敢打开看。”
<br/>　　“拿过来，给我看看。”李星拂袖在身后，倒是要看看程大雷又耍什么幺蛾子。
<br/>　　手下捧过来一个盒子，在李星面前打开，他身边的军师也探过头去。
<br/>　　一瞬间，李星一张脸由白转黑，由黑转红，哇呀呀爆叫起来。
<br/>　　“程大雷，我不杀你誓不为人！”
<br/>　　盒子内清清楚楚，摆着九十七块腰牌，有些上面还沾着鲜血。
<br/>(本章完)
