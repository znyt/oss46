　　第262章
<br/>　　戎族的王城就立于女儿湖畔，相比长安城，这里简直不值一提。
<br/>　　外城是用黄土夯成，方圆不过十里，毕竟，相比日出而作，日落而息的帝国，草原上的儿女更喜欢逐草而居。
<br/>　　这一日，是戎族的火神节，城内举行了盛大的庆祝活动。大家聚在一起，欢庆这个节日，祈祷来年水草丰茂，牛羊成群。
<br/>　　火神节对于戎族来说是比较重要的一个节日，在这个节日，戎族十四岁以上的男儿可以通过摔跤决出胜负，最后获胜者将得到戎族王赏赐的礼物，以及有权选择部落中任意一人成为他的妻子。
<br/>　　欢宴从三天前就开始了，到今日更加热闹，整座城池完全成了一座不夜城，大家歌舞饮酒，直到天明方休。
<br/>　　呼延跋身为整个戎族的王，自然不能陪着大家一起通宵达旦，不过他也是很晚才回到王宫。
<br/>　　虽然已年近六旬，但呼延跋仍有不错的体力，他胸腔中像是燃烧着一团火，这团火只要不熄灭，他就仍爱世间的好马，好酒，好女人。
<br/>　　一个人的苍老，往往是从失去欲望开始的。
<br/>　　今日喝了许多酒，感觉胸膛中的火更加炙热，他屏退左右，独自走入了寝宫。
<br/>　　呼延跋有多少女人，怕他自己都不太清楚，这些女人中有些是他的表妹，有些是他的堂姐，有些是他表妹的堂姐，有些是他堂姐的表妹……反正这些事，在戎族已经成为习惯，大家也不觉得如何。
<br/>　　目前，呼延跋最喜欢的是自己的小侄女，对方才十六岁。毕竟，对于一个老人来说，青春少女身上的芬芳，那牛奶般白皙的躯体，会让他觉得，自己还没有苍老。
<br/>　　他走入了寝宫，看到一个背影坐在凳子上，身上着着粉色长衫。那柔软的躯体，玲珑的曲线，似乎又在他胸膛加了一把火，令他全身都滚烫起来。
<br/>　　他如在弦的箭，已经不得不发，所谓酒是色媒人，今日的呼延跋已经喝了足够多的酒。
<br/>　　他伸出手，想要去触摸对方的肩背，这个时候，那玲珑『美人』慢慢扭过头来。
<br/>　　我擦！
<br/>　　呼延跋一下睁大眼睛，嘴巴险些咧到耳朵。也不怪他，假若心里期盼着是一个窈窕美人，突然变成一个猛张飞，谁都会像他这样大叫。
<br/>　　我裤子都脱了，你给我看这个。
<br/>　　然后，呼延跋看到了一柄剑，一柄很快的剑。
<br/>　　一点寒芒在他瞳孔中扩大，直接插入了他张大的嘴巴中，临死之前他没来得及任何声音。
<br/>　　程大雷……也就是那窈窕美人，站直了身体，本来就不宽大的衣服被他身躯撑裂。
<br/>　　哎，女装大佬还真不好扮呐！
<br/>　　程大雷从床底下脱出一个被剥光的美人，此时她手脚都被绑着，嘴里塞着袜子，睁大眼睛盯着程大雷。
<br/>　　程大雷在她羊脂玉般的身体上拍了一下，指尖的滑腻，令他心中有几分留恋。
<br/>　　做草原王就是好啊……
<br/>　　程大雷心中感慨一声，七手八脚换上自己的衣服，在女人的衣服上擦掉剑上的血，仔细听了听周围，竟没有任何动静，看来还没有被发现。
<br/>　　事情比自己想得顺利。
<br/>　　程大雷收起剑，悄悄推开窗户，身体如安静的夜莺一般遁入黑暗，然后消失不见。
<br/>　　……
<br/>　　女儿湖畔，夜凉如水。
<br/>　　李婉儿抱膝坐在马车上，目视着面前的湖水发呆。她已这样坐了很久，直到寒露浸湿衣服，仍旧坐在那里一动不动。
<br/>　　事实上，李婉儿并不想去戎族王城，成为帝国和戎族的和平使者。可是，明玉公主却必须这么做，为两族的和平牺牲。
<br/>　　其实从离开长安开始，身体中那个叫李婉儿的身份已经死了，她不能再任性，当然也不会再为她的任性买单。她必须学着像一个真正的公主一样，负担自己身份带来的责任。
<br/>　　一路从长安行来，就是一路杀死自己的过程。她在学着做一些事情，照顾手下人的情绪，漠视生死，习惯牺牲，不管是别人的还是自己的。有些事情看上去有些狼狈、滑稽，那是心中的李婉儿还不认输，还在垂死挣扎。
<br/>　　可她终究会被杀死的，刁蛮，任性，天真消失不见，只有那个叫明玉公主的人活在世上。
<br/>　　本来，她以为自己已经做到了，直到，那个山贼出现在自己面前，给她指了另外一条路。
<br/>　　于是，那个本以为已经死透的李婉儿又复活了，一点点恢复了生气，吵着闹着我不要做。
<br/>　　那究竟该如何做？
<br/>　　她一直在程大雷面前表现得很坚定，我是帝国公主，我需要去牺牲。可越是坚定，心理就越动摇，她需要这个山贼拿出一个理由来说服自己，甚至是渴望。
<br/>　　可这个山贼并没有这么做，他选择不告而别。于是那个叫李婉儿的人需要再被杀死一次。
<br/>　　她心里有些恨程大雷，如果你不能改变什么，又何必要给她希望。
<br/>　　长夜漫漫，从今日起，她必须长大，以明玉公主的身份，去对抗这个世界的黑暗。
<br/>　　黎明东方破晓，她终于在马车上站起身，是时候出发了。
<br/>　　咚咚咚！
<br/>　　忽然的马蹄响声，沿着湖边一个人快马奔来，循声音望过去的时候，李婉儿整个人忽然一愣。
<br/>　　只见马车上的程大雷跑得气喘吁吁，整张脸都已涨红了。
<br/>　　“快走，快走。”程大雷直接跃到马车上：“再不走就来不及了，快快。”
<br/>　　他大呼小叫的模样，搞得李婉儿摸不着头脑：“你搞什么，对了，昨天晚上你干嘛去了，看你的样子，难道把戎族王杀了。”
<br/>　　“咦，你怎么知道的？”程大雷一愣。
<br/>　　“什么，你真把他杀了！”李婉儿惊得跌倒在马车上。
<br/>　　“是呐，你可是不知道我做了多大牺牲，算了，不提这个。”程大雷一边驾着马车一边道：“快走吧，估计现在他的尸体该被发现了，人已经快追出来了，咱们得快点。”
<br/>　　李婉儿一惊，用莫名的眼神盯着程大雷的背影。
<br/>　　“呼延跋一死，戎族必定大乱。”程大雷忽然回过头道：“你不是需要五年和平么，我送你五年和平。”
