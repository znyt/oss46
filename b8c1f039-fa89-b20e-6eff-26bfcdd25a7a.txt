　　第734章
<br/>　　这一场战斗起得急，落下去也快。有些人刚刚睡醒，想要大着胆子瞧一场热闹，发现战斗已经结束了。
<br/>　　如今清风寺已经被团团围住，它恰好位于一座土丘上，向上张望，夜晚的寺庙有种庄严气魄。
<br/>　　魏牧冲乘了一顶软轿，此刻远远望着清风寺的轮廓。作为一个读书人出身的城主，他其实不会骑马，这说来是件奇怪的事，主要是小时候家境贫寒，连像模像样的大牲畜也接触不到。
<br/>　　但大家并不会因为这一点，便小看这位城主。毕竟，敢亲手杀掉自己儿子的，古往今来又有几人。
<br/>　　如今兵临城下，魏牧冲并没有下令进攻。帝国重佛，当初明帝贴身护卫青叶佛便是佛门出身。
<br/>　　冲撞这佛门圣地，能杀了程大雷是确定无疑，可也容易失掉民心。更况且，程大雷显然已经走不脱，无非是多苟延残喘几日罢了。
<br/>　　战报已经报上来，结果令人触目惊心，程大雷从城主府冲出来，大杀四门，虽然没有闯出城门，但五千城防兵也的确没对他形成有效阻拦，反而伤亡上百人。
<br/>　　夜幕中，魏牧冲长长的叹口气，这样的恶贼，一旦放出去那还了得，怕是会成为整个帝国的灾祸。
<br/>　　这更让他确信，今夜自己的行动是正确的，决不能让程大雷活着离开汝南城。
<br/>　　清风寺内。
<br/>　　清风明月，树梢轻轻晃动传来沙沙声，说不出的静逸。当真是化外参禅之处，修身养性所在。
<br/>　　不过程大雷身上的血腥气却破坏了佛门的清净，程大雷和刘发财相貌有几分相似，都是魁梧身材，凶恶面貌。此刻浑身沾满鲜血，宛若那地狱爬出来的恶鬼一般。
<br/>　　围着他们的小沙弥表情明显有些畏惧，但却没人后退，只是表情战战兢兢。
<br/>　　一道声音忽然响起：“休得无礼，主持有请。”
<br/>　　开口的是庙里的香火僧，法名慧禅。崔白玉向他行了一礼，缓缓点了点头。
<br/>　　魏牧冲一定没有做充分的调查，否则此刻定会不顾一切冲过来。清风寺在汝南城香火很望，于是修道气派庄严。但当初，汝南城是崔家的汝南城，肯在这里大撒香火钱的施主便是崔家的人。甚至，当时这座庙能建起来，完全就是因为崔家。
<br/>　　百足之虫死而不僵，一个盘根错节三百余年的家族，怎么会是说拔根就能被拔根的。
<br/>　　程大雷洗干净身上血污，换了一套庙里的僧袍，几人由香火僧引着来到主持所局的静室。
<br/>　　清风寺的主持法号了然，是个年逾花甲的老僧。崔白玉走进去后，毕恭毕敬的行了一礼。
<br/>　　程大雷还是很少见到她态度如此端正，也微微躬身，向老僧行了一礼。
<br/>　　其他几人就没有程大雷这样的好脾气。刘发财上下打量，看老僧的脖子那里合适下刀。他也并非是动了杀心，只是刚才杀得手顺，下意识的行为而已。
<br/>　　徐神机更不用提，他眼里从不搁神佛。当需要的时候，他可以是个坚定的信仰者，当不需要的时候，他也可以什么都不信。
<br/>　　了然和尚抬起头，道：“这位便是程施主吧，请坐。”
<br/>　　程大雷在其对面的蒲团上坐下，大咧咧道：“打扰法师清修，还望恕罪。”
<br/>　　“既在红尘中，便称不得清净人，那滚滚红尘，也不是一扇山门挡得住的。”法师缓缓道：“只不过，我这小庙能有一日两日清净，怕也护不了施主几日。”
<br/>　　说着，他面向崔白玉，轻声道：“崔家已经如此，那些与你无关的事，你就不要再想了。”
<br/>　　程大雷本来正寻思着自己的脱身之策，一听这话八卦之魂顿时燃烧起来，这老僧和崔白玉明显认识么。
<br/>　　程大雷并不知道，这老僧本就是崔家的人，算起来是崔白玉的叔叔，崔相的亲兄弟。在上一次家主位置的争夺中，没斗得过老奸巨猾的崔相。他这一支，在整个崔家失势。经历这件事，他倒是大彻大悟，跑到这小庙做了个青灯古卷的高僧。
<br/>　　崔白玉咬住下唇，随之开口：“二叔，是崔家对不起你。”
<br/>　　法师挥挥手，道：“过去的事情已不用提了，如今世上已没有崔猎巢，只有了然和尚。”
<br/>　　程大雷琢磨着这个名字，脑海里也不存他这个人。说来也奇怪，崔相只手遮天这么多年，但他的名字是什么，天下还真是很少有人知道。
<br/>　　崔白玉脸上的表情依旧不太好看，现在总算是遇见一个有血缘关系的崔家人，但对方却完全一副不染红尘的态度。
<br/>　　半晌，了然轻轻叹了口气，道：“昨日因，今日果，今日因，明日果。崔家有此一劫，也是昨日造的虐。这些是崔家该还的，这是我们的债，我们要认账。好在崔家还留下你这一支血脉，也算是法外开恩了。”
<br/>　　说着，他的目光落在程大雷身上：“多谢程施主庇护，为崔家保全一支香火。”
<br/>　　“啊。”程大雷不知道话题为何突然转移到自己身上，话说这种见家长的感觉究竟是怎么会事。他恍然回过神来，忙道：“好说，好说。”
<br/>　　可能是程大雷的表现令了然也有些无语，他沉默片刻才道：“程当家所做的事，老僧也听说过一二，是非功过自有后人评说，但后人又懂什么。程当家所作所为，是值得人佩服的。”
<br/>　　对这样的奉承，程大雷也是来者不拒。他对这和尚也没有多少尊敬，说什么青灯古卷，一心问佛，真如果看破红尘，又怎么会听说自己的事。
<br/>　　估计这和尚年轻时候，也不会是什么老实人。现在装的世外高人一般，年轻时候估计也是鼎鼎有名的纨绔。
<br/>　　“法师，有没有什么办法让我们离开此地？”程大雷开门见山，直接问道。
<br/>　　老僧顿了顿：“有。”
<br/>　　程大雷松了口气，就知道你这和尚不老实，说不定什么时候，还会悄悄去青楼鬼魂。
<br/>　　“路在何方？”程大雷问。
<br/>　　老僧看了看程大雷面前地面：“路在脚下。”
