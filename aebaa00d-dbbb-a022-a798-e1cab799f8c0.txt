　　剑光散去，显现出两道身影，男的神骏，女的空灵清绝，正是离开鬼王宗的慕清霄与陆雪琪。
<br/>　　“佛门有钱，何出此言？”
<br/>　　与慕清霄在一起，陆雪琪也是成长了不少，但年龄终究不大，懂得依旧不错，在她眼中，佛门一心向佛，普渡众生，连吃斋都是素食，怎会有钱？
<br/>　　慕清霄微微摇头，他经历过多个位面，佛门也接触了不少，他们嘴上说普渡众生，但拿着凡人的香火钱，却什么事情也不干，反而蛰伏在山上，小日子过的不要太快活。
<br/>　　当然，这是他所见过的，至于没见过的，有没有真大德和高僧，那就不得而知了，而且，佛门对于普通人而言也是一种寄托。
<br/>　　凡人相信世间有神仙，能够完成他们的愿望，可世间哪里有仙神？
<br/>　　慕清霄不信仰仙神，哪怕有，也只是修为恐怖的修士，仙是自己修的，神是别人封的，他区区元婴修为，可不敢妄自称神。
<br/>　　望着陆雪琪好奇的神色，慕清霄指了指山上金碧辉煌的宫殿。
<br/>　　“每天前来须弥山拜佛的没有一万也有八千，每一个凡人都要给香火钱，其中不乏富贵之人，常年累计下来，任何一个宗门的财富，恐怕都不及佛门九牛一毛，你说他们有没有钱？”
<br/>　　闻言，陆雪琪觉得有些道理，在看看金碧辉煌的宫殿，台阶上走向山顶的凡人，终于明白慕清霄话中的意思。
<br/>　　悠悠钟声，再一次在须弥山上回荡，宣告着新的一天开始。
<br/>　　初升朝阳，从东边天际探出一个小小光晕，将第一缕阳光洒向人间。
<br/>　　清晨山路之上，已经有许多平民百姓沿着山路台阶向那座雄伟的寺庙行去，他们手中多半提着香烛供奉，满面虔诚。
<br/>　　其中有一些人家还带着孩子一起前来朝拜，孩童天真，在这山路上反而并不觉得疲累，许多少年都前后跳跃跑动，一副兴高采烈的模样。
<br/>　　晨雾将散未散，流连在天音寺外，空气中感觉有些潮湿润气，早起的僧人们已经做好了一天之中必要的早课，此刻都在打扫庭院，将昨夜掉落的树叶轻轻扫在一旁。
<br/>　　整座天音寺中，此刻显得肃穆而宁静，沐浴在淡淡的山风里，随风吹过的，还有那若有若无的树叶芳香。
<br/>　　慕清霄目光落在山巅，收回目光后，微笑道：“我们也上去吧。”
<br/>　　说罢，就牵着陆雪琪跨上台阶，一步一台阶，慢慢向半山腰走去。
<br/>　　两人的身体素质不比凡人，几分钟后就来到半山腰上，此处建有寺庙，正是凡人们供奉香火的地方，但此处只是天音寺的外围，而内院非天音寺弟子不得入内。
<br/>　　“两位施主，内部是天音寺重地，请止步。”
<br/>　　慕清霄带着陆雪琪刚想踏入天音寺内院，背后就响起和尚的阻拦声。
<br/>　　慕清霄转头看去，这名和尚面带微笑，年纪不大，只是服饰看似普通，与当初法相和法善所穿大不相同。
<br/>　　“这位道友，我与你们天音寺的法相师兄乃是相识，可否给我禀报一声？”
<br/>　　慕清霄语气和善，小和尚听闻两人与法相相识，顿时受宠若惊，仔细打量两人，顿时觉得两人不凡，想必不会说谎，赶紧道：“两位施主，我只是天音寺外门弟子，你们在此稍等片刻，我这就去给内门师兄禀报。”
<br/>　　“那就麻烦道友了。”
<br/>　　“不麻烦，不麻烦。”说罢，小和尚就快速向内院走去。
<br/>　　片刻后，一名皮肤白净，目光明亮，身穿月白袈裟的年轻和尚快速走了出来，正是空桑山上所遇到的法相。
<br/>　　看到慕清霄和陆雪琪后，法相加快了脚下的速度，口宣佛号，道：“阿弥陀佛，慕师兄陆师姐千里迢迢来到天音寺，是法相失礼了。”
<br/>　　“法相师兄客气了。”慕清霄微微回了一礼。
<br/>　　接下来，两人叙旧一番，谈论起空桑山的事情。
<br/>　　原来，当初在万蝠古窟中，他们与炼血堂余孽激战，最后分开，并没有进入无情海中，至于张小凡他们就不得而知了。
<br/>　　慕清霄只是随便捏造了个理由，只是说他和陆雪琪在另一条通道中遭遇魔教设下的陷阱，待脱困的时候，众人已经离去。
<br/>　　“原来如此，慕师兄和陆师姐脱困后，怎没有回青云门，反而赶来天音寺呢？”
<br/>　　慕清霄也不隐瞒，道：“我是来天音寺，其实是想见一见普泓主持。”
<br/>　　普泓是天音寺方丈主持，也是四大神僧之首。
<br/>　　实力超群，原剧情中，在与兽神及鬼王的战争中都有不俗表现，并因道玄真人与诛仙剑消失而在对鬼王的决战中成为正道领袖，最终归隐世外，交天音寺由弟子法相主持。
<br/>　　“找家师？”
<br/>　　法相眼中有些困惑，显然不明白慕清霄找他师傅有什么事情：“家师一般不见外客，不知慕师兄找家师所为何事，如果是小事就由师弟我代劳。”
<br/>　　闻言，慕清霄也不生气，道：“我找普泓主持，是因为普智大师，以及张小凡的事情。”
<br/>　　话语刚落，法相骤然色变，目光紧紧盯着慕清霄，顿时觉得面前的青年让他有些摸不透了，于是道：“寺内师兄弟都在用早膳，慕师兄不如先去用早膳，我现在就禀报家师。”
<br/>　　“早膳就不必了，法相师兄速去速回吧。”
<br/>　　“也好。”
<br/>　　说罢，法相就快速向山顶奔去，事关重大，他根本无法做主，只是有些好奇，慕清霄是如何得知，普智师兄当年所犯下过错的？
<br/>　　待法相离去，陆雪琪目光在四周看了看，道：“我们就在此处等候吗？”
<br/>　　慕清霄微微一笑，牵着她的手来到一颗梧桐树下的木椅上，道：“事关普智大师，想必法相很快就会带回来好消息，我们稍等片刻就是。”
<br/>　　剧情中，因对师弟普智当年屠草庙村所造杀孽而愧疚，所以普泓对张小凡多加照顾，曾在青云后山救过张小凡的性命。
<br/>　　而后，又于天音寺无字玉璧之前，企图化去张小凡身上的阴晦之气，从而使张小凡学到第四部天书。
<br/>　　普智的事情相对敏感，普泓清楚事情轻重，自然会见他。
