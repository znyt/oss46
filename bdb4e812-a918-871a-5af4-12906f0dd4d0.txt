
<br/>　　<br/>　　    再说墓崖村这边，听了余生婆一席话，又跟李红拂聊了几句，王惊蛰就找了个借口跟她分开了，说是去外面散散步然后独自一人就走了。<br/>　　<br/>　　    李红拂也不是那种腻歪人的小妖精，王惊蛰有意单独行动她也没心思在跟上人家，于是就回到了余生婆那个茅草屋里，她进来的时候余生婆就跟一个农村老妇人一样正在收拾着屋子。<br/>　　<br/>　　    似乎是知道只有李红拂自己进来的，余生婆头也没回的说了一句莫名其妙的话：“你到底还是来了”<br/>　　<br/>　　    李红拂站在她的身后，轻声说道：“看见我的第一眼，你就已经知道我是谁了，对么？”<br/>　　<br/>　　    “历史更替，岁月在变，但血脉是永远都不会改变的，现在叫什么？好像叫基因吧……”余生婆放下手里的活，缓缓的转过身，看着李红拂说道：“你脸上就写着自己是谁呢，我哪里会看不出来？说来我们却是一脉相承的，你是红拂女和李靖的后人，我叫张余生，是虬髯客的后代，一千多年前我们的祖宗叫风尘三侠”<br/>　　<br/>　　    李红拂皱了皱眉，随即舒展开，淡淡的说道：“我还以为你看见我会带着点敌意呢”<br/>　　<br/>　　    余生婆顿时笑了，摇头说道：“哪里有的事，我不是说了么，咱们都是一脉相承的，理应跟老祖宗一样，携手共进才是，你说对么？”<br/>　　<br/>　　    “人都一张嘴，上下两片唇，你怎么说就怎么是吧……”<br/>　　<br/>　　    余生婆这时忽然打开屋子里的一个柜子，翻箱倒柜后从下面拿出三个牌位上来，牌位上赫然写着三个名字，红拂女，李靖和虬髯客。<br/>　　<br/>　　    “逢年过节我都要拿出来供奉祭拜一下的，风尘三侠这个名字可不是被白叫出来的，人死了可人心未必会散啊，李靖和红拂女我不知道，但虬髯客却始终都记挂着他们呢，他临死前就说过，三侠始终都是三侠，尽管他后来不告而别了”<br/>　　<br/>　　    李红拂顿时无言，她看得出来那三个显得很古朴的牌位已经有些年头了，差不多可以久远到一千多年左右了，这也许还真是当年虬髯客跟李靖，红拂女分开后，临死之前立下的。<br/>　　<br/>　　    “你说，那个叫王惊蛰的年轻人，会不会要这几年的阳寿？”余生婆问道。<br/>　　<br/>　　    李红拂当即毫不犹豫的点头说道：“必然会要的，他比任何人都想要，而且他想要的还更多，比如不知几年，如果能借上十年二十年的话，他可能会不惜一切代价的”<br/>　　<br/>　　    余生婆又笑了，笑的时候仿佛脸上的褶子都要开花了，她一连笑了几声，才点头说道：“能要就好，那就好……”<br/>　　<br/>　　    于此同时的王惊蛰就在墓崖村村头的一颗老树下，这是颗老槐树，下面放着几块石头，坐着一些村里的人。<br/>　　<br/>　　    五个村民，一脸呆滞的摊着手掌。<br/>　　<br/>　　    王惊蛰的目光依次从这几人的手上扫过，来回足足看了好几遍，从最开始的皱眉不解，到最后慢慢的把眉头又给展开了，他抿着嘴唇目不转盯，手指微微的颤抖着掐算了许久。<br/>　　<br/>　　    占卜，算命，问卦始终脱离不开三个要素，就是面相，八字还有手相，八字只能预知未来的吉凶，事业，财运或者婚姻，而手相和面相不但能问前路，更主要的是还能卜算出过去，因为一个人的面和手是不会随着任何因素而发生任何改变的。<br/>　　<br/>　　    特别是人手，左为先天右为后天，有句话叫手握乾坤攥八卦，指为长龙掌如虎，一句话就能够概括出人手的格局来了，几乎可以囊获一个人的一生了。<br/>　　<br/>　　    只是这手相中的学问可要比面相上难通透多了，寻常人说给你看个手相什么的，无非就是看看生命线，感情线和事业线，然后在根据两手五指长短来加以推断一下，其实这都是最粗浅的手上工夫，真要是精通看手相的人去看的话，你伸出两只手来，他都能把你这个人看的八九不离十了。<br/>　　<br/>　　    就比如王惊蛰面前的这几个墓崖村的村民，王惊蛰一眼望去，片刻后心里顿时就通透了不少。<br/>　　<br/>　　    半晌后，王惊蛰起身慢条斯理的挪着步子往回走。<br/>　　<br/>　　    此时日头已然西落，再有一会天色可能就要黑了。<br/>　　<br/>　　    王惊蛰再回到余生婆的草屋里时，李红拂正在里面坐着，余生婆在厨房里已经蒸好了一锅馍馍，桌子上还放着一盆菜叶汤。<br/>　　<br/>　　    “村里没什么吃食，只能用粗茶淡饭的来招待你们了，不要见怪”余生婆蒸的馍馍卖相上不怎么样，不过味道闻起来却不错，黄澄澄的看起来挺诱人，<br/>　　<br/>　　    王惊蛰点头说道：“有口吃的就行，我不挑的……”<br/>　　<br/>　　    五六分钟后，三人围坐在桌子旁吃饭，开始吃的时候谁也没有说话，王惊蛰吃了一个馍馍肚子里有点饱感之后，就又拿起一个一边揪着一小块，一边随意的说道：“婆婆这三五年的阳寿，得是个什么价呢？”<br/>　　<br/>　　    正在喝汤的李红拂，手忽然一顿，两三秒钟后，她又面无表情的端着饭碗抿了一小口。<br/>　　<br/>　　    余生婆说道：“我一把年纪了，要什么钱财，哪怕就是座金山和银山给我，也是什么用都没有的，你说对吧？”<br/>　　<br/>　　    王惊蛰“嗯”了一声，说道：“是这个道理，要是我的话也不至于要钱了”<br/>　　<br/>　　    “呵呵，理解就好”余生婆拿起汤勺主动给王惊蛰添了一碗汤，然后说道：“人性都是贪婪的，想要的东西有很多，但除了钱么，剩下的无非也就是宝了”<br/>　　<br/>　　    王惊蛰点头说道：“天才地宝”<br/>　　<br/>　　    “在我这里有个明码标价的规定，那就是阳寿三五年，你可以拿任何有用的东西来换，佛道两门的法器，要成精了的何首乌，人参，太岁，还有那些不传世的珍品都可以，这些东西么驳杂的很，常人肯定弄不到，但有一少部分的人还是不难搞到手的，有人拿过长白山上一支八百年的野山参，也有人拿过唐伯虎的一副真迹，对我来说这些东西其实都一样，只要是罕见的东西就行了……”<br/>　　<br/>　　    王惊蛰挠了挠鼻子，放下手里的汤碗，笑道：“那要是想再多几年的阳寿呢，比如十年，或者更多……”
