        “可否跟你共用一下课本？”郎安对上连星儿那惊怔的目光，面无表情地问道。
<br/>
<br/>        连星儿回过神来后，不禁皱起了眉头，但是还是好心把课本挪到了中间。
<br/>
<br/>        一堂课下来后，课间休息时间，郎安去了一趟卫生间，却在卫生间里被几个男生给围住。
<br/>
<br/>        “明司寒的女人，你也窥觊，你是不是……”其中一个气势汹汹的男生一把揪住了郎安的领口。
<br/>
<br/>        但是，他的话还未说完，便被郎安一拳打趴在地。
<br/>
<br/>        另外三个男人面面相觑后，不约而同地挥起了拳头向郎安冲了过去。
<br/>
<br/>        赵安彤上完洗手间回来后，向连星儿和顾千凝，还有路娅玟她们三个人围了上去。
<br/>
<br/>        “告诉你们一个惊人的消息，刚刚在男厕所里，那个新来的把咱们班的四个男生给打趴了！”赵安彤一惊一乍地说道。
<br/>
<br/>        顾千凝和路娅玟顿时吃惊不已，纷纷询问：
<br/>
<br/>        “是什么原因打架啊？”
<br/>
<br/>        “一来就跟咱们班男生干上了，只怕以后在咱们班难混了。”
<br/>
<br/>        连星儿皱起了眉头，补充了一句：“反正他来我们班，肯定不是为了读书也不是为了交友。”
<br/>
<br/>        “你怎么知道？”赵安彤十分诧异地看着连星儿。
<br/>
<br/>        连星儿刚准备说什么的时候，嘴巴才张开，便看到了站在赵安彤身后的郎安。
<br/>
<br/>        顾千凝和路娅玟看到郎安后，立即闭嘴了。
<br/>
<br/>        赵安彤见她们三突然间不吭声，下意识地转过身去。
<br/>
<br/>        当她看到郎安那双冷冽的黑眸时，浑身冷不丁地打了个寒颤，立即给郎安让道。
<br/>
<br/>        郎安在连星儿的左边坐下来后，双手插在裤袋中，面无表情地看着前方的黑板。
<br/>
<br/>        连星儿叹了口气，从抽屉里拿出一小袋洗好的蓝莓，递到郎安的跟前，问道：“我妈咪种的蓝莓，你要不要尝尝？”
<br/>
<br/>        就在赵安彤、顾千凝和路娅玟以为这个新来的男生会拒绝连星儿的时候，岂料他毫不犹豫地便伸手探入那个袋子中，利索地拿了一颗蓝莓塞入嘴中。
<br/>
<br/>        上午的课程全部结束后，连星儿和她们三一起回寝室的路上，郎安隔着不到三米远的距离，就跟在了她们的身后。
<br/>
<br/>        到了寝室楼下，连星儿让她们三个先上去，自己一转身，便迎上了走到了跟前的郎安。
<br/>
<br/>        “你来我们学校，现在又跟着我，到底是想做什么？”连星儿一脸不悦地问道。
<br/>
<br/>        郎安面无表情地回答道：“你外公让我跟着你的，我也不知道是做什么。”
<br/>
<br/>        好机械的回答，更确切地来说，是敷衍！
<br/>
<br/>        连星儿索性掏出手机给外公连中凯打电话。
<br/>
<br/>        “我的乖孙孙，今儿是想外公了吗？”电话一接通，连中凯便笑呵呵地问。
<br/>
<br/>        “外公，郎安是怎么回事？”连星儿瞥了郎安一眼，纳闷地问道。
<br/>
<br/>        连中凯倒是很淡然地回答道：“你爹地给你派的保镖又不能贴身保护你，所以，外公把郎安派给你当贴身保镖。”
<br/>
<br/>        “我不需要贴身保镖的。”连星儿二话不说，直接拒绝。
<br/>
<br/>        连中凯却突然严肃地劝道：“怎不需要？万一，再发生昨天在街上的那件事情。你妈咪这一辈子都不会再原谅我了！”
<br/>
<br/>        “郎安他……”连星儿欲言又止，再次看向郎安时，心不在焉地应道，“好吧！我知道了！”
<br/>
<br/>        本来她是想说，郎安攻击过她，但这事口说无凭，也就作罢了。
<br/>
<br/>        收了线，连星儿瘪了瘪嘴，向郎安伸出手来：“把你手机给我！”
<br/>
<br/>        郎安微微摇了摇头。
<br/>
<br/>        连星儿不禁没好气地说道：“你不给我手机号码，我们怎么联系？”
<br/>
<br/>        “我没有手机。”郎安淡淡地应了声，顿了一下，又接着说道，“我不需要用手机。”
<br/>
<br/>        “？！”连星儿顿时惊愕了。
<br/>
<br/>        她想了想，还是算了，索性转身回到了寝室。
<br/>
<br/>        连星儿刚进寝室，就被三个室友轮番“审问”。
<br/>
<br/>        因为实在是编不出什么谎言了，连星儿才老实招供说郎安是自己的保镖。
<br/>
<br/>        顾千凝第一个对连星儿的身份产生怀疑：“星儿，你这是既有未婚夫又有保镖的，莫非……你是豪门世家出身？”
<br/>
<br/>        “这还用问吗？我早就发现了！”赵安彤咧嘴笑道。
<br/>
<br/>        路娅玟倒是十分稀奇地揽住连星儿的肩膀，兴奋不已地说道：“星儿，那以后你得照着我们啊！”
<br/>
<br/>        “那还用说吗？我们四个永远都是最好的朋友！”连星儿美滋滋地回答。
<br/>
<br/>        本来确实是四个小团体，现在，连星儿除了在寝室里以外，其余地方无论她走到哪里，郎安都跟在她身边不出五米的距离。
<br/>
<br/>        所以，郎安的事情，很快就传到了明司寒的耳中。
