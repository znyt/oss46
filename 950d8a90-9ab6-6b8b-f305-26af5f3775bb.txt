夏夕绾惊呆了，她知道那个神秘姑娘就是她的妈咪林水瑶！
<br/>
<br/>        可是，她的妈咪不是帝都人吗，夜老说她的妈咪突然降临帝都，那她的妈咪究竟是哪里人？
<br/>
<br/>        妈咪究竟从何处来？
<br/>
<br/>        这个夜老竟然是妈咪的关门大弟子，李长清院士也是妈咪的弟子，按照时间来推算的话，苏希应该是妈咪收的最小的弟子，也是最后一个弟子。
<br/>
<br/>        夏夕绾现在觉得疑云丛生，也许苏希知道些什么，但是自从她来了帝都城，苏希就没有露面过。
<br/>
<br/>        她问过夏邦和蓝烟，他们是江湖中人，人脉非常广的，想打听一个人轻而易举，可是他们都没有听说过苏希这个人。
<br/>
<br/>        苏希说过会在帝都等她的，还带走了妈咪的仆人林婶，但是他已经消失在了帝都城，毫无音讯。
<br/>
<br/>        一切都变得更加扑朔迷离。
<br/>
<br/>        夜老的脸色变得非常的复杂，他喃喃道，“我很敬爱我这个师傅，她医术上的造诣和奇思妙想简直让我视她如神明，但是很快我们就发生了分歧，她说我封建而固执，被权利醺了心，还误入了歧途，走火入魔了，她再三劝诫我，我都没听，后来她说…说要亲手清理门户，所以我们反目为仇了。”
<br/>
<br/>        原来妈咪和夜老之间还有这么一段曲折的故事，虽然夜老说的很简洁，但是夏夕绾还是觉得惊心动魄。
<br/>
<br/>        帝都城的夜家世代从医，每一代的继承人在医术上都有得天独厚的天赋，这也是林水瑶收夜老为关门大弟子的原因。
<br/>
<br/>        但是林水瑶当年没有想到后来夜老慢慢的走错了路，而作为帝都根深蒂固，枝叶繁盛的四大家族之一的夜家也是奋力反扑，当年或许还有别的力量的加入，现在无从得知，总之林水瑶和夜家一战，林水瑶退出了帝都城，而夜家差点覆灭，在这二十年间才慢慢的休养出了生息，这一辈更培养出了善毒的夜无忧。
<br/>
<br/>        这时夜老扭头，看向了夏夕绾，“我知道了，我知道为什么我对你总有一种熟悉感了，因为你的身上好像有她的影子！”
<br/>
<br/>        夏夕绾觉得夜老这就是废话了，她是妈咪的女儿，这母女之间，她身上当然有妈咪的影子。
<br/>
<br/>        不过现在不能暴露身份，夏夕绾不想让自己置身在这些漩涡里，她的首要目的还是尽快拿到妈咪留下来的第二个宝箱。
<br/>
<br/>        还有一件事，非常重要的一件事，她的亲生爹地究竟是谁！
<br/>
<br/>        林婶还有夏老爷子，都是妈咪贴身的人，他们竟然都不知道她的爹地是谁，只能初步断定她是妈咪偷基因生下来的，或许她的爹地也不知道她的存在。
<br/>
<br/>        现在夜老就在她的面前，作为妈咪的关门大弟子，那些年一直陪在妈咪身边的夜老，他是不是知道她的身世？
<br/>
<br/>        夏夕绾一颗心变得激动，雀跃，但是她的表情很平静，让人看不出一点的波澜，“夜老，你说的那个人，我不认识，或许她有什么后人，你可以找她的后人。”
<br/>
<br/>        提到后人，夜老收回了目光，“没错，我师傅当年生了一个女儿，这些年我一直在寻找那个女儿的下落，但是一直没有想到。”
