　　“噗通……”
<br/>　　就在众人惊慌之时，突然有人无声无息的倒下，砸落地面，发出响声。
<br/>　　“噗通……”
<br/>　　“噗！”
<br/>　　“噗嗤！”
<br/>　　“……”
<br/>　　紧接着，倒地声接二连三地响起，顿时惊动了所有人。人们顺着声音望去，只见有好几人都倒在尘埃中，一动不动。
<br/>　　在他们的脸上，充满惊恐的神色，在其额头处有一个拇指般粗细的血洞，鲜血汩汩流出，他们死前似乎是见到了极为惊恐的事情。
<br/>　　“怎么回事？！”众人直感觉脑门凉飕飕的，一股毛骨悚然的感觉直袭心头，内心顿时充满恐惧。四五条鲜活的生命就这样突然地死去，而且还不知道他们的死因，这样给了他们极大的冲击感。
<br/>　　未知的，才是最可怕的！
<br/>　　有的东西，因为别人知道了真相，反而不怕了。当然，也有人无知而无畏。然而，它却不是这种情况。
<br/>　　“不要靠近他们！”夜邪突然阻止了两个刚要接近地面上尸体的几人。
<br/>　　“嗖……当……”
<br/>　　突然，钟声悠扬，声音浩荡庄严，似黄钟大吕在震动。
<br/>　　一道极为璀璨的光芒自王子文身上冲起，他全身笼罩在光辉之中，像是穿上了一层厚厚的黄金战衣。他的那口残破的黄钟在轻轻摇晃，宏大的声音正是从那里发出来的。
<br/>　　“怎么回事？”离他最近的一人紧张问道。
<br/>　　“刚刚好像有什么东西在攻击我……”王子文有些惊恐道。
<br/>　　“你看清是什么吗？”庞博继续问道。
<br/>　　“没有，只觉得有一股巨大的危险笼罩我全身，然后这口黄钟便突然震动起来了。”王子文心有余悸道，他的脸色极为难看。
<br/>　　听闻此话，所有在大雷音寺有收获之人全都紧了紧手中的器物，这东西如今被证实了，的确有不凡之力。
<br/>　　叶梵与庞博突然看向夜邪，他们顿时想起了刚才夜邪的叮嘱，让他们不能放弃手中的佛家器物，如今看来，他早已知晓。
<br/>　　“大家赶紧走，此地不宜久留！”夜邪突然大声喊道。
<br/>　　“啊……”
<br/>　　惨叫声再次响起，又有一名同学仰天倒地，在他的额头之处，一样有一个拇指般大小的血洞。
<br/>　　“嗖！”
<br/>　　破空声响起，夜邪双耳一动，他面色冷峻，暗道一声“找死。”
<br/>　　他挥动手中的金刚杵，向着虚空劈去，金刚杵顿时发出金色光辉，与一个黑影撞击在了一起。
<br/>　　“轰！”
<br/>　　突然空中发出一阵火花，随着“叽”的一声，有一团黑影在火光之中被焚灭了。
<br/>　　“走！”
<br/>　　夜邪一言不发，刚进向着远处冲去。其他人见此，如梦方醒，赶紧向着远处跑去，同时心中也极为吃惊，这人竟会武功，这怎么可能？
<br/>　　“噗通！”
<br/>　　再次有人倒地，死状依旧如之前那般，极为凄惨，触目惊心。
<br/>　　“夜邪，那是什么东西？”叶梵与庞博跑到夜邪身边，问道。
<br/>　　夜邪看了两人一眼，说道：“传说大雷音寺是佛祖用来镇压十八个妖魔的，下面封印着十八层地狱。”
<br/>　　“你是说……那是佛祖镇压的妖魔？！”两人顿时瞪大了双眼，眼中流露出不可思议的眼神。
<br/>　　“若是妖魔真的如此弱小的话，还用得着佛祖镇压吗，如果我没有猜错的话，那是第一层地狱中镇压的神鳄的后代。”夜邪面无表情道。
<br/>　　“也是。”两人想了想，觉得真是如此。
<br/>　　“噗！”
<br/>　　一道乌光闪过，一朵妖异的血花在虚空绽放，一名男同学双眼睁得大大的，仰天倒地，在没了声息。
<br/>　　“呜呜呜……谁来救救我们？”有人比较脆弱，被这接二连三的死亡打击到，心里的防线，已经被彻底摧毁了，顿时哭了出来。
<br/>　　“啊，我们没有得到神物，怎么办？”有同学没有得到佛门器物，顿时惶恐不安。
<br/>　　“这东西应该可以共用，我们尝试一下！”有人提议两人共用一件神物，可惜，谁也不想自己的生命受到危险，这个险，谁也不敢冒。
<br/>　　越是危险之际，就越能显现出人们的心性。平时最好的朋友，此刻也可能比不上一个路人。人与人之间的信任，在生命遭受危险这一刻，变得脆弱不堪。
<br/>　　当然，也有人愿意尝试，与他们共用一件神器。
<br/>　　人们不断地朝着混沌神殿跑去，在这一刻，混沌神殿成了他们唯一的救命之所，他们实在是想不出，如今还有什么地方能够保护他们。
<br/>　　一路奔跑，一路的死亡，一路与死神擦肩而过，一路的恐惧，有人疯了，有人哭了，也有人起了别样的心思。
<br/>　　毕竟，还是有很多人没有这所谓的“神器”的，也就有人在名为刘云志的怂恿下，对叶梵动手，想要借助与他共用一盏青灯的机会，将他的青灯抢走。
<br/>　　可惜，他遇到了夜邪，刚有行动，便被发现了。
<br/>　　砰！
<br/>　　夜邪的金刚杵一闪，便直接将那人击昏在地。他眼神冷厉，淡淡地扫了周边众人一眼，而后直接走到刘云志三人身边。
<br/>　　“你，就是刘云志？”夜邪那淡淡的声音在三人耳边响起，声音之中，充斥着无尽杀意，宛若从地狱之中传了出来。
<br/>　　“是我有如何？你敢把我怎么样？”刘云志眼神缩了缩，心头有种不祥的预感，被夜邪这样盯着，他有些发毛。
<br/>　　“呵呵，是你就好。”夜邪轻笑一声，手中金刚持轻轻一甩，直接向着刘云志砸去。
<br/>　　“轰”
<br/>　　金光辉煌，神光弥漫，刘云志手中的戒尺同样发出金光，进行抵挡，可惜，夜邪的力量太大了，那戒尺只能自动抵挡一部分力量，而且因为同是佛门器物的原因，那守护之力并不强。
<br/>　　巨大的力量径直通过金刚杵传到刘云志身上，刘云志神色顿时大骇，他如何也想不到，眼前这人竟会有如此的的力量，他也想不到这人竟然丝毫不顾及其他人的看法，直接对他出手，让他防不胜防。
<br/>　　巨大的力量冲击在他身上，让他直接倒飞出三丈之远，至于他手中的戒尺，也因为这巨大的力量震散开来。
<br/>　　夜邪一跃而起，他手中的金刚杵发出巨大的光辉，径直将刘云志贯穿。
<br/>　　“你……你怎么敢……”刘云志眼神中满是不可思议，直到死去，他都不明白，为何夜邪敢杀他。
<br/>　　“白痴，这里早已不是地球了。”夜邪淡淡出声道。夜邪的话很简单，这里不是地球，在地球上的法则早已管不了这里了。宇宙之中，向来都是弱肉强食。
<br/>　　他抽出金刚杵，来到边上捡起戒尺，这戒尺，应该能够在混沌神殿中换到不少价值点。
<br/>　　“你……”还剩下两个刘云志的狗腿子，看到夜邪向他们走来，他们顿时脸色变得苍白，想要逃跑，可脚下为何没有丝毫的力量。
<br/>　　他们只能眼睁睁看着夜邪将他们杀死，就算他们逃跑，也不行了。夜邪原本便是杀手，如今神兵在手，对付两个普通人，再简单不过了。
<br/>　　杀了两人，夜邪再次得到两件佛门器物，一件铜铃，一件鱼鼓。
<br/>　　“这……”
<br/>　　其他人此刻全都睁大了眼睛，满脸不敢置信，同时，他们也有些醒悟过来，是啊，这里是宇宙中，地球上的法则已经不适合他们了，这里一切都是以实力为尊。
<br/>　　尽管如此，但他们依旧不敢去靠近夜邪，刚刚夜邪杀人的样子，十分可怕。以至于他们看向夜邪的眼中都带着深深的敬畏。一时之间，现场气氛变得微妙起来。
<br/>　　人心难测，莫不如是！
<br/>　　唯有叶梵与庞博两人敢来到他身边，与他谈话。
<br/>　　“好，我早就看刘云志那货不爽了，现在死了，真是太好了。不过，夜邪，你刚刚可是很威风啊，你是不是学过什么？”庞博大大咧咧道，虽然心中对于夜邪杀人有些冲击，但并不大，更何况，夜邪是为了他们好。
<br/>　　“我之前学过一些武术。”夜邪笑了笑，并没有说自己之前是杀手，而且还是穿越过来的杀手。
<br/>　　“原来如此，怪不得你如此生猛！”两人恍然大悟。
<br/>　　“我们还是赶紧走吧，我看危险还远没有解决。”
<br/>　　“对，我们走！”
