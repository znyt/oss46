　　其实连她自己，也说不上来那种感觉，总觉得自己最近，似乎少了几分必胜的信念，以往纯粹无比的剑意之中，似乎多了几分杂念。
<br/>　　恍然之间，洛冰玄脑海之中，也不知道怎的，陡然又蹦出了叶尘的笑脸。
<br/>　　轰！
<br/>　　这道念头一旦生出，洛冰玄隐隐感觉自己心中的某种信念，仿佛随时都要崩塌一般。
<br/>　　一时间，洛冰玄出手变得越发迟疑，气势也顿时大为削弱，剑意更加不如之前那般凌厉。
<br/>　　而云破天虽然不知道洛冰玄发生了什么，却立即抓住机会反扑，反而气势大涨，一举压过了洛冰玄，
<br/>　　“哈哈哈哈！洛冰玄！看来今日，你这天骄榜第二的位置，也该易主了！”
<br/>　　云破天反守为攻之后，顿时越发的得意，忍不住哈哈大笑起来。
<br/>　　“哼！”
<br/>　　洛冰玄冷哼一声，当即收敛心神，将心头的杂念抛开，很快又稳定住了局面，但是依然落入了下风。
<br/>　　就在云破天正得意无限之时，
<br/>　　忽然！
<br/>　　轰！
<br/>　　一道充满毁灭气息的刀芒，突然出现在他头顶上空，朝他狠狠劈落了下来！
<br/>　　原来却是一旁远处的叶尘，终于按捺不住，对云破天出手了！
<br/>　　“小子！你敢偷袭……”
<br/>　　云破天心中暴怒，来不及多想，当即顾不得继续追击洛冰玄，只好凝聚出一道剑芒，迎头挡了上去。
<br/>　　咔嚓！
<br/>　　伴随着叶尘的毁灭刀芒落下，云破天的剑芒，瞬间便粉碎开来。
<br/>　　“什么！！”
<br/>　　云破天显然没有想到，叶尘的一击之威，居然强大到如此地步，当即全力运起护体真元，想要挡住那道诡异的刀芒。
<br/>　　咔嚓！
<br/>　　云破天的护体真元顿时再次被斩成粉碎！
<br/>　　铿！
<br/>　　虽然经过两次阻击，叶尘施展出的那道刀芒，已经黯淡了许多，但最终还是斩落在了云破天的肩头之上。
<br/>　　然而，
<br/>　　就在刀芒落在云破天肩头的同时，他的身上陡然发出一道璀璨的光芒！
<br/>　　却是他的那件仙器法衣起了作用，替他挡住了那最后一道毁灭刀气。
<br/>　　不过，云破天也并不好过，直接狂喷了一口鲜血之后，整个人倒飞而出，足足退到数千米之外，方才勉强站稳了身形。
<br/>　　“混蛋！你竟然敢背后偷袭，真是卑鄙下作！”
<br/>　　云破天站稳身形之后，顿时破开大骂，两眼气得发红，几乎快要喷出火来了。
<br/>　　他原本就没有将叶尘放在眼里，再加上叶尘突施偷袭，他一时间也反应不及，所以这才被叶尘一击得手，此时已经受了不轻的伤，这让他如何不怒？
<br/>　　“彼此彼此！当日你过河拆桥，偷袭冰玄仙子之时，可曾想过会有今日？”
<br/>　　叶尘出言嘲讽了一句，同时身形在空中留下一道道残影，已经闪电般冲到云破天的跟前，二话不说，再次一道刀芒斩落！
<br/>　　“你！……”
<br/>　　云破天此时，体内气血还正翻滚不止，哪想到叶尘居然如此卑鄙，气得差点又再喷出一口血来，当下也只好全力抵挡。
<br/>　　轰轰轰！
<br/>　　叶尘得势不饶人，一连数刀斩出之后，云破天顿时节节败退，若非他身上有仙器级别的法衣在身，恐怕早就被叶尘一刀劈成两半了。
<br/>　　当然了，云破天的实力其实并没有那么弱，主要是之前和洛冰玄打了这么久，已经消耗了不少真元，而且又是被叶尘突然偷袭，再加上他对叶尘一直比较轻视，这些种种原因，这才导致他一败涂地，已经完全没有翻盘的可能。
<br/>　　“龙老大！救我！！”
<br/>　　云破天万般无奈之下，居然向龙辰呼救起来。
<br/>　　轰！
<br/>　　眼看叶尘这一刀就要落下，云破天就算不死也得重伤。
<br/>　　就在这个时候，忽然一道金色巨龙虚影，自一旁飞出，挡在了云破天的跟前。
<br/>　　轰隆隆！
<br/>　　叶尘的刀芒，斩落在那头金色巨龙虚影之上。
<br/>　　咔嚓！
<br/>　　彭！
<br/>　　那道刀芒和金龙虚影，连同周围的虚空，顿时都尽皆破碎开来。
<br/>　　蹬蹬蹬！
<br/>　　叶尘顿时感觉到一股强大的反震之力，不由得身体一晃，忍不住倒退了三四步。
<br/>　　而等到周围的空间，再次恢复了正常，只见在那云破天的跟前，出现了一道高大的青年身影。
<br/>　　此人正是龙辰！
<br/>　　叶尘不由得眉头微微一皱，
<br/>　　这几乎应该算是他，第一次看到这位天骄榜首出手。
<br/>　　不得不说，这龙辰的实力，比他想象的还要厉害！
<br/>　　即便是如今的他，也没有绝对把握，可以将此人击败。
<br/>　　要知道，两人如今的修为，都压制在元婴境，就算龙辰被他的修为要高出一些，却也高不了太多！
<br/>　　而两人若是完全恢复了修为，龙辰恐怕至少也是化神境后期，而叶尘却仅仅只有元婴巅峰境界，那个时候就相差的实在太多了，他就算使出所有底牌，恐怕也几乎没有把握击败此人。
<br/>　　不愧是天骄榜首！
<br/>　　叶尘在心中暗暗感慨了一句，没有再继续出手。
<br/>　　龙辰同样也深深的望了叶尘一眼，呵呵一笑道：
<br/>　　“叶兄弟，大家能够走到这里，都已经极为不容易，何必要在这里内讧呢？更何况，你背后偷袭，实在有些不妥！能否给我一个面子，此事就此作罢，可好？”
<br/>　　叶尘听到这话，不由得眉头微微一皱，
<br/>　　这个龙辰，为人处世未免有些太圆滑了，竟然谁也不得罪，而总喜欢在别人最需要帮助的时候，突然站了出来，未免有拉拢人心之嫌，让人不得不怀疑他的用心。
<br/>　　不过，既然龙辰已经出面，叶尘再想要干掉云破天，显然是不太可能了，于是便顺势下坡道：
<br/>　　“好，我给龙辰兄这个面子，只要他不来招惹我和冰玄仙子，之前的事情，我可以当做没有发生过！”
<br/>　　说完这话，叶尘身体微微一晃，来到洛冰玄的跟前，
<br/>　　“你没事吧？”
<br/>　　不料，洛冰玄的美眸之中，却忽然闪过一抹复杂之色，继而冷冰冰道：
<br/>　　“我有没有事，和你有什么关系？请你以后不要再多管我的闲事！”
<br/>　　撂下这么一句冷冰冰的话，洛冰玄直接转身回到了自己的修炼道场之中。
<br/>　　叶尘顿时一阵愕然。
