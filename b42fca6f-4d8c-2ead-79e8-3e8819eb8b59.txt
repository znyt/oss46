　　“你家儿子，辈分是个什么字？”张道长问道。
<br/>　　“这一辈正好轮到个‘全’字，请道长赐个名字。”
<br/>　　张道长手拈胡须，想了一下说道：“自古父母取名，先生赐字，我们道门就没有字了，我赐他个道号吧……就叫悟得如何，悟得悟得，有悟才有得。”
<br/>　　刘四夫妇连忙道谢。
<br/>　　叶少阳却是撇了撇嘴，悟得……这个道号取得可真是够省事的。叶少阳本想调侃张道长几句，把这孩子的姓和道号连在一起读，刘悟得……嘴角已经绽开的笑容僵住了。
<br/>　　叶少阳怔怔地看着张道长，深吸一口气，喃喃道：“刘悟得，终南山弟子……”
<br/>　　张道长笑了笑，望着襁褓中的婴儿，想到他那傲人的天赋，心中也是十分得意，说道：“暂时还不是呢，不过以他的天赋，我相信进入终南山内门，不是问题。”
<br/>　　“肯定不是问题……”
<br/>　　他会在十五岁那年进入终南山内门，二十岁晋升天师牌位，一生纵横法术界，参与过很多法术界的大型行动，声名显赫，活到七十岁，无疾而终，往生修罗……
<br/>　　他是终南山近百年来出的最为了得的弟子，少年时为道门奇才，尤其是他二十岁晋升天师牌位，一度是法术界有记载的最快记录，后来被一个十六岁晋升天师的人打破记录，这个人就是叶少阳自己……
<br/>　　算起来，这位刘悟得天师，是跟青云子平辈的人物，青云子跟他也挺熟，说这个人天赋极佳，因为小时候开了天知，体质不同，修行道路一路开挂，用当代的话说，是道门的明日之星，本来造诣应该在他自己之上，差一步晋升灵仙，不是天别的，只是因为他这人太老实呆板，不敢迈出这一步。
<br/>　　名字一样，小时候的经历也类似，又都是终南山的弟子……
<br/>　　叶少阳怔怔地看着刘四媳妇怀中的婴儿，实在难以想象，自己竟然一个巧合之下，救了一位之后的道门奇才——想来应该说是“之前”，因为这家伙是自己的前辈。
<br/>　　救了自己的前辈……这种感觉，让叶少阳体会到了一种错乱。
<br/>　　“你居然是我前辈……”叶少阳伸手摸了一下刘悟得的脸，刘悟得喝了奶，又没有恶鬼纠缠，晒了会太阳，人也略微精神了，仿佛是看到了叶少阳，冲他咧嘴，露出了一个婴儿的标准笑容。
<br/>　　叶少阳无奈地摇了摇头。
<br/>　　张道长让刘四夫妻先带孩子回去，自己稍后会去他们家里。刘四夫妻对二人千恩万谢，然后抱着孩子离开。
<br/>　　“贤侄！”刘老汉让刘四夫妻等着自己，眉飞色舞地走到叶少阳面前，眼中满满都是兴奋。刘四一家不用去流浪，这一点让他很兴奋，不过他更兴奋的是，叶少阳没有让他失望，他不图任何物质上的回报，只求在老邻居面前不要丢了颜面，叶少阳的所为，让他面上装光，比叶少阳本人要兴奋的多。
<br/>　　“贤侄！你先去跟张道长谈正事，我先回家准备酒菜，随便把事情告诉你老姑，让她也高兴高兴！”刘老汉口吻之中，俨然已经把叶少阳当成了真的侄子。
<br/>　　跟叶少阳道别之后，刘老汉追上刘四夫妻，跟他们一起回去，很装逼地问他们：“我怎么跟你们说的，我这侄子，是不是真有法力的？”
<br/>　　刘四夫妻感激的不行，让刘老汉心里更是开心。
<br/>　　等他们走后，张道长看了两棵槐树一眼，对叶少阳说道：“让他们在这收场吧，我们也走，我还有事要请教。”
<br/>　　两人一路慢慢地走回思故村，路上叶少阳不说话，张道长不知道他在思索什么，也不打扰。
<br/>　　叶少阳在想自己无意中救了刘悟得这件事：张道长给他取道号叫悟得，收入门下，如果不出意外，按照之后叶少阳所知道的轨迹，这个孩子长大之后，能够进入终南山内门，一路开挂，成为扬名天下的一代宗师……自己在他的命运中，起到了关键性的作用。
<br/>　　叶少阳开始想，假如没有自己，刘悟得的命运会怎么样？
<br/>　　连张道长都在一个劲地说，是自己救了刘悟得，如果自己没有穿越，或者没有出手相救，那么，刘悟得的父母就要带着他远走千里之外，但是并不能真正摆脱那三个恶鬼，最后很可能会因为吸入太多阴气而死……这样，历史上就没有刘悟得了？
<br/>　　但是自己那个世界的历史中，刘悟得是自己的前辈，不可能是自己救下的……有点乱，叶少阳摇了摇头，努力把这个逻辑弄的简单一点：自己救活了刘悟得，所以刘悟得能成一代宗师，在自己那个世界，成为自己的前辈。但是，在自己那个世界，刘悟得小时候遇险时，自己还没出世……
<br/>　　那么，当时究竟是谁救了他？
<br/>　　叶少阳想来想去，有两种可能，要么就是，如果没有自己，也会有别人救下他，然后他的人生轨迹还是没变。
<br/>　　要么就是……有两个自己，在那个属于自己的世界，自己还没有出身，但是后世的自己已经穿越到了百年前，救下了刘悟得……就像眼前自己经历的一样，假如自己能在这个世界生活到九十多年，应该是可以看到另一个自己出身……
<br/>　　这个念头，让叶少阳心中感到一阵可怕。
<br/>　　他仔细思索，觉得如果前者是真相，那么证明历史是可以修改的：本来刘悟得应该是被别人救下，而出于巧合，才被自己这个穿越者意外拯救。
<br/>　　如果是第二种可能……那只能证明，整个时空的历史，是一个轮回……
<br/>　　无论真相是哪一种可能，都会引发更多的难以解释的疑问。叶少阳自认以自己的文化程度，根本就没法相通这些问题。
<br/>　　最后干脆还是不想了。关于时空运转的轨迹和真相，要么属于天道，要么属于物理，反正都不是自己一个法师该去思考的。自己的目的就是找到徐福，从哪儿来从哪儿回去。
<br/>　　叶少阳这样安慰自己，心头仍然体会到了一种深深的孤独。这个时候，他多么希望捉鬼联盟的小伙伴们在身边，哪怕一个人也行，至少能跟自己讨论一下，而不是像现在这样，自己一个人独自茫然地前进。
