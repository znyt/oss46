　　“太二真人？”
<br/>　　太乙真人闻言，额头升起一丝黑线，这熊孩子。同时他口中无奈道：“是太乙真人。”
<br/>　　“太乙不就是太二吗？”小孩满不在乎道。
<br/>　　“道长见怪了，我这小儿胡闹，你别在意。”一旁李靖也是尴尬不已，连忙解释道。
<br/>　　“无妨！”
<br/>　　太乙真人看向李靖怀中的小孩，也是满心欢喜，这小孩虽然是个熊孩子，但资质逆天啊，他的道统总算有个可以传承之人了。
<br/>　　接着，太乙真人简单说明了一下来意，说着孩子是灵珠子转世，他想收其为徒弟，李靖听后，自然大喜，答应了，同时，他也知道他儿子为何要怀孕三年多了，原来是神仙转世，不是怪物啊。
<br/>　　太乙真人为其取名为李哪咤，随后打入一道功法在哪咤脑海之中，又送了哪咤几件灵宝，有混天绫，乾坤圈与风火轮，最后，他还送李靖一柄银色宝剑。
<br/>　　李靖自然大喜过望，接过宝剑，拔出来想要实验一番宝剑的锋利。
<br/>　　“呲吟——”
<br/>　　宝剑出鞘，一声清脆的剑吟声响起，李靖对着虚空便是一划。瞬间，一股莫名的法则波动向着四周荡漾开来，谁也没有发现。紧接着，长剑对面不远处一仆人便不由自主飞了过来，用双手迎向宝剑锋刃。
<br/>　　“啊？怎么回事？，我怎么不受控制了？”
<br/>　　“哗啦——”
<br/>　　一地血渍溅起，一个人头落地！
<br/>　　怎么回事？我不过是实验一番，怎么就死人了？李靖惊讶不已。
<br/>　　同时，一道神秘信息出现在李靖脑海中：“因果律武器之绝杀白刃，效果：百分之百被空手接白刃！”
<br/>　　……
<br/>　　金乌飞，玉兔走，时光瞬息，寒来暑往，忽忽七载而过。
<br/>　　这日武成王发来令箭，李靖接箭，忙传军令，命各处守军把持关隘，每日操练三军，训练士兵，堤防马岭等边关要塞。
<br/>　　正是五月天气，大日炎炎似火烧，三公子哪咤坐在家中觉得十分闷热，心中烦躁不已，便禀告母亲，出去玩耍。殷夫人同意了，哪吒带着家将李元出得关外。
<br/>　　两人行至一里多，天气燥热，哪吒忽闻前方崖旁流水潺潺。他问李元，那是哪里。李元回答道：“那是九湾河，是东海的海口，连通着大海。”
<br/>　　哪吒一听，走到河边，看着那波涛汹涌的水势，叫道：“此处甚好，方才出关来，全身燥热，一身是汗，今日便在这河中洗一个澡。”
<br/>　　李元劝道：“公子小心，只怕老爷快要回来了，我等还是早日回去，不让老爷骂你。”
<br/>　　哪吒却道声无妨，说完便脱下衣裳，直接跳入河中，用那七尺混天绫搅动河水，洗干净身子，之后他玩心大起，便在河里泼水嬉戏。
<br/>　　这混天绫乃是乾元至宝，有翻天覆地之威，放在水中，把水映得通红，摆一摆，江河便晃动起来，摇一摇，乾坤震动。
<br/>　　哪咤正在玩得欢快，却不知他把这水底的水晶宫晃动个不停，东海龙王在宫殿之中，听得地动河晃，急忙询问左右侍从，为何会出现如此情况。之后连忙传令巡海的夜叉，出去看望。
<br/>　　夜叉得令，出去海面，来到九湾河一望，便看到一个小孩正手持红色绫条在水中嬉戏。夜叉分开水浪，冒出头来，声若雷霆，喝问道：“哪家的小孩，竟敢在此作怪，将河水晃动，扰乱龙宫安宁？”
<br/>　　哪咤回头一看，却见海面上站着一个怪物，形貌甚是丑陋，约有两丈，绿色皮肤，红色的头发，獠牙露出嘴外，手持黑色三角叉，狰狞恐怖！
<br/>　　哪咤本是应杀劫而生，将来更是海会大神，要降服九十六洞妖魔，如今看到这夜叉丑样，却也不害怕，反而觉得好笑，说道：“你这畜生，长得好生奇怪，竟然也能说人话？”
<br/>　　夜叉听后大怒：“吾为巡海夜叉，竟然被你说成是畜生，看打！”
<br/>　　说着，夜叉将黑色三角叉晃动，黑风旋转，向着哪吒头顶扎来，欲将哪咤擒住，捉回龙宫。
<br/>　　哪咤见此，也不害怕，他轻轻一跳，便闪过夜叉攻击，直接跳到他背后，右脚一蹬，正踢中那夜叉的臀部。夜叉先前扑出，落入水中，水花四溅，哪咤站在水上见此一幕，拍手大笑。
<br/>　　夜叉见此，不由恼羞成怒，当下直接使出真本事，身体一晃，便幻入虚空。哪咤见此，以为那夜叉跑了，不敢出来见人，却忽敢身后有恶风袭来，他转身一看，却见那夜叉双目通红，手中黑叉卷起重重叉影，向他席卷而来。
<br/>　　哪咤大惊，急忙将手中乾坤圈向前一推，将那三角叉挡住，手中混天绫挥舞，打在夜叉身上。
<br/>　　“啊！”
<br/>　　夜叉痛呼一声，连忙隐入虚空，之后忽隐忽现，忽左忽右，叠影重重，向着哪咤打来。
<br/>　　原来夜叉有三种，一种在天，一种在地，一种在虚空，哪咤见到的，便是虚空夜叉，力大无穷，来去如风，善于隐藏身形。
<br/>　　哪咤见此，也觉有些畏惧，只好将太乙真人给他的混天绫乱舞起来，舞动之处，红芒遮天，将那夜叉打落虚空，哪吒人小，出手亦不知轻重，一不小心，便将那夜叉的脑袋砸碎。那夜叉落入水中，死了。
<br/>　　哪咤见此，神情一呆，他却不知这混天绫与乾坤圈是元始赐给太乙真人的宝物，夜叉是鬼物，虽然小有神通，但终究抵挡不住这至宝的威力，当场死亡。
<br/>　　夜叉死亡，龙宫内的龙王立刻便得到了消息，龙王大惊，立刻就要点兵，前去查看。这是三太子敖丙进来，询问了缘由，三太子听后说道：“谅他小儿又有多大神通，父王不急，待儿子出去将他擒拿过来便是。”
<br/>　　之后敖丙连忙调动数百虾兵蟹将，手提画戟除了龙宫，向着九湾河而来。
<br/>　　敖丙点兵出海，声势浩大，只见百余里乃乌云汇聚，波涛用来，如同大山，海浪怒吼，奔腾咆哮，海浪竖立起来，直接天际，颇为壮观。
<br/>　　哪咤见此，只觉得好玩，叫道：“好大的水！”
<br/>　　海浪之上，敖丙大声喝道：“是什么人打死我龙宫巡海夜叉？”
<br/>　　哪咤并无一丝害怕，无惧道：“是我。”
<br/>　　“你又是何人？”敖丙问道。
<br/>　　“我乃陈塘关总兵李靖第三子哪咤是也，我父亲乃一镇之主，镇守此间。我在此洗澡，那夜叉来骂我，我即便打死了他，又有何妨？”
<br/>　　敖丙见此人打死夜叉，竟然毫不悔改，还洋洋自得，勃然大怒：“好贼子，你将人打死，还在此处胡言乱语！”
<br/>　　三太子将手中画戟一振，煞气翻腾，向着哪吒迎面劈来，哪咤身体小巧玲珑，轻轻一晃，便躲了过去，问道：
<br/>　　“你是何人，报上名来，怎能无端动手？”
<br/>　　敖丙道：“小娃娃你且听好，吾乃东海龙王三太子敖丙是也，你大胆打死龙宫夜叉，扰乱龙宫安宁，我要拿你命来抵偿。”
<br/>　　哪咤笑道：“你原来是敖广之子，你这小泥鳅口气倒是不小，我不怕你，就连你那老泥鳅来了，我也敢把他的皮给剥了。”
<br/>　　三太子一听，大叫道：“气煞我也，你这小贼，好生无礼！”
<br/>　　敖丙动了真怒，手中画戟快速舞动起来，破空声响起，向着哪吒连刺而来。哪咤丝毫不惧，手中乾坤圈晃动，金光灿灿，夺日生辉，向着敖丙的兵器打来，顿时空中铿锵声不断，震耳欲聋，一瞬间两人竟然交战上百次。
<br/>　　一时之间，两人不分胜负。
