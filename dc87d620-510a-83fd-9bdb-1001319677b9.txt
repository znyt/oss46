　　第737章
<br/>　　这一员小将白盔白马手持一杆亮银枪，靠近山门时忽地勒马停下，用枪指着程大雷。
<br/>　　“狗贼，过来受死。”
<br/>　　程大雷白了他一眼：“娃娃，回去歇着吧，本当家怜你小小年纪，连女人都不曾碰过，不忍心送你上路，快快回去吧。”
<br/>　　小将冷哼一声：“休要废话，快来与小爷一战。”
<br/>　　程大雷撇撇嘴，这明显就是给脸不要脸了：“娃娃，你可知道，你就算死在本当家马下，也是个贼人的身份，连家族祠堂都入不得，你当真愿意？”
<br/>　　“贼人，你畏畏缩缩，不敢近前，是怕了小爷不成。”
<br/>　　程大雷无可奈何的叹口气，年轻人血气方刚，初生牛犊不怕虎也不算什么大错。但明显不知天高地厚就有些不太好了。
<br/>　　他目光转了一圈，捡起门前一块半截砖：“娃娃，你可要小心了，你可要仔细了。”
<br/>　　魏牧冲在山下观战，他是书生出身，上阵打仗并不在行。之所以迟迟没有令大军一拥而上，也是顾忌程大雷本领高强，牺牲太多。
<br/>　　这员小将名叫郝学武，是寒门出身。魏牧冲这次掌权，很是提拔了一批有能力没出路的寒门子弟，所以手下对他都很忠心。
<br/>　　这郝学武便是其中之一，他年纪不到二十，却学了一身好枪棒。魏牧冲给了他个校尉的差事，看他能不能胜任，如果可堪造就，日后更要委以重任。
<br/>　　年轻人想要出头，魏牧冲还未下令，他便冲了出去。魏牧冲也就只好冷眼旁观，试探程大雷本领如何。
<br/>　　也不知郝学武面对他有几分胜算。
<br/>　　“呔，瞧我打神砖呐。”
<br/>　　正琢磨着，一道黑影从天而降砸向郝学武。他早已全神戒备提防，此刻一枪抡了出去，将飞到身前的『打神砖』击飞。
<br/>　　“哼，贼人不过如此。”
<br/>　　郝学武得意笑了一声，忽然一怔，只见半截砖头又砸了过来。一前一后，程大雷出手也是神出鬼没，郝学武挡得住第一块却挡不住第二块。
<br/>　　事实上，程大雷真如果全力出手，他一块也挡不住。不过这方面的本事，没什么值得夸耀的，两军对垒，程大雷也不可能躲在暗处丢砖头。传出去，也不是什么露脸的事。
<br/>　　郝学武被程大雷从马上闷了下来，头破血流，被己方人马抬回去的时候，气得破口大骂。
<br/>　　“暗器偷袭，卑鄙。”
<br/>　　程大雷根本没有搭理他，两军对垒，哪里来得卑鄙不卑鄙，还是太年轻呐。
<br/>　　他拍了拍双手，叉腰立在山门前。
<br/>　　“嘿，还有那个不怕死的，在本大当家面前走上几合，传出去也是你们的荣耀，本当家今天大放送，不会太为难你们的。”
<br/>　　还颇有几人跃跃欲试，战场上升迁除了军功，便是名声。假若能和程大雷交手一次，哪怕是被打得落花流水，也是光荣。
<br/>　　但魏牧冲制止了手下的冲动，程大雷的本事深不可测。真如果让手下兵丁轮流上去，挨个被他击败，怕是会折损士兵的士气。
<br/>　　程大雷等了一会儿，见无人上前，连魏牧冲也不再和自己对骂。
<br/>　　他百无聊赖的叹了口气：“你们要不过来，我就先歇着了，等你们什么时候想明白了，再叫醒我。”
<br/>　　说着，程大雷大摇大摆回到清风寺，山门在其身后关上。
<br/>　　“程当家，可想到脱身的法子？”了然和尚问。
<br/>　　“还没有。”程大雷摇摇头，单枪匹马闯出城门难度太大，他现在也不想冒险。
<br/>　　“法师，我还担心一件事。”程大雷道：“我走了你怎么办？”
<br/>　　“程当家果然宅心仁厚。”了然和尚呵呵笑了一声：“不过这一点程当家可以放宽心，贫僧自有应对之策。我在城中也有些香客，魏城主顾忌他们的存在，便不会对我赶尽杀绝。更况且，我可说是施主以强权迫我，他们不会太难为我的。”
<br/>　　“那便好，那便好。”
<br/>　　程大雷心想：这老和尚不简单么，扯起谎随口就来，不是说佛家不打诳语么，看着也不是个老实和尚。
<br/>　　乱世之中，恶人多的是，程大雷也没有细琢磨。他现在在意的，还是如何从此地脱身。
<br/>　　了然和尚也在替程大雷想，绞尽脑汁也没有半分收获，他摇摇脑袋叹了一声：“想要安然无恙离开汝南城，怕是只有大罗神仙下凡，有腾云驾雾的本事。”
<br/>　　程大雷正在苦苦思索，忽地眼睛一亮：“腾云驾雾，未必不可能啊。”
<br/>　　“施主，你说什么？”
<br/>　　“嘿，你说巧不巧。”程大雷大手拍在了然和尚肩上：“偏偏我就是个呵气成云，挥手成雨，有腾云驾雾本事的仙人传人。”
<br/>　　了然肩膀酸疼，险些倒在地上。他揉着肩膀，心道：程施主该不会是急火攻心，得了失心疯吧。当然，平常他看着也不太正常。
<br/>　　程大雷有了思路，便顺着想了下去，来来回回琢磨，觉得虽然有些冒险，但未必不可能做到啊。
<br/>　　也是时候给这个世界露一手了。
<br/>　　“庙内可有牛皮，竹竿之物？”
<br/>　　了然仍旧是一头雾水：“庙内不杀生，没有施主想要的这些。”
<br/>　　“这样啊。”程大雷叹了一声：“如此说来，只能冲外面的魏牧冲要了。”
<br/>　　“他怎么会给施主送来？”了然奇怪道。
<br/>　　“嗳，这不就是要委屈委屈法师了，也正好让你的谎言更逼真些。”
<br/>　　“我？”自诩也有几分才智的了然和尚，觉得自己越来越跟不上程大雷的思路了。
<br/>　　程大雷没有为他解惑，只是将庙里的和尚绑了一串出去。这一幕倒是令魏牧冲一惊，莫非程大雷要用这些人的性命要挟出城。
<br/>　　假若真是如此，自己就只能舍掉这些和尚的性命，拼死留下程大雷。
<br/>　　程大雷没打算用人质出城，他只是用他们的性命要求魏牧冲送来兽皮和竹竿。
<br/>　　魏牧冲当真不知程大雷要做什么，也存着几分看他耍得什么把戏的打算，竟依样给程大雷送了过来。反正只要围兵不策，程大雷纵然插翅难逃。
<br/>　　可惜，程大雷想的就是插翅的法子。
<br/>(本章完)
