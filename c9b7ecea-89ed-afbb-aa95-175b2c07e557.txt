        唐倩兮伸出食指，戳了戳诺筱颖的额头，娇嗔：“臭丫头！”
<br/>
<br/>        “不如，嫂嫂替我想一个吧！”
<br/>
<br/>        “那你就叫小谢好了！”唐倩兮信手拈来道。
<br/>
<br/>        诺筱颖微笑着反问：“为什么叫小谢？”
<br/>
<br/>        “因为我们都是来自聊斋里面的女鬼！专门‘勾引’娱乐会所里那群男人的！”唐倩兮眉飞色舞地回答。
<br/>
<br/>        诺筱颖噗嗤一笑，她的这个嫂嫂看似不正经，其实是最正经的。
<br/>
<br/>        夜幕降临的时候，夜魅娱乐城，是滨江城最大最豪华的娱乐会所，这里囊括各项夜生活的娱乐活动。
<br/>
<br/>        在唐倩兮的推荐下，诺筱颖成为了今晚娱乐城的头牌驻唱。
<br/>
<br/>        看着在台上表现得落落大方的诺筱颖，唐倩兮的上司费尔南不禁赞许地问道：“小倩，你这是上哪儿找来这么一个会唱歌的女孩？”
<br/>
<br/>        “我男朋友的亲妹妹。”唐倩兮勾唇一笑，放眼看向舞台上的诺筱颖。
<br/>
<br/>        费尔南挑眉道：“刚刚有个官二代跟我说，看中这妞了，想这妞给他陪酒。”
<br/>
<br/>        “得了！别打我家小姑子的主意！”唐倩兮白了费尔南一眼。
<br/>
<br/>        费尔南抹了抹额上刘海，悻悻地只身离开。
<br/>
<br/>        唐倩兮又看向舞台上的诺筱颖，不禁会心一笑。
<br/>
<br/>        与此同时，台下某个豪华的雅座里，有一双愤愤不平的眼睛，真盯着台上唱歌的诺筱颖，甚至恨不得要将她五马分尸就好。
<br/>
<br/>        诺筱颖怎么回来滨江城？她怎么就这般阴魂不散啊！
<br/>
<br/>        苏漫雪一直以为，诺筱颖已经滚回她那鸟不拉屎的家乡去了，却没料到，诺筱颖竟然会出现在滨江城！
<br/>
<br/>        她好不容易才得来的荣华富贵，可不能让诺筱颖给破坏了。
<br/>
<br/>        此时的苏漫雪，已经不同往日，她一身限量版的名牌，穿金戴银，珠光宝气的像个贵太太。
<br/>
<br/>        不，她现在已经是贵太太了。
<br/>
<br/>        托诺筱颖的福，她如今是赫赫有名的傅氏集团继承人傅大少爷的未婚妻！
<br/>
<br/>        就在这个时候，傅夜沉一袭黑色西装革履，英俊成熟中透着难以抵挡的王者风范，他在几个同僚的陪同下，霸气十足地进了夜魅娱乐城。
<br/>
<br/>        路经大厅，听到熟悉的歌声，傅夜沉驻足，循声望去。
<br/>
<br/>        看到台上站在话筒前唱得忘情忘我的诺筱颖，傅夜沉不由地握紧了拳头。
<br/>
<br/>        镁光灯下，她一袭白裙，出淤泥而不染。略施粉黛而显得更加甜美的脸庞，确实俘虏了不少男人的心。
<br/>
<br/>        但是！
<br/>
<br/>        苏、曼、雪！
<br/>
<br/>        身为他傅大少爷傅夜沉的未婚妻，竟然降低身份，来这种地方卖唱！
<br/>
<br/>        她是嫌他给的钱不够多，还是她本性就是如此不放浪形骸？
<br/>
<br/>        “四少，怎么了？”韩剑锋见傅夜沉愤怒地瞪着台上那个女人，下意识地问。
<br/>
<br/>        当他看清楚台上那女人的样子后，不禁纳闷了起来：“诶？！那个女人……不就是……”
<br/>
<br/>        傅夜沉回过神来，闭了闭眼睛，冷冷地打断了韩剑锋的话：“没事，我们走。”
<br/>
<br/>        对于苏漫雪的行为，真的令他有些失望了。
<br/>
<br/>        韩剑锋看了看台上的诺筱颖，又看了看已经走远了的傅夜沉，下意识地摸了摸自己的后脑勺，继续挪动步伐往前走。
<br/>
<br/>        上次他被毒蛇咬了，若不是嫂子及时出现救了他，恐怕他早就没命从那大山里活着回来。
<br/>
<br/>        怎么说，嫂子对他有救命之恩，他是不是应该去跟嫂子打声招呼？！
<br/>
<br/>        不过，现在不是打招呼的时候，还有正事要干！
<br/>
<br/>        韩剑锋心里想着，连忙跟上了傅夜沉的步伐。
<br/>
<br/>        另一边的雅座里，几个男女在谈笑风生。
<br/>
<br/>        “漫雪，你知道吗？我们的官二代郭裕郭大少爷竟然被台上那个唱歌的女的给拒！”坐在一旁的好友陈思伊笑着说，刚刚郭裕去找会所老板，让老板把台上驻唱的那个女人喊下来陪酒。
<br/>
<br/>        结果，老板说那女人卖艺不卖身，还着实给郭裕泼了一身冷水。
<br/>
<br/>        陈思伊的话，拉回了苏漫雪的思绪。
<br/>
<br/>        诺筱颖卖艺不卖身？呵！来这种地方卖唱了，还立贞节牌坊？真可笑！
<br/>
<br/>        苏漫雪在心底嘲笑着诺筱颖，然后拿出手机给台上的诺筱颖拍了张照，说不定以后还有用处。
<br/>
<br/>        话说回来，陈思伊是某品牌服饰公司老板的女儿，苏漫雪来滨江城后，是傅大少爷的弟弟傅昼景刚介绍给她认识的。
<br/>
<br/>        陈思伊一听苏漫雪是傅大少爷的未婚妻，便对苏漫雪格外殷勤，而且还自告奋勇，主动给苏漫雪当导游。
<br/>
<br/>        两个女人刚相识，一个女人对另外一个女人好，肯定是别有用心。
<br/>
<br/>        苏漫雪知道陈思伊想从她这得到什么。
<br/>
<br/>        说到傅家，那幢豪华到只剩下冰冷的别苑，她都住进去两个多月了，竟然还没见过那位傅大少爷的真容。
<br/>
<br/>        这傅大少爷说来也太神秘了吧！刘管家连大少爷的真名都不肯透漏给她！
<br/>
<br/>        苏漫雪有些无奈，但她却不无聊，反正傅大少爷给了她想要什么就能得到什么的权利。
<br/>
<br/>        所以，她先是收购了临海城原先她和诺筱颖所在的那家装饰公司，顺道把诺筱颖炒了鱿鱼，又下令让整个临海城所有与室内设计有关的公司高层，务必要拒收一个名叫诺筱颖的女人。
<br/>
<br/>        苏漫雪逼走诺筱颖后，以为自己可以高枕无忧，随后她又听说傅氏集团的总部在滨江城，于是让傅家的刘管家给她订机票，她也想来滨江城玩玩。
<br/>
<br/>        怎么说，傅氏集团是傅大少爷的，等她和这位大少爷结了婚，那傅氏集团不就是她苏漫雪的吗？
<br/>
<br/>        苏漫雪想想都觉得这事美滋滋的，所以，她更加要来滨江城看看。
<br/>
<br/>        “多少女人想爬我们的郭大少爷的床啊！那女人竟然还不识抬举！”见郭裕郁郁寡欢地坐回到雅座上，闷闷不乐地喝酒，陈思伊打趣地说。
<br/>
<br/>        苏漫雪回过神来，听到陈思伊这话，不由地脸色一沉。
<br/>
<br/>        一道灵光闪过脑海，苏漫雪嘴角露出一抹阴森的笑意：“郭大少爷，你若是对台上那个女孩很感兴趣的话，我倒是有个主意，可以让你抱得美人归。”
