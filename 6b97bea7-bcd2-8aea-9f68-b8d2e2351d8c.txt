一人一兽再次交锋，末日公牛迅速发挥出了它的优势，它手中的战斧如同断头台上的铡刀，每一次交锋都会让最勇敢的战士心生畏惧，它野蛮的嚎叫如地狱的号角一般摧残着正常人类的心智，使他们失去战心。
<br/>
<br/>    所幸，莱恩并不是正常的人类。
<br/>
<br/>    白狼骑士的嗓子内也发出低沉的怒喝声，他明亮的双眼就像雪原的白狼一样饱含着惊人的斗志，莱恩的战锤上电光闪烁，在和末日公牛的交锋之中丝毫不弱下风，他的每一击都饱含着惊人的力量，在双臂的驱动下，莱恩顶开了末日公牛手中的战斧，庞大的灵能力量在他的身上涌动，战锤朝着塔乌斯的身体放出了金色的闪电：“雷霆之力！”
<br/>
<br/>    “啊啊啊啊！！！”金色的闪电烧灼着塔乌斯的身体，它的身体表皮多处被烧成焦黑，野兽人兽王发出了痛苦的嚎叫，可是它的身体并未出现任何停顿，末日公牛使用拿着战斧的左臂强行顶着金色闪电的烧灼，冲至莱恩身前，右手的大砍刀从空中斩下，似有万钧之力。
<br/>
<br/>    莱恩不得不立即收回自己的灵能之力，立即侧滚翻躲开，他在地上滚了几圈才要起身，脑袋左前方出现的阴影让他几乎是本能地举锤抵挡。
<br/>
<br/>    正是多年以来的丰富战斗经验拯救了他，就在他翻滚的时候，末日公牛双臂撑地，朝着白狼骑士正在翻滚的身影猛扑而去，巨大的牛角直接顶在莱恩举起的锤柄上，白狼骑士倒飞数米，在高山的草甸上翻滚了数圈才勉强起身，莱恩感觉到自己的五脏六腑被这一下撞得乱七八糟，一缕缕鲜红色的血液从他的嘴角流下。
<br/>
<br/>    “唔~这是~龙车。”莱恩手捂胸膛，不止嘴角，他的鼻腔也流下了鲜血，每一次呼吸他能感觉到肺部在痛，喉咙处一阵阵甜腥味和铁的味道刺激着他的味觉。
<br/>
<br/>    龙车是一种强大生物前肢伏地然后极速冲撞的战技，野兽人将其称为“野蛮冲撞”，而人类则将其称为“龙车”。
<br/>
<br/>    现在不是感受痛楚的时候！莱恩使劲一拍地面站起，勉强半跪着，鲜血一滴一滴地从他的鼻子处和嘴角滴落在地面上，深吸数口冷气：“为了尤里克，白狼骑士绝不失败！”
<br/>
<br/>    另一边，塔乌斯的状态其实也不好，它的身上有多处烧焦的痕迹，刚才使出野蛮冲撞直接撞在了莱恩的锤柄上，头脑处的强烈晕眩感让末日公牛也花了不少时间摆脱，稍一恢复知觉，它充满着狂暴和怨恨地喊道：“很好……人类，你是第一个……这样都没死的……昔日那个湖里伪神的骑士……就这样被……削掉四肢，眼睛……挖出来，把舌头……剁碎，我将他的身体扔去喂……我的仆人！他也是这么讨人厌……他至死都还在咒骂我，而你……就是下一个！”
<br/>
<br/>    “女士在那段黑暗的岁月为骑士们带来了光明和戒律，我不允许你如此侮辱她的骑士，你必将死于我手！”莱恩能感觉到四肢百骸的剧痛感，他的眼中闪烁着庞大的怒火，嘴里默念道：“愿父亲与我同在。”
<br/>
<br/>    膝盖和盆骨传来阵阵响声，莱恩再次站起，朝着塔乌斯走去：“我们的战斗还未结束！”
<br/>
<br/>    说完，趁着塔乌斯尚未完全恢复，莱恩一锤击向末日公牛的头颅：“这一击是为了女士！”
<br/>
<br/>    塔乌斯当然不会坐以待毙，只是在它试图躲避的时候，身体内部突然冒出了强烈的金色闪电，灼痛着他的神经，它立即意识到这是眼前人类的阴谋，先前金色的闪电有部分潜入了它的身躯之中，等待着最好的爆发时机。
<br/>
<br/>    “可恶的……骑士！！！”末日公牛立即举起右臂抵挡，莱恩的战锤正中其手肘处，强壮如树干的手臂也经不起如此巨大的冲击，骨头碎裂的脆响证明莱恩的攻击颇有收获，末日公牛的大砍刀掉在了地上，它的右手直接垂下，伤口处喷出小股黑红色的鲜血，露出了森森白骨。
<br/>
<br/>    “呃呃呃啊啊啊啊啊！人类！我一定要杀了你！！！”末日公牛突然扬起自己的铁蹄，莱恩心知良机已失，立即躲开，塔乌斯趁势重新站起，它突然伸出血盆大口将自己的右小臂的肌肉直接咬下，将撕扯开的大块血肉扔在一边，在莱恩惊骇的目光中，末日公牛的右边断臂中的骨头疯狂伸长，最终长成一把锋利的骨刃，闪烁着可怕的寒光：“以吾主……恐虐之名！”
<br/>
<br/>    “为了尤里克！！！”莱恩咽下自己喉咙口的鲜血，再次挥锤和塔乌斯战成一团。
<br/>
<br/>    斯卡维尔山的高山上，无人知晓的血战扔在继续。
<br/>
<br/>    不，并不是无人知晓，在黑暗的虚空之中，有双眼睛正在注视着这一切，他的外貌是一个高大的人类战士模样，身披灰色的狼皮，身上有多处狼的头骨装饰在他的全身板甲上。
<br/>
<br/>    他正是白狼战神尤里克，帝国北方隆冬的白色火焰。
<br/>
<br/>    白狼之主正紧张地注视着白狼骑士和末日公牛决斗的身影，火焰和雷电在一人一兽的周围不断地炸响，随着战斗的进行，塔乌斯的精神越发疯狂，血神恐虐的力量在它的身上越发污秽和腥膻。
<br/>
<br/>    只是莱恩也丝毫不落下风，寻常人类的体能极限早已被他超越而去，白狼骑士战锤横扫，侧面刺来的骨刃被战锤挡住，正面攻击的战斧也为莱恩强大的腕力弹开，这场可怕的战斗暂时还分不出输赢，又或许输赢仅仅处于旦夕之间。
<br/>
<br/>    身后有金色的光影出现，尤里克头也没有回，只是很不客气地说道：“啊，这是鲜花和清泉的气味，还有青草的芬芳，我以为你只会躲在那该死的湖泊里面，唬骗着你的骑士们去对付敌人。”
<br/>
<br/>    回答他的声音充满着清冷和淡然：“我的骑士死在了这头邪物的手下，这个年轻人呼唤着我的名字和它决斗，我没有理由不出现。”
<br/>
<br/>    “也是。”
<br/>
<br/>    山峰上的战斗进入白热化，莱恩的脸上汗水和污渍糊成一片，他的战锤上从金色的雷光闪烁变成了旋风围绕，在这种情况下，白狼骑士的挥锤的动作变得非常迅捷，他不断地出锤，每一击都朝着塔乌斯的脆弱之处攻去，末日公牛也不甘示弱，它闪过莱恩的冲击，大号的战斧直冲莱恩的面门而来。
<br/>
<br/>    狮鹫锤头再次和战斧交锋。
<br/>
<br/>    “你失败了，白狼骑士！”塔乌斯巨大的牛首猛地前伸，想要咬掉莱恩的脖颈，白狼骑士早已感觉不妙，他连忙顶开末日公牛庞大的身躯，可是末日公牛的攻击并未停止，它扔掉了手中的战斧抓住莱恩的战锤，另一只手臂上长长的骨刃直接刺下。
<br/>
<br/>    灵能护盾抵挡了不到一秒就在骨刃面前破碎，深深地扎入莱恩的右肩中，鲜血染红了他的链甲和里面的衬衣，莱恩也扔掉了战锤，他左手牢牢地抓住插在肩膀上的骨刃，右手手腕处在他的意志下伸出了长长的灵能利刃，将骨刃齐根而断。
<br/>
<br/>    “吼啊啊啊啊！”剧痛让塔乌斯的身体不住地往后退，莱恩将肩膀上的骨刃用力拔出，完全不管自己血流不止，他右手的灵能利刃刺穿了末日公牛的膝盖，末日公牛则是一拳击打在他的胸口，莱恩的胸膛在巨力打击中陷了下去，大口大口的鲜血从他的嘴里留下，白狼骑士将手伸向天空，落在他身后的战锤以极快的速度回到他的手中。
<br/>
<br/>    抄起战锤，牟足所有力量，白狼骑士发出了歇斯底里的大吼：“失败的是你！”
<br/>
<br/>    碎颅击，这是莱恩最强的战技之一，战锤上爆射出金色的弧光裹挟着无可匹敌的巨力，正中塔乌斯的脑袋，野兽人的头骨被莱恩的巨力打碎，白色的脑浆和红色的血液流了下来。
<br/>
<br/>    末日公牛惨嚎着后退，他的嘴巴里面咸咸的脑浆味道告诉它这一切都是真的！
<br/>
<br/>    “人类……你很强……你的颅骨，我要定了！”塔乌斯突然撕烂了自己的胸膛，然后做出了一个诡异的礼节。
<br/>
<br/>    “不好，它这是在向血神祈求力量！”莱恩想要起身却发现自己实在是站不起来，他的头脑因为失血过多而传来阵阵晕眩感，肩膀的伤口还在淌血，莱恩赶紧将灵能转化为绿色的治愈能量，治疗着自己的伤口，同时抓紧时间恢复自己因为使出碎颅击而处于低谷的体力。
<br/>
<br/>    虚空之中，似乎有什么东西在回应着末日公牛的祈祷，可怕的血色能量被灌注进入它的身体，更多的异变。
<br/>
<br/>    它的伤势在疯狂增生的皮肤和兽角中痊愈了，更可怕的是，它的后背上长出了新的两条手臂，每一条手臂的前端都是巨大的骨刃，它属于自己的意识越来越少，逐渐向混沌的造物越来越近，显然之前的血腥战斗取悦了邪神，这是邪神全新的赐予。
<br/>
<br/>    这是传说中的四臂牛魔！莱恩马上就认出了这种传说中的怪物。
<br/>
<br/>    这源自于大陆上的恐怖传说，这种可怕的生物曾经在一夜之间屠尽数个村落，没有任何人幸存，它的每一次出现都需要用无数的士兵和骑士的生命去填。
<br/>
<br/>    有一个很有名的故事，现任帝国皇帝卡尔-弗朗茨-弗雷德里希在年幼时去血松林中打猎，就遇到了可怕的四臂牛魔，如果不是突然出现的木精灵军队救了他，或许现在的帝国皇帝就是另一个人了。
<br/>
<br/>    “天啊，吾主可没有告诉我会遇到这种情况。”莱恩喃喃自语，他现在情况很糟糕，他的内脏已经多处破裂，肩膀有伤，体力也正处于低谷，他面对卷土重来的塔乌斯处于下风，勉强地抵御着，不断地后退。
<br/>
<br/>    直至退无可退。
<br/>
<br/>    塔乌斯再次发动了野蛮冲撞，这次莱恩也无法抵挡，他整个人被撞飞数十米，锋利的骨刃割穿了他身上的链甲，内里的衬衣被鲜血浸染，莱恩痛苦地呻吟着，英俊的脸蛋皱成一团，战锤掉在了距他十几米的地方，他的身体掉在地上后滑出两米有余，留下大片血迹。
<br/>
<br/>    “唔~好冷~”勉强回复了意识之后，莱恩发现自己正身处冰面之上。
<br/>
<br/>    对啊，这里是……河流，这条河穿过山上的堡垒，流到山下。
<br/>
<br/>    “你失败了……真是可悲……你的伪神并不会怜悯你……他只会嘲笑你的弱小……和懦弱……哈哈哈哈哈！”身型如山峰般的塔乌斯渐渐靠近了，它踩在冰河之上，一步一步朝着莱恩靠近。
<br/>
<br/>    就好像，末日的来临。
<br/>
<br/>    在冰冷黑暗的虚空之中。
<br/>
<br/>    “你不帮助他么？”清冷的声音带着丝丝不解：“他已经为你战斗到如此地步了。”
<br/>
<br/>    “不，弱者，是不值得同情的，如果他死在这里，那就是他的宿命。”尤里克脸色平淡地摇头：“这头末日公牛居然能进化成四臂牛魔确实令我感到意外，但是弱者就是弱者，我没有义务为弱者的弱小买单。”
<br/>
<br/>    “是你的神谕强迫这个年轻人去面对末日公牛，现在你看着他被远胜于他的敌人打倒却毫无反应？”清冷的声音中微微带着怒火：“那么我将~”
<br/>
<br/>    “请停下你的动作，外来者，要知道，那个小子是我的人，你忘了么？你并没有权力干涉这件事，无论他是死是活，这都是我的事！和你没有关系。”尤里克阻止了神秘来客的进一步举动。

<br/>    是啊，我是父亲的儿子。
<br/>
<br/>    我带着巨大的使命来到这里。
<br/>
<br/>    还有无数生灵等待着我去拯救，我还有重要的任务还没有完成。
<br/>
<br/>    我怎么能在这里倒下？！

<br/>
<br/>    我要如何才能击败眼前的强敌呢？
<br/>
<br/>    等等，这是？
<br/>
<br/>    莱恩注意到，被冰封的河面在四臂牛魔的铁蹄践踏下，出现了丝丝裂纹。
