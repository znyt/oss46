　　泡面之所以让人觉得美味，觉得好吃，其实就是泡面调味包的缘故。
<br/>　　一些想要吃到泡面的美味，但是又不愿意吃到不健康的油炸方便面，会选择将泡面里的调料包拿出来放入煮好的挂面中。但是慕容飞并不打算这么做，泡面调料包也是各种食品添加剂弄出来的好吗？慕容飞可不会让现在的佟千雪去吃这么多食品添加剂。
<br/>　　黄虹买来的菜很多，加上之前冰箱里还有剩下的，要弄出一碗能把佟千雪从卧室里吸引出来的面条，简直就是轻而易举。
<br/>　　一阵阵诱人的香味从厨房飘出，飘进卧室中，原本闻到食物就会感到恶心想要呕吐的佟千雪，这次在闻到飘进卧室的面香后竟然完全没有感到恶心，甚至还因为之前把胃吐空了的关系，竟是感觉到了饥饿感。
<br/>　　不过佟千雪却是没有立刻起身去客厅，等着慕容飞把面碗端上来，而是躺在床上，用拳头捶打着自己抱在手里的玩偶熊，并且把这个玩偶熊当成了慕容飞。
<br/>　　“坏家伙！坏家伙！坏家伙！”佟千雪捶打着玩偶熊，嘀咕着骂道：“让你不给吃方便面！让你不给我吃方便面！我就要吃泡面！就要吃！”
<br/>　　很快，慕容飞就把煮好的面条端出了厨房，放在餐桌上，对还赖在卧室里的佟千雪说道：“赶紧出来吃面，不然一会儿糊了可就不好吃了。”
<br/>　　可是卧室中的佟千雪并没有要出来的意思，慕容飞等了两分钟，又说道：“你不出来吃，那我可就自己吃了，这么好吃的面如果糊了那就太可惜了。”
<br/>　　就在慕容飞拿起筷子准备去夹碗里的面条时，一只白皙的手就把他手里的筷子给抢走了，并且把放在他面前的面碗也一并拿走了。
<br/>　　佟千雪拿着筷子坐在慕容飞对面，说道：“这是给我煮的面，我又没同意你吃，谁让你动手的？”
<br/>　　“你不是不要吃吗？那就我吃了。”慕容飞回答道。
<br/>　　“谁、谁说我不要吃的？”佟千雪嘴硬着说道：“我就是看看你态度诚恳不诚恳，现在看来你的态度一点都不诚恳，差评！”说完，用筷子夹了一筷子面条送进嘴里。
<br/>　　佟千雪漂亮的眼睛弯成两道月牙，显然是对这碗面条的味道非常满意。不多一会儿，一碗面条就被佟千雪消灭的干干净净就连面汤都没剩下，就佟千雪现在这个胃口，很难让人把她跟有了一个多月身孕，因为孕吐而闻不得任何食物味道的孕妇联系到一起。
<br/>　　放下面碗，佟千雪心满意足的打了一个饱嗝，对着慕容飞指挥道：“去把碗洗了，今晚的事情就原谅你了。”
<br/>　　“遵命！”
<br/>　　慕容飞拿起桌子上的面碗和筷子进了厨房，整理去了。等到慕容飞回到卧室的时候，佟千雪已经重新刷了牙又躺在床上了，吃饱了就容易犯困，慕容飞躺下的时候佟千雪已经是哈欠连连，显然是因为要等慕容飞才勉强着让自己没有睡着。
<br/>　　“坏家伙！”
<br/>　　佟千雪转过身，伸出手指戳着慕容飞的胸口，说道：“如果每天能吃到你刚才做得那种面条，我想我也不会吐得那么死去活来了，完全没有胃口了。”
<br/>　　慕容飞喉结滚动了一下，想要说什么，但最终还是没有能把想说的话说出来。
<br/>　　佟千雪伸手抱住慕容飞，靠在他怀里，说道：“我知道你想说什么，我知道你不可能就这样离开部队的，我就是跟你开个玩笑，你有你的责任和使命。”
<br/>　　很快，佟千雪就在慕容飞怀里睡着过去。慕容飞抱着睡着的佟千雪心中感慨良多，龙隐部队作为总参直属的特种作战部队自然是有着很多其他部队所没有特权，比如他们可以根据自己的需求，选用目前世界上最先进的轻武器，仅仅是这一点就是绝大多数部队甚至是特战大队所羡慕的，当然，跟这些特权相对应的就是责任和使命，他们需要随时做好准备在任何时间，以任何方式投入到任何情况的行动中，在行动中他们可能会受伤，也可能会牺牲，埋骨他乡。
<br/>　　从踏出国门的那一刻起，就已经是一个死人！这就是最能反应龙隐部队队员在执行任务时情况的一句话。
<br/>　　当慕容飞沉浸在这短暂但却温馨的时光中，在潮湿闷热的南美洲却是另外一种情况。
<br/>　　……
<br/>　　南美洲，“Cocaine”王国。
<br/>　　这是一个处于高海拔，并且被包围在一片丛林中村庄。
<br/>　　这里原本是个平和，靠务农为生的村庄，以种植高原小米、棉花、咖啡为主要经济收入，可是自从布克尔特来到这里，小村的平静就被彻底打破了。
<br/>　　原来用来种植高原小米、棉花、咖啡的农田全都被用来种植cocaine，原来居住在这里的村民，要么服从布克尔特的安排为他服务，要么就被赶离这里，离开自己的家园。
<br/>　　村庄内的道路上，还有楼房房顶上，随处可见背着AK，M16来回走动的武装毒贩，用沙袋堆起来的射击掩体，经过改装加装了重机枪和防弹钢板的皮卡。
<br/>　　这个村庄在布克尔特手中，已经完全变成了一个军事化管理的制毒贩毒的毒窝。
<br/>　　一栋装修豪华的楼房顶楼房间，房间周围窗户的百叶窗紧闭，隔绝了外面的光线，房间内唯一的光亮来自于桌子上那台笔记本电脑。
<br/>　　布克尔特坐在椅子上，双手交叉着支在下巴下，盯着笔记本电脑上的资料，在笔记本电脑旁边，还放着一把银色的沙漠之鹰手枪。
<br/>　　这种大口径、大威力、大后座力的手枪，很受人欢迎，很多人都把它当做力量的象征。
<br/>　　布克尔特这把银色沙漠之鹰跟西伦科特在货轮上使用的那把银色沙漠之鹰一模一样，这两把枪从一开始就是一对。
<br/>　　笔记本的USB接口中插着那个从货轮船长办公室找到带回来的U盘，回来之后，布克尔特就把自己关在这个房间中，谁也不见，也不让人靠近这个房间，只是看着U盘里的内容。
