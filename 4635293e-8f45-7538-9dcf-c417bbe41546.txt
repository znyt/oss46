<br/>    在四个亲传弟子的服侍下舒舒服服的享受了一番，卓青瑶询问道：“师父，我们也想帮你，我知道我们现在很弱，但是，请让我们尽一份力。”
<br/>    
<br/>        陈明思虑片刻，周围这些大多都只是一些小势力妖王，妖月王庭要是知道了妖月陨落的消息，恐怕直接就乱成一锅粥了，不足为虑。
<br/>    
<br/>        这一次剩下的战斗，倒不是特别的危险，也是时候让这三千多弟子还有亲传弟子，出去历练历练了，本来陈明还要想办法在哪儿去找三百妖王完成自己的进阶任务，杀这些小妖王吧，名声太差，估计燕山在世人眼中的形象就是一个暴君统治，找其他有三百妖王的势力吧，至少都是大妖王，而且还是超级强大的大妖王，惹不起。
<br/>    
<br/>        现在倒是好，直接送上门来了，第二天一早，陈明睡了个饱。黑虎，火燕，黑蟒知道陈明归来的消息，连忙赶到燕山，向陈明禀报情况。
<br/>    
<br/>        黑蟒道：“禀报山主，这一次向着燕山宣战的妖王，足足有九十三位，妖月王庭也在进军！”
<br/>    
<br/>        黑虎扫了陈明一眼，小心询问道：“山主，不知道那妖月，到底怎么样了？”
<br/>    
<br/>        他们想要知道妖月的消息，毕竟像是妖月和陈明这般实力的，都有着影响整个战局的作用，而且若是妖月还在，对方的士气绝对是气焰滔天！
<br/>    
<br/>        陈明没有犹豫片刻，直接开口道：“放心吧，妖月，回不来了。”
<br/>    
<br/>        毕竟任务还没有完成，这只能说明一点，妖月还没死，妖月若是死了，任务早就结束了。
<br/>    
<br/>        陈明只是说妖月回不来了，并没有说妖月已经死了，但是在黑虎，黑蟒，火燕三人的耳中，这句话就只有一个意思，那就是妖月已死。听银翼说，妖月可是带着十多位妖王一起去追杀的陈明，但是，最后却是只有陈明一人回来了，山主的实力，也未免太恐怖了吧。
<br/>    
<br/>        瞄了一眼自己的师德，已经又有一万多了，主要是李素衣突破到入道境之后突飞猛进，他毕竟只要炼化饕餮之种便可以突破，凌仙的境界倒是比不上李素衣，从李素衣的口中，陈明知道，凌仙要到了悟道境之后，实力才会进入一个快速增长期。
<br/>    
<br/>        而卓青瑶，得到了那一条木船，加上卓青瑶只修闻道境，燕山又是闻道境的丹药随便吃，她的实力，可谓是突破很快的，现在已经快要达到王级的实力了。
<br/>    
<br/>        现在的卓青瑶，已经是闻道境三十六层，和陈明并驾齐驱。
<br/>    
<br/>        陈明倒是觉得没什么，估摸着自己门下的弟子都已经超脱成大君了，自己还是悟道境，而现在，自己还是入道境，不过境界和陈明的实力关系并不大，陈明也不怎么在意。
<br/>    
<br/>        陈明不太敢想象自己超脱的时候会是什么局面，“我若超脱，群仙束手，应该是这样吧。”
<br/>    
<br/>        而且，现在陈明即便是完成了进阶任务，恐怕也不敢突破到悟道境，入道境突破到悟道境可是有着天劫的，而自己的天劫，鬼知道会有多猛！
<br/>    
<br/>        嗯，鬼也不一定知道。
<br/>    
<br/>        陈明回想起白骨饕餮的经文，在心中道：“顿悟白骨饕餮！”
<br/>    
<br/>        片刻之后，陈明已经彻底明白了《白骨饕餮》的修行之法，融合对道经这万经之首，对道的理解，陈明开始对《白骨饕餮》做一些改进，顿悟这《白骨饕餮》，也是为了提升自己麾下妖王的实力，为什么不直接劝降其他神府境的妖王？
<br/>    
<br/>        看看银翼这种劝降的什么德行？
<br/>    
<br/>        自己不过出去失踪不久，银翼就已经闹着要散伙了，而黑虎，黑蟒，火燕三人却是在仔仔细细调整燕山的布防，准备死守燕山。
<br/>    
<br/>        陈明看向身前的三位妖王，认真道：“接下来，我所说的话，不能被任何人知道。”
<br/>    
<br/>        三位妖王都是露出郑重之色，拱手道：“我等遵命！”
<br/>    
<br/>        陈明深吸一口气，终于还是要把白骨饕餮这一本邪功放出来了，“接下来我要讲解的这本经文，叫做《白骨饕餮》，我的实力突飞猛进，并不是因为这一本功法，但是这一本功法，我确实修行了，而且修行到了大圆满的地步。所以现在，我将这门功法改进之后的第一卷传给你们，然后将我修出的大圆满饕餮之种分出一半给你们三人，有着这一半饕餮之种，你们就不需要去吞食凡人造孽了。”
<br/>    
<br/>        三位妖王在陈明面前恭敬跪下，“我等，谢山主大恩！”
<br/>    
<br/>        陈明挥手，示意他们起来，“你们三人，都是我燕山最忠诚之人，不像银翼那个家伙，不必见外。”
<br/>    
<br/>        道完，陈明开始讲解他修改之后的《白骨饕餮》，“饕餮者，吞噬之主也，又名暴食，而饕餮，最后也死在了暴食的路上，将自己撑死了，所以，你们要记住，要能抑制住心中的暴食之念，每一个月，都要到我面前让我查看，以免魔念横生。”
<br/>    
<br/>        三位妖王也是知道这其中的恐怖之处，都不想成为一具只知道吃的行尸走肉，“我等记住了。”
<br/>    
<br/>        陈明开始讲述经文，“灵者魂也，肉者，身也，饕餮者，吸收肉之精华，以补自身。天之道，损有余而不不足，饕餮之道，损不足而补有余.......”
<br/>    
<br/>        将经文讲解完毕，陈明问道：“你们可曾记住了？”
<br/>    
<br/>        三位妖王毕竟是妖王境界，讲一遍自然就记住了，而且作为妖王，哪里需要其他什么讲解，很快便懂了，陈明顿悟了《白骨饕餮》，已经是有了饕餮之种，而且比青狮的饕餮之种更为强大，将自己的饕餮之种分出一半，然后又分成了三份，三枚绿色的道种浮现在三位妖王身前，“吞下去吧，记住了，目前传给你们的经文，最多只能修行到神府境后期，若是贪图力量妄进，只有死路一条。”
<br/>    
<br/>        三位妖王吞下饕餮之种，再次拱手道：“多谢山主授道大恩！”
<br/>    
<br/>        让三位妖王自行下去修行，陈明也是开始做战前的准备，经过这一战，十万大山之中，燕山就要真正的威临四方了，所以这一战，一定要打得漂亮！
