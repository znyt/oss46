　　第67章
<br/>　　廖甲没有废话，极快的时间又扑向另外一个人，抓住对方胳膊，硬挨对方一拳，扑过去紧紧抱住。
<br/>　　两个粗糙大汉相拥的姿态总得来说有些诡异，廖乙的刀已经到了，他抓住对方无法动弹的手，照着手腕切过去。
<br/>　　好快的刀！
<br/>　　一支血手掉在地上，然后是惨叫，似乎想掀破房顶。
<br/>　　廖家和廖乙的本事其实不如何，而这种街头斗殴，比的并不是本事，而是狠。廖甲和廖乙是两个山贼，而山贼最不缺乏的就是狠。
<br/>　　又合力扑向另外一个人，将对方撞到，廖甲把对方死死摁在地上，由廖乙下刀，刀顺着脸颊滑下来，切下一只耳朵。
<br/>　　然后目光看向另外一个人，对方已经被吓倒在地上，脸色苍白，张大嘴巴喘着粗气。
<br/>　　廖乙抓住酒楼顶门的杠子，照着对方脑袋砍下来，对方昏迷。
<br/>　　然后，两兄弟又干脆利落得把四个人抛出门外，再没有多看一眼。
<br/>　　死寂！
<br/>　　苏樱也被唬住，她虽然曾被抢到蛤蟆寨，但总的来说，程大雷还是很文明的。至于之后的传言如何如何，她也不往心里去。因此，她一直对山寨存在某种误解。
<br/>　　山贼，可是真狠呐，全程没有一句话，四个人已经废了。
<br/>　　“大嫂，我们兄弟来迟了！”
<br/>　　苏樱一张脸腾地红了，这都哪儿跟哪儿，她愣了半晌，才略微明白一些。
<br/>　　“他让你们来的。”
<br/>　　“是大当家让我们来保护大嫂，有什么事，大嫂您尽管吩咐。”
<br/>　　苏樱脸上又红了红：“这话不要再提了，这里不方便说这些。”
<br/>　　小蝶走到苏樱面前，道：“小姐，是他派的人，他还给我们带了钱，足足有一百两呢。”
<br/>　　苏樱一颗心落了踏实，她最近这几天担惊受怕，周围痞子混混的行为越来越出格，如果不是程大雷派人来，自己怕还真不知该如何是好。
<br/>　　“把东西先搬进来，其他事慢慢再说。”
<br/>　　酒楼上板之后，廖甲和廖乙把马车从后门牵到院子里，等苏樱和小蝶回房后，小蝶才把那封信拿出来。
<br/>　　“小姐，他还给你写了一封信。”
<br/>　　苏樱将信拆开，还没看清楚上面内容，表情先顿住：“女人的笔迹……”
<br/>　　“呃……小姐的直觉，还真是敏锐呐。”
<br/>　　“怎么会事？”苏樱先将信扣住，盯着小蝶，心中有几分不太舒服的感觉。
<br/>　　“是……柳家小姐的啦。”
<br/>　　无奈，小蝶把程大雷出卖了，但她也搞不清楚具体怎么会事，只是说看到炕上两床被褥。
<br/>　　其实，如果苏樱真知道柳芷做了程大雷的奴婢，却也不是什么大不了的事情。大户人家的公子，十几岁就有陪房的丫头，就像贾宝玉和袭人有了风雨之事，连爱吃醋的林妹妹都没往心里去。
<br/>　　总之，奴婢在很多人眼里，是不被当做人看的。
<br/>　　可程大雷如此藏着掖着，本来不是事情的事情，就变得有问题了。
<br/>　　苏樱冷哼一声，嘴里冒出两个字：“呵，男人……”
<br/>　　她这才把信拿起，一字一字的看上面的内容，信的主要内容，就是程大雷所指导的酒店经营内容。
<br/>　　有些在苏樱看来，简直就是异想天开，她也估摸不出是好，还是不好，看上去是有几分道理的。
<br/>　　不过，有程大雷送来的一百两银子，酒楼就可以重新装潢一番，现在的酒楼，真的是破得不能再破了。而更让苏樱欣喜的，还是程大雷送来的辣椒。
<br/>　　之前她把辣椒带回家中，放在四海楼试着卖一下，没想到，一拿出去后，反响极好。只不过数量太小，现在已经成了专供大户人家的奢侈品。
<br/>　　也就是靠着这一点，四海楼打响了名气，你如果请人吃饭不在四海楼，不点一份辣椒煮的菜，就显得你身份不够。
<br/>　　而这次程大雷送来的辣椒，一共有两麻袋，而据小蝶所说，蛤蟆寨有上万斤的辣椒。
<br/>　　“只是……”苏樱目光落在纸上，眉头皱得更紧：“什么叫做……湖底捞……”
<br/>　　目前要做的，还是按照程大雷所说的样式，把酒楼先装修起来吧。
<br/>　　……
<br/>　　每天上午，孙九就会换上自己的制服，意气风发的指挥一群妇人做饭。
<br/>　　他腿上的伤因为及时治疗的原因，竟没有留下后遗症。
<br/>　　他被评为丙等，之前觉得这个等级不高，但在这次加入蛤蟆寨的所有人中，也不过三四个人。
<br/>　　丁等要和所有人挤在七八个人一间的屋子中，不管曾经在柳家是什么身份，现在都是一样。而自己却可以住两人间，对了自己的舍友是孟子云。
<br/>　　之前孟子云眼睛长在额头上，可是从不会把自己看在眼里，但现在却要和自己挤一个屋子，想到这里，心情总有些莫名的愉悦。
<br/>　　在这里，是没有什么仆人主人划分的，大家一视同仁，自己这个之前伺候人的下等人，也能昂起头说话。
<br/>　　自然，每天的训练是很累的，不过，自己是厨子，训练额度相对少些，而自己从小就干活，也不怕出力。
<br/>　　关键是这里能吃饱，有新衣服穿，每天开饭的时候，其他人都刻意和自己套近乎，想要自己能多打一些饭，那种尊重的眼神却是从前没见过的。
<br/>　　总之，孙九是很满意现在的生活。
<br/>　　而且，这几天大当家总是单独找自己，聊一些自己之前没有听过的事情，比如宫保鸡丁，鱼香肉丝，火锅之类的。
<br/>　　关键是，自己可从未听说过宫保鸡丁和鱼香肉丝，对了，火锅是知道的，不过大当家说火锅是辣的，这就是不懂了，火锅哪里有辣的。
<br/>　　大概大当家也不懂吧，当然，这一点自己也就是心里想想，可不敢说出来。
<br/>　　今天大当家又找自己去，也不知道怎么会事，不过，这总是器重自己的表现吧。
<br/>　　在大厅见到了程大雷，孙九微垂着头：“大当家，你找我？”
<br/>　　“嗯呐。”程大雷打了个哈欠，停下在纸上写写画画的笔，抬起头问道：“孙九，你听说过有种东西，叫做……面包么？”
<br/>(本章完)
