        最有趣的一幕是，当那个又高又壮的男生抢到球后，在明司寒面前做了一个假动作，谁知这假动作被明司寒一眼识破。
<br/>
<br/>        男生直接将篮球扣入明司寒的手中，傻乎乎地将球拱手相送，惹得场外男男女女一阵嗤笑。
<br/>
<br/>        而明司寒此刻就站在三分线外，双手抱稳篮球后腾空起跳，将手中的篮球用力推了出去。
<br/>
<br/>        篮球在半空中划出一个优美的弧度轨迹，“哐当”一声，落入球框之中，就在这一刻，结束的哨声突兀地响起。
<br/>
<br/>        最后一个完美的三分球，33比2，明司寒他们赢了，在场所有人不约而同地鼓起掌，纷纷叫好。
<br/>
<br/>        明司寒抹了把额头上的汗珠，本能地回头向连星儿那边望去。
<br/>
<br/>        他的薄唇微微勾起，朝她意味深长地笑了笑。
<br/>
<br/>        连星儿怔怔地看出了神。
<br/>
<br/>        下一秒，身旁的女生们蜂拥而上，瞬间将明司寒、张子晋，邢乐游和仲正浩给包围。
<br/>
<br/>        递水的递水，送纸巾的送纸巾，只差没有要签名了。
<br/>
<br/>        连星儿回过神来后，下意识地摸了摸自己的额头。
<br/>
<br/>        “星儿，你不舒服吗？”顾千凝见状，关心地问道。
<br/>
<br/>        连星儿立马摇头，赧然一笑，掩饰自己的心慌意乱。
<br/>
<br/>        而就在这个时候，明司寒却从女生堆里挤了出来，直接走到连星儿的面前，问道：“你有没有带水？”
<br/>
<br/>        “诶？”连星儿起初还在跟顾千凝搭话，压根就没注意到明司寒是何时走过来的，所以被他这么仓促的一句问话，本能地接了一句，“那现在就去买。”
<br/>
<br/>        听着她这傻乎乎的冒出这句，明司寒忍俊不禁起来，然后直接抓起她的手腕，将她从顾千凝的身旁拉到了自己的跟前。
<br/>
<br/>        “我跟你一起去。”明司寒咧嘴一笑。
<br/>
<br/>        顾千凝连忙推手表示：“你两快去快去！顺便帮我买一杯珍珠奶茶回来！”
<br/>
<br/>        “我要一杯柠檬茶！”赵安彤立马凑上来，笑嘻嘻地说。
<br/>
<br/>        路娅玟也跟着点单：“我要抹茶芝士奶盖。”
<br/>
<br/>        “……”连星儿看着这三个没事凑热闹的室友，只能傻笑着被明司寒给牵走了。
<br/>
<br/>        不对，是抓走了。
<br/>
<br/>        明司寒没有牵她的手，而只是抓着她的手腕，拉着她往校园内那条小吃街走去。
<br/>
<br/>        众目睽睽之下，就算是不知情的人，看到这一幕，也能明白他俩是什么关系了。
<br/>
<br/>        问题是，他抓着她的手腕也就算了，还偏偏要走在前头。
<br/>
<br/>        连星儿觉得自己像只宠物，可怜兮兮地跟在他身后。
<br/>
<br/>        明司寒的背影比较修长，身上的白色背心球服已经湿透，双臂的肌肉饱满，上面沾满了汗珠，看上去十分禁欲。
<br/>
<br/>        呃——
<br/>
<br/>        她怎么突然莫名其妙地觉得明司寒好帅了。
<br/>
<br/>        就在他俩买完饮品从奶茶店里出来后，走到一处人少的巷口时，一行人突然围了上来。
<br/>
<br/>        不仔细看，连星儿还真没发现，围着他们的这四个人是刚刚跟明司寒比篮球的那四个男生。
<br/>
<br/>        “明司寒，在学校里，他们怕你，我可不怕你。今天给你脸，陪你打篮球，你居然好意思赢我？”壮男咧着嘴，满脸横肉。
<br/>
<br/>        “你拦着我们是几个意思？”明司寒淡漠地问。
<br/>
<br/>        壮男气势汹汹地说道：“看你不顺眼，想揍你。”
<br/>
<br/>        明司寒有些头疼地挠了挠额头，睐了一眼身旁的连星儿，问道：“你是学过防身术的吧？”
<br/>
<br/>        “嗯？”连星儿讷讷地点头。
<br/>
<br/>        下一秒，明司寒直接将连星儿抱了起来，喊了声：“用脚踢！”
<br/>
<br/>        “？！”连星儿一脸懵，虽然还未弄清楚是什么状况，但是她的身体很配合他的命令，直接一脚踢向那个壮男。
<br/>
<br/>        但是，人家也不是摆看的，敏锐地往后一退，躲过了这一脚。
<br/>
<br/>        壮男身后的三个小跟班见状，立即一窝蜂地冲了上来，
<br/>
<br/>        明司寒再次抓住连星儿的手腕，将她往自己身后一拉，一脚踹在了冲锋在最前头的男生的肚子上。
<br/>
<br/>        随后上来的两个男生不知从哪儿抽出两根银晃晃的棍子，直接向明司寒挥了过来。
<br/>
<br/>        这四打二，还带“武器”，肯定是打不赢了。
<br/>
<br/>        连星儿二话不说，直接牵着明司寒的手，撒腿就跑。
<br/>
<br/>        明司寒还未打过瘾，就被这丫头拉着临阵逃脱，也是一头雾水。
<br/>
<br/>        更重要的一点是，这丫头还跑反了方向，往后跑可不是回学校的路。
<br/>
<br/>        连星儿拉着明司寒往人多的地方跑，直到把那四个男生甩了后，她才气喘吁吁地停下脚步。
<br/>
<br/>        而实则上，明司寒倘若没看错的话，那四个男生是被四个黑衣保镖给拽走的。
