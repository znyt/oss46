        &lt;h3&gt;第三卷第一百九十四章  米粒之珠也现芳华&lt;/h3&gt;
<br/>
<br/>        呼延寰宇、乔小东及李凌菲约到了一起，他们在工作中经常有交集，时间长了就处了朋友，李凌菲调到北京上班，回来的日子不多，大家约好了一起吃顿饭。
<br/>
<br/>        无论岁月如何变迁，有些人之间的友谊是不会变的，这个世界上什么最可贵，是友情，那一些永远存在于人们心中的友情，无关金钱无关地位，无关高矮胖瘦，无关种族性别。
<br/>
<br/>        呼延寰宇准备等耗材推向市场之后和陈子昂一起去甸城，陈子昂已经联系了甸城那边的有关人员，他们过去没有个三年五载是不会轻易回来的，要回来也只能是过年过节，肯定会匆匆而来，匆匆而去，到时候想要和乔小东聚在一起怕也是要看情况了。
<br/>
<br/>        过完年，他们约到了一起，李秀和谭九儿也到了。
<br/>
<br/>        李秀被江俞轩和呼延寰宇刺激了一次之后，大大方方开始追求谭九儿，你有情我有意，俩个人倒是琴瑟和谐，工作生活上都是很好的搭档。
<br/>
<br/>        五个人坐下来喝喝小酒聊聊天，讨论起了上班之后的工作情况。
<br/>
<br/>        乔小东笑着说：“我一个半老头，好不容易有个休假的机会，被你们这一帮年轻的工作狂拉出来，拉出来也就算了，还要不停的讨论工作，生活如此艰难，能不能让我乐呵乐呵了？九儿，来，给你蜀黍说一哈，上班之后你如何进行集采方面的工作？”
<br/>
<br/>        谭九儿笑了笑，她虽然接替李凌菲的工作时间不长，但是她聪明好学，对于政策上把控到位，况且有李秀在一起，工作上就更没有什么问题，俩个人配合得天衣无缝。
<br/>
<br/>        “今年的工作大致方向还是会跟着总公司走，今年除了要坚守阵地之外，最主要的还是配合总公司将耗材推向市场，至于年底入围集采供应商不是有蜀黍你嘛。”谭九儿甜甜的一笑，带上了乔小东。
<br/>
<br/>        乔小东：“去年你们丢掉了好几个项目，你们怎么想的？”
<br/>
<br/>        李秀：“此一时彼一时，去年丢掉的项目主要还是有人在里面动了手脚，我们的方案基本都被透露，比喻洛城的项目，我们的江总是遭受到了威胁。”
<br/>
<br/>        乔小东：“据说，你们陈总和洛城的项目负责人是同学？”
<br/>
<br/>        呼延寰宇：“具体情况我们也不是很清楚，但是那个项目确实是费了我们很多心血，8个人在那里驻守了半年。”
<br/>
<br/>        “邪不胜正，无论是生意人，还是我们这样的工作人员，只要行得端走得正，正义的一方总是偏向我们。”乔小东语气似乎是安慰，也似乎是坚定。“我从这个部门成立以来，一直都兢兢业业，坚持原则。这么多年以来，很少有营私舞弊现象发生，就算是专家有点倾向的意思，那也是看着方案做得好，有能力有实力的公司，说白了就是想把事情做好。”
<br/>
<br/>        李秀附和着：“是啊，我们公司要发展要生存，最重要的也是要把事情做好，这个社会人情是一方面，做事情更重要。”
<br/>
<br/>        乔小东：“洛城的项目我已经安排人调查了，不过有些事情不是我们份内的工作，将会交给有关部门处理，至于结果如何，我也不得而知，总归来说能让我和大家心安就好。”
<br/>
<br/>        “夫子不愧就是夫子，做事情总是会顾全大局，为大义所想。”谭九儿适时的补上一句。
<br/>
<br/>        李凌菲：“要说，洛城弃标的事情其实和我有一定的关系，当初夏颜违规做事，被我拒绝和举报，她心怀怨恨，一直想找机会打击我，我才辞职到了现在的公司，只可惜当初我的行动微乎其微，夏颜依旧是夏颜。”
<br/>
<br/>        李秀：“洛城弃标的事情不怪你，我们当初是坚决实施江总和陈总的命令，你就不要心存愧疚了，生意场上的事情不是你来就是我往，谁能一辈子高高在上？”
<br/>
<br/>        “哎呀，我们今天是好友聚会，怎么就讨论工作了？来，我敬乔蜀黍一杯，祝乔蜀黍越来越帅，愈来愈高。”谭九儿笑着举起酒杯。
<br/>
<br/>        “帅就不敢当了，我已经老了，满脸沧桑，一片焦黄，那个高嘛，我也不行了，人老了都会是向下发展。”乔小东哈哈哈大笑。
<br/>
<br/>        “大叔那里满脸焦黄了？白里透红，满面红光，至于那个高嘛，据说你已经升职了，啊，大叔你要罩着我啊。”谭九儿将手中的酒一饮而尽。
<br/>
<br/>        “九儿小小年纪，做事情是倒是经常让我刮目相看，我有时候再想，你们陈总在对用人方面倒是一个伯乐。”乔小东豪爽的喝下杯中酒。
<br/>
<br/>        “哈哈哈，在用人方面，我们陈总是不及江总的，陈总性子耿直，她只会闷着头向前冲冲冲，而江总，说话让人如沐春风，慧眼识珠，用人调度他是做到恰到好处。”李秀笑着夸奖江俞轩。
<br/>
<br/>        李凌菲：“如果陈总当初不用江总，谁会知道江总是一个伯乐呢？”
<br/>
<br/>        谭九儿：“说到底，还是陈总知人善用，不过是因为有了江总她就收起锋芒而已。”
<br/>
<br/>        呼延寰宇：“你们不觉得陈总和江总是绝配吗？唉，我从来都没有见过像他们如此默契的一对，一个眼神都能明白对方在想什么。”
<br/>
<br/>        乔小东：“他们俩夫妻还真像寰宇说的那样，我见过他们很多次，别看你们陈总一副女强人的样子，在你们江总面前就是小鸟依人。”
<br/>
<br/>        李秀和谭九儿他们惊掉了下巴似的：“夫妻？”不过貌似他们要是在一起也很好的，夫妻档！
<br/>
<br/>        乔小东倒是并没有在意俩个人诧异的神情，现在的人们流行“隐婚”，那天吃饭也就是几个人，都是他们自家人，所以在心里对江俞轩和陈子昂的感觉无形之中又亲近了一点。
<br/>
<br/>        乔小东：“当初啊，我可不知道你们江总还是江家大少，你们陈总给我介绍的时候我还奇怪，江家一个上市公司的少东家竟然在你们公司当副总，后来才知道是签了卖身契，哈哈哈。”
<br/>
<br/>        大家相视一笑，并没有揭穿乔小东的误会，连外人都看出来他们的感情很好，何况他们这些员工。
<br/>
<br/>        “凌菲什么时候去北京？”乔小东又问李凌菲。
<br/>
<br/>        李凌菲：“我们八号上班，我提前一天到。”
<br/>
<br/>        乔小东：“我记得你是想有出国的计划啊，现在不准备出去深造了？”
<br/>
<br/>        李凌菲：“我现在和我们公司另外的一个同事就是管理涉外事宜，后期可能会去国外发展，在拓展业务的同时我也想提升自己的能力。”
<br/>
<br/>        乔小东：“陈子昂这是想要大干了？还准备将产品推向国外？”
<br/>
<br/>        呼延寰宇：“她的心思不小呢，今年的甸城也将是她表演的舞台！”
<br/>
<br/>        乔小东：“你们江总也去吗？”
<br/>
<br/>        李秀：“江总现在是江家的人，他负责西城南郊的规划建设已经快到后期了，估计他是走不了的。”
<br/>
<br/>        乔小东：“你们这一帮年轻人啊，是社会前进的推动者，有你们这一帮生龙活虎的生力军，何愁我们不强盛？”
<br/>
<br/>        “夫子，说得你好像很老一样，你才是国之栋梁，因为有了你这样刚正不阿的人，社会秩序才会越来越好，而我们生意人也才能公平竞争。”谭九儿脆生生的话语响起。
<br/>
<br/>        乔小东举起手中的酒杯：“哈哈哈，我们都是，我们都是，因为有了我们这样微不足道的人们，构成了整个群体，用我们微薄的能量爱着我们的家、我们的国！米粒之珠也现芳华！来，愿我们的祖国更加强盛，愿我们永远幸福美满！”
