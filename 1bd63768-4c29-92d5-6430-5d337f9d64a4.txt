　　汪小跳，国服第一流浪，今天他也在看着叶帝他们的比赛，当看到叶帝拿出流浪这个英雄的时候，汪小跳第一反应就是，大魔王看过自己的直播，要不然的话他怎么敢拿这个冷门英雄打中路。
<br/>　　对于流浪这个英雄汪小跳还是非常的自信的，就算是对方是叶帝，他也有信心在这个英雄上与之一战，所以这一局在看到叶帝和蒋念对线的时候，汪小跳实际上是以一种挑剔的眼光去看的。
<br/>　　“嗯，这一波能够躲过对方两人的gank，意识还是不错的。”看着叶帝躲过了娜美两人的gank，汪小跳开口点评着。
<br/>　　不过当汪小跳看到叶帝回家做出的装备后，顿时懵圈了，随后便是有些气愤。
<br/>　　你说你玩个流浪，出个女神之泪有什么用，这玩意能打伤害么！
<br/>　　然而这还不是更让他气愤的，等到十分钟的时候，当看到叶帝再次回家做出的那件装备的时候，汪小跳却是彻底的无语了起来。
<br/>　　只见叶帝在做出了女神之泪后的第二件装备赫然是时光之杖的小件饭盒，这一次汪小跳终于是再也忍不住了，直接登录到了自己的微博账号。
<br/>　　“我去，大魔王在搞什么啊，流浪法师这个英雄这么玩，你上哪去打伤害！”
<br/>　　作为一个冷门英雄上到国服王者组，汪小跳的人气还是非常高的，所以他的这个微博刚一发布便是引来了无数粉丝的评论了。
<br/>　　当然了汪小跳也没有膨胀到去喷叶帝的地步，如今叶帝在英雄联盟中的地位无异于始祖级别的，如果真有人赶去喷他，恐怕不止那些粉丝会不乐意，就算是官方也会采取一些办法抵制对方吧。
<br/>　　能够进入名人堂的选手，这都是对英雄联盟做出过很大贡献的选手，他们自然是受到官方的围护的。
<br/>　　而汪小跳的这一条微博一经发布，直播间的弹幕上几乎是在第一时间便是被带起了节奏。
<br/>　　直播间的弹幕其实是个最有意思的地方，无论有什么大消息，弹幕都会第一时间暴露出来。
<br/>　　“国服第一流浪说大魔王的出装是错误的，这么打没伤害。”
<br/>　　“大魔王这次真的是玩大了，连国服第一流浪都出来打脸了。”
<br/>　　“我之前还以为大魔王拿出这个英雄是因为他看汪小跳的直播了呢，现在看来不是那么回事，要不然他绝对不会这么出装的。”
<br/>　　“不是说流浪的伤害会根据蓝量来提升的么，大魔王这么出装应该没什么问题吧。”
<br/>　　“蓝量才能够提升多少伤害，大魔王的想法真的是太天真了。”
<br/>　　“大魔王这么出装，中期就是根本没想着打团，他这是一心想打后期，这样玩真的是太极端了。”
<br/>　　.......
<br/>　　汪小跳的微博一发，直播间里的无数观众便是跟着开始议论了起来，原本还对叶帝满心期待的众人，这一刻也都是有些忐忑了起来。
<br/>　　场上的叶帝自然是不知道外面发生的这一幕，不过估计他若是知道，此刻肯定会更加的无语。
<br/>　　一个英雄的错误出装成为被大家认可的东西，而自己这个正宗的无敌出装却被大家看作是异类，这简直就是滑天下之大稽了。
<br/>　　流浪这个英雄主要就是靠蓝量对输出，而英雄联盟中许多加蓝的装备又是肉装，也正是如此才成就了流浪法师这个法坦的存在。
<br/>　　等到了中期的时候自己的女神之泪也差不多叠了四五百点蓝量了，加上饭盒为自己带来的血量和蓝量，到了那个时候自己不仅比对面的中单肉，而且输出也根本不会逊色对方。
<br/>　　也许一套输出他没有蒋念打的高，但是不要忘了流浪这个英雄可是被称为法术机关枪的存在啊，一波团战中如果玩的好的话，他的技能是可以释放很多次的。
<br/>　　一波团战中只要能够打出两次w技能的禁锢效果，那么毫无疑问，这一波团战他们这边是必胜的，而且叶帝对于自己是非常自信的。
<br/>　　当时间到18分钟的时候，这个时候双方都聚集到了小龙这里，显然是准备打这一波团战了。
<br/>　　蓝队比赛室，
<br/>　　“这一条火龙打了，他们流浪那个装备现在应该没有多少作战能力的。”秦霄开口指挥道。
<br/>　　“他们如果真敢过来团的话，那么这一波绝对会是咱们赢下来的。”楚南跟着点头说道。
<br/>　　其余几人也都是如此，而白队比赛室，此刻叶帝也是在那里疯狂的指挥着。
<br/>　　“准备团战！”这个时候他已经是将自己的第一个核心装备时光给做出来了，有着这件装备对于团战叶帝更加的有信心了起来。
<br/>　　“你这装备可以打么？”犹豫了一下，夜雪狂歌开口说道。
<br/>　　“一会你就知道这个英雄为什么会被我作为大招一直留存到现在了。”叶帝自信的说道。
<br/>　　为了发挥这个英雄的最大功效，叶帝这一局不仅在大精华上带了三个移速符文，甚至就连天赋上他也是点出了风暴骑手的狂涌。
<br/>　　加上他手中的疾步，只要他的伤害能够打出来，那么流浪这个英雄绝对会在场上飞起来，到时候天下皆可去的，有着那样的跑速，叶帝根本不担心自己的连招会断下来。
<br/>　　听着叶帝这么自信的语气，夜雪几人这个时候也不再犹豫，跟着叶帝向着小龙这边赶来。
<br/>　　“这一波白队是准备和对方打团？”看着叶帝几人的动作，米粒顿时大喊出声。
<br/>　　“现在流浪虽然做出了时光之力这件装备，不过他这件装备的层数貌似并不高啊，而且他这两件装备说实话确实有些拖节奏了，跟发条完全没的比，这一波如果真打的话，感觉白队要崩啊。”林雅楠跟着开口说道。
<br/>　　“蓝队的阵容，无论是圣枪游侠还是巨魔，或者蜘蛛，他们的伤害现阶段都压制着大魔王他们，这一波不该团的啊。”米粒急需喊着。
<br/>　　而就在米粒开口的同时，在众人的视线当中茄子手中的扇子妈已经是开启了自己的大招，随后一个e技能群体加速便是放了出来。
