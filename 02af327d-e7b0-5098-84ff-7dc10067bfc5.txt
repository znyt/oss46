出租车平稳的疾驰在路上，半个小时后到达了目的地，停在了高新路的四胡同里。
<br/>
<br/>        宋晓付了车费然后下车，走进了偏僻的胡同里，驾驶座上的夏夕绾缓缓抬头，她透过挡风玻璃向前看去，胡同里已经有一个黑衣人在等候了，宋晓走过去，和那个黑衣人在低声说些什么。
<br/>
<br/>        夏夕绾澄亮的翦瞳里绽放出了清冽的碎光，这个黑衣人不是苏希，那是苏希的心腹手下？
<br/>
<br/>        她猜的没有错，宋晓就是媒体发布会上的变数，看得出来宋晓对范恬的感情是真的，但是苏希竟然能让宋晓站出来手撕了范恬，就可以看出苏希这个人…极善揣摩蛊惑人心！
<br/>
<br/>        苏希连范恬喜欢夜老这种秘事都能知晓，他身上究竟藏着什么神秘的力量？
<br/>
<br/>        夏夕绾心里对苏希生出几分警戒和忌惮，自古最善变的就是人心，苏希竟然能将人心玩转在鼓掌之间，真的是可怕。
<br/>
<br/>        她不喜欢这种感觉，她太不喜欢这种被苏希偷窥的感觉了，她要亲手将苏希给抓出来！
<br/>
<br/>        这时胡同里的那个黑衣人一回头，直接看到了夏夕绾。
<br/>
<br/>        那个黑衣人顿了一下，很明显没有料到夏夕绾会出现在这里，他和宋晓结束了交流，快速的转身走了。
<br/>
<br/>        夏夕绾当即拉开了驾驶车门跑了出去，但是那个黑衣人已经消失的无影无踪了，好像他根本就没有出现过。
<br/>
<br/>        夏夕绾懊恼的剁了一下脚，她没能抓住苏希！
<br/>
<br/>        现在只能等他下一次露面了。
<br/>
<br/>        夏夕绾转身离开。
<br/>
<br/>        夏夕绾走了，那个黑衣人从黑暗里的角落里走了出来，他看着夏夕绾消失的方向拿出了手机，拨出了一个电话号码。
<br/>
<br/>        很快，那端接通了，黑衣人低声汇报道，“宫主，夏小姐玩了一招螳螂捕蝉黄雀在后，她冒充司机跟了过来，我差一点就暴露了。”
<br/>
<br/>        苏希清润的嗓音从那端徐徐传递了过来，“她不是一般的女孩子，以后跟她交手要万分留心，事情处理好了就回来吧。”
<br/>
<br/>        “是，宫主。”黑衣人消失在了胡同里。
<br/>
<br/>        ……
<br/>
<br/>        夜家采取了紧急公关，当即发出律师声明，声明科学院跟范恬划清了所有的界限，并且对恶迹斑斑的范恬做出了立刻逐出科学院的严肃处理。
<br/>
<br/>        夜家的公关团队相当强的，除了对现在的舆论进行删帖封锁外，还买了水军引导大众舆论风向，很快就将这件事给平息了。
<br/>
<br/>        但是夜老和范恬的桃花新闻还是被编成了小故事，成了帝都城茶余饭后的谈资。
<br/>
<br/>        现在大家看到夜老，第一个想到的不是他是科学院的院长，得高望重，而是他身上的桃花小故事。
<br/>
<br/>        夏夕绾后来听说夜老发了很大的脾气，将办公室里的东西都给掀翻了，她一笑了之，夜老曾经让她一度深陷被几个男人玩了的丑闻里，那她只好回赠他一段桃色新闻广为流转，成为他身上抹不去的印迹！
<br/>
<br/>        这件事告一段落，夏夕绾就在想怎么去找夜滢手里的钥匙，这是当务之急了。
