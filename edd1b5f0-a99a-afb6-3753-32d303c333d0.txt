　　毛小方也走到他边上，从腰间接下了一根紫色的竹节，拿在手中。这是他从师门继承的一件厉害的法器，为天道宗圣地所产的紫竹，一竹七节，做成的这件法杖就叫七节杖，在宗门中也传承了几百年，历代持有者都会在上面刻下符文，加持法力，这是毛小方在晋升天师的时候，宗门按照规矩赐下的一件法器，七段光，虽然跟叶少阳手中的七星龙泉剑比不了，但也是一件非常上等的法器了。
<br/>　　“两位都是有来头的。”五通神冷哼一声，“不过，今天本王要治你们冒犯之罪！”
<br/>　　言毕，两腿一瞪，冲了上来，一双大手中灵气喷吐，朝两人笼罩过来。
<br/>　　“七星拱月！”
<br/>　　叶少阳急忙后退，手中七星龙泉剑在空中连点七下，在空中留下七点灵光，汇聚成七星拱月的阵法，托住五通神手中射来的灵光，汇聚成一束，左手从包里抄出阴阳镜，将这一束灵光汇聚在镜面上，叶少阳立刻觉得吃力，握着阴阳镜的手在颤抖，这五通神……实力果然不一般！
<br/>　　不过叶少阳这样一来，不仅化解了五通神的攻势，也通过阴阳镜的反噬之力，成功将五通神托住。
<br/>　　五通神虽然不认得阴阳镜，但也不是傻比，感受到了镜面反噬之力的存在，只要收回攻势，阴阳镜立刻会将自己这一波灵力反噬回来，让自己受伤，因此也只能不断释放灵力，想要将叶少阳震飞出去。
<br/>　　眼下的叶少阳只有真人的修为，凭自己的实力根本无法抵挡，只能放下七星龙泉剑，双手捧着阴阳镜，借助阴阳镜的灵力，死死撑着，体内气血翻涌，几乎要吐出血来。
<br/>　　好在毛小方也看出情况，手持七节法杖，对着五通神展开了一轮进攻。
<br/>　　五通神被叶少阳拖住了一只手，脚下不敢移动，只能站桩，用一只手召唤出仙家灵光，跟毛小方缠斗。
<br/>　　说起来，以他的实力，如果真正打起来的话，现在的叶少阳，加上毛小方根本就不是对手，不过叶少阳上来就用上了阴阳镜，反而成功将他拖住。
<br/>　　五通神不能移动，毛小方于是利用这一点，在他身边游走进攻，让五通神疲于应付，最后总算成功绕到他身后，一仗下去，击打五通神的天灵盖。
<br/>　　这一棍，是利用了一个绝佳的机会，志在必得，结果五通神闷喝一声，体内突然爆发除了一股妖气，隐约泛着莹莹绿光，猛然迸发出来，将叶少阳震飞出去，落在地上，同时妖气也裹住了毛小方，化作数股，对他进行上下左右的绞杀围攻。
<br/>　　毛小方被打了个猝不及防，一通防守，狼狈退开，头发也散了，衣服也被划出了很多口子，几乎站都站不住，用七节杖撑着地面，才没有勉强倒下。
<br/>　　“我已成神，多少年没用过本家妖力，是你们逼我的。”五通神收了妖气，望着被震翻在地的叶少阳，轻蔑地笑了笑。“无知的法师，居然妄图抵挡仙家神威。”
<br/>　　他起身走向云儿。
<br/>　　云儿瘫坐在地上，早已经陷入迷茫，此刻看见他走过来，这才回过神来，绝望地叫起来：“不要！”
<br/>　　五通神不理他，一双鼓鼓的大眼睛仍然盯着叶少阳看，冷笑道：“你不是口口声声要带她走吗？”
<br/>　　说完大笑数声，伸手去拉云儿。
<br/>　　突然一道灵光从天而降，落向五通神的脑袋，五通神情急中反手一抓，却是被灵光从掌心划过，留下了一道长长的伤口。
<br/>　　七星龙泉剑，斜插在地上。
<br/>　　叶少阳强撑着站起来，走了过来。
<br/>　　刚才，已经受伤的他不顾一切地念起咒语，将七星龙泉剑祭了出来，一来他有伤在身，五通神对他没有防备，二来七星龙泉剑的灵力太强，五通神仓促间也是没想太多，反手去抓，这才受了伤。
<br/>　　“我说过要带她走，就一定会带她走！”叶少阳走过来，从地上拔出七星龙泉剑，剑指五通神，盛怒的表情让人不寒而栗。同时浑身戾气缠绕，只有法师邪物才能看得见，看上去像是一团炽烈的火焰。
<br/>　　五通神本来以为他是天师，但是刚才一经交手，五通神觉得这家伙虽然狡猾，但是实力不过是真人，还当他是个绣花枕头，很不以为意，结果看他现在的样子，五通神惊呆了。
<br/>　　这绝不是真人，也不是一个普通的天师！
<br/>　　五通神几乎立刻就得出了这个结论，不由皱眉问道：“你身上怎么会有这么强的戾气，你到底什么人？”
<br/>　　“茅山天师，叶少阳。”叶少阳冷冷说道。
<br/>　　五通神捂着自己受伤的手掌，冷哼一声，道：“不管你是谁，敢伤我，今日我也无所顾忌了。”说完，浑身抬起双手，又擒出了两束妖气，只是比之前更加炽烈几分，朝叶少阳走了过去。
<br/>　　叶少阳手握七星龙泉剑，寸步不退，也做好了死战一场的准备。
<br/>　　毛小方也走了过来，与他并肩而立，一脸的决然。
<br/>　　五通神猛然出手，对着二人拍出了一掌，妖光四射。两人刚要抵抗，身后突然冲出一道玉符，敌住了妖气。一个清亮的声音响起来：“五通神，切莫动手。”
<br/>　　五通神一惊，也不敢硬来，急忙收手，定睛看去，却是一个书生打扮的俊俏郎君，从不远处快步走来，看面相却不认得，于是皱眉问：“来者何人？”
<br/>　　叶少阳和毛小方也回头看去，见是萧逸云，顿时松了口气。
<br/>　　萧逸云走到叶少阳二人身前，拱手说道：“我是天子殿押司萧逸云，见过五通神。”
<br/>　　五通神一听，也急忙还礼，“原来是萧郎君，失敬失敬。”五通神长期在人间，对阴司的这些大佬们很多都没见过，不过萧逸云毕竟是天子殿第一红人，名字总是听过的，不敢失礼。
<br/>　　“不知萧郎君此来，所为何事。”五通神见萧逸云站到叶少阳二人身边，立刻有点警觉起来。
<br/>　　萧逸云笑道：“这两位，是我在人间的好朋友。”
