　　第720章
<br/>　　李行哉出门的排场不是程大雷能相提并论的，从书院弟子中择几名老成持重的，再加上漕帮中的好手，凑齐百人。人一多，事情便复杂起来，这一百人的衣食住行都需要考虑。
<br/>　　于是赫赫一队人马，踏上了去往长安的路。于此同时，各路诸侯也纷纷上路。
<br/>　　免不了有两家正进行着刀兵之争，逢到这样的事，也只得暂时两家罢兵。巧合的是在去长安的路上又遇见了，也只好白刀子进红刀子出。
<br/>　　程大雷五人一路往东走，有两三日光景，已进入京州境内。偏生逢着一场大雨，五人只好在一处破庙中躲雨。
<br/>　　大雨滂沱，劈头盖脸砸下来。屋檐下雨珠成帘，这庙宇年久失修，外面下大雨，庙里下小雨。随身携带的干粮也被淋湿不少，几个人都搞得什么狼狈。
<br/>　　程大雷立在大门前，看着外面的瓢泼大雨，咧嘴道：“这雨当真来得不是时候，如果一时半会停不了，咱们就只有在这庙里过夜了。”
<br/>　　崔白玉撇撇嘴，心道：出门不行善，在外大雨灌，你若多做几件好事，也不会赶上这大雨了。
<br/>　　“嘿，俗话说的好，龙行云动，虎行有风，大当家龙虎般的人物，碰上这样一场雨，也是好兆头呐。”刘发财。
<br/>　　“呃……哈哈，言之有理，言之有理。”
<br/>　　黑纱下，崔白玉的嘴已快咧到耳朵。
<br/>　　“不过，这雨下起来没完，也不是个法子？”
<br/>　　“飘风不终朝，骤雨不终日，这样的大雨下不了多久的。”刘发财。
<br/>　　偏巧被刘发财说对了，这雨来得急，去得也急，等云开雾散，天光放晴时，已经是黄昏傍晚。西边彩霞烧成了血色，摇摇望过去，也是份外壮观。
<br/>　　“好了，咱们再往前走一遭，看有没有人家投宿。”
<br/>　　这一路走来，程大雷也是苦不堪言，能找个住处好好歇息一晚，再赶路也不迟。
<br/>　　五人以程大雷为首，程大雷感觉自己像个取经小团队一样，踏上了一路向东的路。
<br/>　　也是运气，行了三五里路，前方出现一座大庄园。瓦房连片，绿柳成荫，北地很少有江南那样的小桥流水，但十几处院落连在一起，也是自成气派。
<br/>　　“刘发财，你去前面问问，看他家能不能让我们投宿？”程大雷道，如今饥肠辘辘，也是盼着有些热汤热面，再能好好睡上一觉。
<br/>　　刘发财答应一声，快步来到院门前，看到门前挂着『陈宅』的牌匾。他举手拍门，从门内走出个年逾花甲的老者。
<br/>　　“老丈，我们是去京城访亲的客人，能否在贵处留宿一夜，如今这雨天路滑，也是走不动路了。”
<br/>　　老者往远处的程大雷身上打量一眼，道：“我们老爷向来好做善事，既然是行路的客人留宿一夜也无妨，但今天有些事，不便留下几位客人，你们若要饭食，府上倒是可以准备。”
<br/>　　“嗳，老丈既然已经舍了一顿饭，何妨再留我们住一宿，我们也不是白吃白住，到时候多给你银子。”
<br/>　　刘发财从袖子里掏出一锭小银元宝，在老者面前晃了晃。这次出门，程大雷还是多带银钱，如这样的小元宝也有数百颗。
<br/>　　“呵，你可知道这里是什么地方，我家老爷人称陈百万，家里粮米成山，金银成堆，稀罕你这点银子，本来还想留你吃顿饭，你们竟然显摆银子，快走快走，一顿饭也没有了。”
<br/>　　刘发财一听这话眼睛就亮了，粮米成山，金银成堆，这完全可以劫一把嘛。此地距离凉州城也不远，急行军有几日也就到了。
<br/>　　“呔，退下。”
<br/>　　这时候门内走出一老者，身上绫罗绸缎，虽然上了些年纪，但保养得极好。
<br/>　　“陈忠，何事在门前聒噪。”
<br/>　　名唤陈忠的仆人在其人耳边低语几句。那人目光看了看刘发财，又看了看远处的程大雷。
<br/>　　“陈忠，这便是你的不是，出门在外，谁也有走窄的时候，与人方便也就是与己不便，留几位客人住上一宿又如何。”
<br/>　　说着冲刘发财拱拱手，道：“客人勿怪，乡野鄙夫，不懂道理，客人还不要和他们一般见识。”
<br/>　　刘发财也急忙还礼，看这老者话里话外的意思，倒也像个闯荡江湖的角色。
<br/>　　程大雷几人迎了过来，那老者知道该冲谁说话，冲程大雷道：“老朽陈德山，几位客人权且在此处留宿一夜。只不过，无论今夜有什么动静，也请几位客人不要出来，也是为几位的性命着想。”
<br/>　　程大雷抱拳还礼：“多谢老丈，是府上出什么事了么，若是有用的着我们的，我们也能出把子力气。”
<br/>　　“小事，小事，不敢麻烦几位客人。”陈德山冲陈忠挥挥手：“阿忠，请几位客人入府，安排饭食，不可慢待。”
<br/>　　那名唤陈忠的仆人在前面引路，给五人安排了一个偏僻的院落，倒是清净的很。杀鸡宰羊，不多时汤食便端了上来，北地饭食比不得江南精致，无非是重盐重卤，酒水也给得足。
<br/>　　见左右无人，刘发财道：“大当家，这府上是出什么事了？”
<br/>　　程大雷抿嘴笑笑，道：“你看出来什么？”
<br/>　　刘发财略想了想：“陈德山虽然年逾花甲，但身不歪，背不驼，呼吸绵长有力，怕年轻时也是个练家子。”
<br/>　　程大雷点点头，刘发财毕竟也是蛤蟆寨出来的，这点眼力还是有的。
<br/>　　“那陈忠也不是普通人，他双目聚而不散，像也是有功夫的。”崔白玉忽然道。
<br/>　　刘发财注意力搁在陈德山身上，倒是没对那老仆留意。崔白玉也是证明自己的价值，被人呼来喝去的滋味其实并不好受。
<br/>　　“大当家，要不要我去探探，看他们到底出了什么事，老汉人不错，冲着这顿饭，咱们也得帮他们一帮。”
<br/>　　“稍安勿躁。”程大雷挥挥手：“他们不想说，问也未必能问出什么，不管有什么事，咱们到晚上不都知道了。”
<br/>　　说着，程大雷将一根鸡腿递给崔白玉：“来，刚才你说得好，奖励你的。”
<br/>(本章完)
