<br/>　　SHADOW和GHOST终极一战，双方同归于尽的消息并没有被隐瞒下来，因为影天使那个战书视频是全球播放的，很多人为了能鉴证这历史性的一幕，甚至不远万里来到K国，所以写字楼发生爆炸，SHADOW和GHOST葬身火海的消息，很快就出现在网络上，有人甚至附上了用手机拍摄的写字楼爆炸视频。
<br/>　　“SHADOW和GHOST葬身火海！”
<br/>　　“黑客世界的两颗传奇之星陨落！”
<br/>　　“对决现场发生爆炸和枪战，SHADOW和GHOST同归于尽！”
<br/>　　类似标题的帖子在网络上铺天盖地，不需要去专业级别的黑客论坛，网上随便一个搜索引擎一搜就能搜出一堆。
<br/>　　这么多人看到了当时发生的一幕，由不得不信。很多人都认为GHOST已经随着那场爆炸消失了，这个世界上再也不会有GHOST这个人，这个世界上再也不会出现GHOST这个特有签名的时候，GHOST重现了！
<br/>　　GHOST重现，并且以最简单直接的方式向人宣告GHOST并没有消失，它还存在！就跟“GHOST”这个代号一样，他们这群网络幽灵潜藏在虚拟的网络世界中，随时都有可能重新出现，给人一个巨大的“惊喜”！
<br/>　　GHOST的重现让世界上那些以GHOST为目标，立志要成为GHOST并且超越GHOST的黑客狂欢，也让世界上那些曾经被GHOST光顾过的武器公司技术部门都疼不已，这家伙又要回来了吗！？
<br/>　　完成此次攻击后，云熙就让参与攻击的信息作战部队解散了。即便是最后有人查到了攻击源来自何处，也可以用一句“不知情，不了解，不解释”直接糊弄过去，毕竟，耍流氓这样的事情大家都会做。
<br/>　　这次龙隐部队几个人回国没有再遭遇任何阻拦，K国代总统派出了总统卫队护送罗昊、云熙等人前往机场，竭尽全力要修复跟中方之间的关系。
<br/>　　裂纹造成很容易，可是修复却并不那么容易……
<br/>　　……
<br/>　　中东，某地。
<br/>　　这里可以说是目前世界上最混乱的地方，这里被称为“无政府地带”。
<br/>　　这里有恐怖组织在活动，有反政府武装在活动，也有一些要钱不要命的雇佣兵在活动。
<br/>　　这里混乱无比，就连自认为“天下无敌”的美国大兵，以及无孔不入的CIA特工都不敢深入其中。进入到这片区域活动的雇佣兵的雇用价格也是最高的，但是这么高的佣金很多时候都是有命接但是没命拿。
<br/>　　这里没有政权存在，因为即便存在，也不可能存在很长的时间。没人执政，也无人敢执政，这就是“无政府地带”的由来。
<br/>　　那五个来自K国的“炼狱”学员就被骑士授意扔到了这片“无政府地带”中，仍由他们自生自灭。死了是正常，能活下来只能说明他们命真的很大，命不该绝。
<br/>　　当骑士把这五个K国学员单独分到一架直升机上，并且告诉直升机驾驶员他的投放地点时，差点把那个驾驶员吓得从直升机上摔下来。
<br/>　　没有人敢驾驶直升机进入“无政府地带”，因为只要直升机进入其中，十有八九会被从四面八方飞来的火箭弹给打下来，然后飞行员被俘，向飞行员所属国家索取高额赎金，如果拒绝支付就是被当众斩首的下场。
<br/>　　而且一旦被抓，就连特种部队都很难进入其中营救。把一支特种部队投放到一个四面环敌的区域中，即便这些特种部队队员武装到了牙齿，也很难在营救人质后成功撤离。
<br/>　　只要在“无政府地带”被俘，基本就是个死人了。
<br/>　　直升机在把五个K国学员放在“无政府地带”外围后，掉头就跑，根本不带任何犹豫的。如果跑得慢，可就连跑的机会都没有了，直升机上的两个飞行员才不想给那五个K国学员陪葬。
<br/>　　五个K国学员很快就意识到了自己来到了什么地方，但他们已经没有退路，他们深入到“无政府地带”腹地，没有通讯设备无法跟训练基地联系；而且就算是有通讯设备，五个K国学员也不认为训练基地会派出直升机来把他们从这片“无政府区域”接离。在知道自己这些人被投放的区域是“无政府地带”的时候，五个K国学员就意识到他们已经被训练基地抛弃了，不管是因为什么原因，他们被抛弃了，他们不会天真的认为“炼狱”会在这里放置什么补给箱，让他们获得武器和食物，让他们能够离开这里，返回训练基地。
<br/>　　在“无政府地带”放置补给箱，那跟送死有什么区别？根本没有区别！
<br/>　　五个K国学员身上没有任何装备，所以根本不知道发生在K国的事情，可即便知道K国的事情，他们也很难跟自己被抛弃建立起联系来，云熙跟云岳之间的关系整个炼狱只有各个军事训练基地的军事主教官知道，而且在云岳交出鬼王面具，离开“炼狱”之后，也没有人再提起过。
<br/>　　五个K国学员就算是抓破头皮也不会想到自己国家的总统，因为一时的糊涂招惹到了“炼狱”的隐藏boss！他们现在所要想的是怎么活下去，并且逃出这里！
<br/>　　五个人在“无政府地带”已经待了两个礼拜，五个人中，一个在逃脱反政府武装追杀中被乱枪打死，一个被子弹重伤，眼看难以活命；剩下的三个虽然还活着，但是他们或多或少都受了一些伤。
<br/>　　他们需要躲避反政府武装的搜捕和追杀，可是他们缺少武器弹药和医疗用品，他们的状态已经支撑不了他们再接受一场战斗了，即便是不再跟敌人交手，他们没有医疗用品，等到伤口恶化，发生感染，他们也只能是死路一条！
<br/>　　“汪汪汪！”
<br/>　　房子外的街道上传来狗叫声，反政府武装饲养的狗闻到了血腥味，循着血腥味找到了他们！
