　　　　
<br/>　　随着一行人逐渐深入，古窟中越来越暗，洞顶的蝙蝠似乎无穷无尽，越深入越多，尖牙利齿，喃喃低鸣似乎都在耳边呼啸。
<br/>　　足足走了半刻钟左右，众人的面前出现两条岔路，幽幽深深，漆黑无比，不知通向何方，宛若妖魔张开的大口一般。
<br/>　　而在道路的中间，竖立着一块足足有六人高的巨大石碑，上面雕刻着四个血红色的大字：“天道在我！”
<br/>　　就在这时，法相看向慕清霄，道：“慕师兄，眼下这洞穴中危机四伏，眼前就有一个难题，我们该走哪一条？”
<br/>　　闻言，慕清霄沉吟片刻，道：“据说当年正魔大战，两条岔路后都是魔教巢穴所在，既然如此，我看不如兵分两路，我和雪琪去左边查看，你们去右边查看，如何？”
<br/>　　法相愣了愣，这万蝠古窟中凶险难测，谁不知道下一步会遭遇什么，慕清霄居然在此刻提出兵分两路，实在是下下策：“不知道诸位觉得如何？”
<br/>　　“那就按照慕师弟所说，我们兵分两路。”
<br/>　　说罢，张小凡就率先向右边通道走去，心中根本没有担心一说，林惊羽看了两人一眼也跟了上去，曾书书见他们离去，连忙追了上去：“哎，等等我啊！”
<br/>　　慕清霄朝法相微微点头，牵着陆雪琪消失在左边的洞穴。
<br/>　　李洵见陆雪琪的身影消失不见，脸色顿时阴沉下来，冷哼一声道：“真是狂妄自大，这小子到底是什么人，凭什么指挥我们？”
<br/>　　法相苦笑一声，道：“李施主还请以大局为重，相信慕师兄有他的用意吧，我们也快点出发吧......”
<br/>　　走在左侧黑暗的通道中，陆雪琪忍不住问道：“清霄，万蝠古窟中危险重重，与师弟他们分开不是明智的选择。”
<br/>　　慕清霄笑而不语，带着李洵这些拖油瓶才不是明智的选择，浪费时间不说，那种气氛他就不喜欢，万蝠古窟虽然古怪，对他来说却算不上凶险。
<br/>　　“别怕。”
<br/>　　“...嗯。”
<br/>　　话语刚落，在陆雪琪震惊的目光中，慕清霄抬起右手，掌心黑色火焰跳动：“去。”
<br/>　　一缕黑色火焰弹射而出，接触洞顶的瞬间，以极快的速度向四周蔓延，没有火焰的温度，可接触到火焰的蝙蝠，连惨叫声都无法发出，直接化作灰烬。
<br/>　　火焰完全没有停息的意思，不断向洞穴深处蔓延，陆雪琪捂着粉唇，美眸盯着慕清霄，道：“这是什么火焰，感觉不到温度，可却给我一种恐怖的感觉。”
<br/>　　“你的直觉没错，它叫九霄琉璃焱，是一种本命火焰，能够焚尽万物，刚才有外人在，我不方便使用，否则你也不用担心受怕了。”
<br/>　　听到慕清霄说她不是外人，陆雪琪心中甜滋滋的，没想到他选择分开，都是为了自己。
<br/>　　慕清霄选择与张小凡几人分开，有一半确实是因为陆雪琪，毕竟这可是自己的女人，第二就是万蝠古窟中的天书，若是让别人知晓，只会徒增麻烦。
<br/>　　“我们暂且放慢点速度，等九霄琉璃焱将蝙蝠清理干净我们在加速前进。”
<br/>　　“好，都听你的。”
<br/>　　陆雪琪自然没有意见，没有外人在，也收起冷若冰霜的模样，绝美脸蛋上牵起若有若无的笑意。
<br/>　　“走吧。”说罢，慕清霄抱起陆雪琪脚尖轻点，闪烁间已经深入百米。
<br/>　　随着两人继续深入，由于通道实在太多，慕清霄只好用神识探路，就在此时，忽然听到前方有打斗声传来，不由的停下脚步。
<br/>　　“前面有人在斗法，我们先过去看看。”
<br/>　　陆雪琪紧握着天琊，美眸中满是凝重，心想着难道是魔教妖人。
<br/>　　待两人穿过数条曲折的通道，终于来到一处出口，放眼望去，果然有一群人正在斗法，其中一批正是张小凡等人，另外一批是前些日子，在河阳城遇到的鬼王宗等人。
<br/>　　而与他们激斗的另一批人身穿黑红色服饰，身上散发的凶煞气息颇为浓厚，看样子估计是魔教炼血堂的人。
<br/>　　鬼王宗，金丹修为的蒙面女子正巧与炼血堂的年老大对上了，一时间胜负难分，张小凡几人对付炼血堂的残存势力，倒是颇为轻松。
<br/>　　“清霄，我们要下去帮忙吗？”
<br/>　　“不必。”
<br/>　　慕清霄屈指一弹，一道凌厉的剑气，以迅雷不及掩耳之势向下方的年老大刺去。
<br/>　　此时，年老大正与幽姬势均力敌，而且占据上风，舔了舔嘴唇，道：“朱雀圣使，我们都是圣教中人，算起来也是同门，你又何必与我炼血堂为难，让我宰了这帮青云门的小虫子，我便放你们离去如何？”
<br/>　　“你们炼血堂已经没落，你不过风中残烛，有什么资格来指使我。”幽姬喘息一声道。
<br/>　　“哼，不识抬举。”
<br/>　　“噗嗤...!”
<br/>　　年老大刚要发起进攻，小腿突然传来锥心的刺痛，转头望去，发现自己的小腿已经被刺穿，鲜血飙射，惨叫一声后飞身而退。
<br/>　　“谁，是那个混蛋偷袭我！”
<br/>　　幽姬虽不清楚状况，却知道年老大吃亏，又怎能让他逃走，金丹修士爆发出的力量碾压上去。
<br/>　　暗处，陆雪琪暗暗吃惊，下方两人修为她根本看不透，绝对是太清境修为，根据刚刚的一招，慕清霄绝对能够悄无声息的击杀对方，可见他的修为有多么恐怖。
<br/>　　最起码，不会低于太清境。
<br/>　　陆雪琪美眸一眨也不眨的望着自己的恋人，有些恍惚，回想起两人在无名山谷中相遇的时候，慕清霄果然是让着她，否则有怎会受伤。
<br/>　　慕清霄嘴角噙着一丝笑容，虽然他不想帮忙，却也不希望这帮魔教中人来打搅自己，让他们拖延点时间也是够了。
<br/>　　“我们走吧，滴血洞中有我需要的东西。”
<br/>　　陆雪琪乖乖的点了点头，她是越来越看不透慕清霄，自己的男人就向谜一样，看不透猜不透，忍不住问道：“清霄，你怎么知道，万蝠古窟中有你需要的东西？”
<br/>　　“天机不可泄漏。”慕清霄神秘一笑。
