　　叶少阳一听，当然不愿意错过这个机会，于是让小陈在前面带路，飞快到朝赵四家跑过去。来到赵四家门外，小陈听了一下，里面没有哭声，确定人还没死，于是让叶少二人在外面等着，自己进去再探探，毕竟叶少阳是个陌生人，突然闯进人家家里，有点太唐突了，而且他的动机实在是让人捉摸不透。
<br/>　　小陈进去之后，很快就出来了，告诉叶少阳，赵四不行了，呼吸困难，估计也就个把时辰的事了。家里已经开始准备后事，连寿衣都穿上了。
<br/>　　“现在咋办，要进去吗，如果进去的话，怕是不好跟他们解释呢。”小陈有些为难。
<br/>　　叶少阳道：“不用，你们都先走吧。既然他真的要死了，我也不需要进去。”
<br/>　　翠云听了说道：“不用我们帮什么忙？”
<br/>　　叶少阳表示不用。
<br/>　　小陈说道：“翠云嫂子可以先去我家坐着喝茶，叶先生办完事去我家找她就行。”
<br/>　　叶少阳表示同意，让翠云跟他一起走，自己绕着赵四家的房屋走了一圈，这时代的房屋建筑跟百年之后一样，多是坐北朝南，房后是正北方，那里有一片高粱地，正合叶少阳的心意，当下躲了进去，踩倒了一片高粱秆子，在中间布置法坛，然后盘膝坐下，静静地听着赵四家的动静。
<br/>　　大概等了有半个小时左右，从赵四家里突然传来一阵啼哭声，还是不止一个人的声音，交织在一起，声音十分响亮。
<br/>　　要说哭丧，其实也是一门学问：家里有人去世，当然是悲伤的，不过就拿赵四这种情况来说，久病在床，就算家里人不嫌弃，也早就做好了他即将死去的心理准备，真正到人死这一刻，并没有多么强烈的悲伤。不过总是要哭一场，表现的越是悲痛，证明自己就越孝顺。
<br/>　　因此哭丧在过去，是儿媳妇们无师自通的一门技能，连哭带唱，看上去痛不欲生。
<br/>　　叶少阳听着一个嘶哑的女声连哭带唱，长出了一口气，心想总算是死了啊。
<br/>　　接着又觉得自己这想法有点那啥，好像盼着人死似的。
<br/>　　叶少阳摸出一张灵符，点燃之后，念了咒语。
<br/>　　一道人影，从赵四家里凌空飘了过来，晃晃悠悠，如果有人看见的话，准会当成是一件被风吹走的衣服。
<br/>　　不过没有人能够看见，因为它不是衣服，而是鬼魂。
<br/>　　赵四的鬼魂。
<br/>　　一般来说，人死之后，鬼魂离体，会有一段茫然的时期，就像从梦中醒来一样，看见自己的尸体，会先是迷茫，不确定是怎么回事，所以有句话叫人生一场梦。
<br/>　　之后，鬼魂听见亲人的哭声，才意识到自己已经死了，这时候会感到悲伤，不过因为刚死，魂力虚弱，一般没法显形让亲友们知道。当然也有例外，例如是带着强烈的怨气而亡，死后受到怨气支配，会通过一些形式让人察觉到自己的存在。
<br/>　　悲伤之后，就是留恋，会绕着自己的身体转圈，甚至试图重新回到自己身体里去，不过还是因为魂力虚弱，无法附身在死尸上，因此在试了几次之后，一般就放弃了，这时候才真正意识到自己已经死了，开了天知，明白了生死之道和自己的去向。
<br/>　　到这一步，一般鬼魂都会离开人间，先去阴司销帐报道，等候天子殿勾画生死簿，判定今生来世，如果发落轮回，可以在酆都城中暂住一周，等候回魂夜再去人间看望一下亲友，了却心事，之后就去轮回……
<br/>　　赵四刚死，按说从迷茫到觉醒，再到离开，至少也要半个小时，不过叶少阳可是等不及了，之前烧的那一道招魂符，就是将他引过来的。
<br/>　　赵四的鬼魂飞了过来，一脸的茫然。
<br/>　　叶少阳先在他身上贴了一张符，形成一道简单的结界，可以吸收照在他身上的阳光中的日精，不然这大日头的，他一个生魂会有点受不了。
<br/>　　叶少阳看他一脸懵逼的模样，左手结印，在他眉心处点了一下，帮他开了天知。
<br/>　　赵四浑身哆嗦了一下，仿佛一个人从梦中猛然惊醒，左右张望起来，最后望着自己家的房子，怔怔发呆。
<br/>　　叶少阳抱着手站在一旁说道：“别看了，你已经死了。”
<br/>　　“死了……”赵四叹了口气，突然呜呜地哭了起来。“死了，我这一辈子就这么过来了，啥也没干……”
<br/>　　叶少阳听到这话忍不住乐了，“可不是嘛，你还想干啥，只要没做伤天害理的事，还能投胎，争取下辈子再享受嘛。”
<br/>　　“呜呜……我都当了八辈子贫农了，上次当一个小吏，还是明朝时候了，因为贪污军饷被斩首了……”
<br/>　　叶少阳噗嗤一笑，“这你怪谁来着。”
<br/>　　人的命，虽然在投胎之初，就有了不同区别，也就是命格，有的人天生就是枭雄的命，有的人天生就是普通人，所谓心比天高，命比纸薄，就是这意思。不过，只要前世没作孽的话，在小范围内还是改命转运的机会，只是很多人没有抓住机会，自甘平庸罢了。
<br/>　　每一个平庸的人死后，鬼魂都会不甘，想着来生一定要好好奋斗，不过喝了孟婆孟婆汤之后，什么都给忘了，投胎转生，又是一场新的轮回……
<br/>　　赵四哭了一声，情绪也缓和下来，注意力这次放在叶少阳身上，开了天知的他，知道叶少阳是个法师，作为生魂，也是得罪不起的，当下拱手说道：“大法师，召唤我来做什么？”
<br/>　　“没什么，让你帮我个忙。”
<br/>　　“什么忙？”
<br/>　　“一时解释不清，你也不需要知道，反正我肯定不会害你。”叶少说完，拿出一截红绳，上面串了一枚铜钱，然后把这个简易的挂坠往赵四脖子上一挂。
<br/>　　“这是……”
<br/>　　“没什么，法师一文值千金，这是千金坠，能让你快速过关，进入天子殿。”
<br/>　　千金坠，是一种法师的信物，普通的五帝钱，在人间非常轻的重量，对鬼魂来说却是重若千斤，不过不会把鬼魂怎么样，只会帮他坠入鬼域，直奔天子殿。
