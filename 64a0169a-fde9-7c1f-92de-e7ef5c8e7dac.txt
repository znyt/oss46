　　第520章
<br/>　　城主府烧起一场熊熊大火，惊动了整座蛤蟆城的人，不论在蛤蟆城的任何一地方，都能看到火光。
<br/>　　消息传到了兵营，关鱼站在高处看着，口里道：“干嘛呢，这是怎么了？”
<br/>　　早有消息传到这里，副将道：“是那要杀大当家的老尼姑又回来了，蛮字军已经在处理，据传来的消息说，他们围了一座院子，要将那老尼姑活活烧死。”
<br/>　　一听有事情要做，关鱼还是蛮激动的。但随之听见蛮字军已经出动，他不免又有些遗憾。
<br/>　　“瞧瞧，对付一个人，需要搞这么大声势么。”关鱼指着远处道：“这就是没本事呐，像咱们这种有本事的人，就不需要做。明天将训练任务加重，咱们以后可不能办这种丢人事。”
<br/>　　“是。”
<br/>　　副将干脆答应，心里却叫苦不迭：现在的训练任务还不够重么。
<br/>　　城主府内，却没有关鱼这种轻松随意。
<br/>　　无论如何，白衣老尼终究是从火海中闯了出来。虽然她现在的情况……不太对！
<br/>　　她身上的衣物被引燃，已经烧成一个火人。蛮字军不得不拉开距离，将她围在当中，避免引火烧身。
<br/>　　她张牙舞爪，似乎已觉察不到痛苦，口中不停喊着：“程大雷在哪里！程大雷在哪里！”
<br/>　　饶是程大雷见惯了杀人的场面，此刻也觉得胆战心惊。心道，自己出世以来，也没怎么做坏事啊，有必要玩着命杀自己么。
<br/>　　纵然白衣老尼已经威胁不到程大雷的安全，但一个人意志决绝的要杀你，总会令人脊背发寒。
<br/>　　在所有人中，秦蛮是为数不多保持镇定的人，他的拳头握紧又松口，口中只有简单干脆的一个字：
<br/>　　“放！”
<br/>　　一次又一次，不停重复，每一个字落地，便是一阵箭雨。老尼姑拥有不错的力量，出色的速度，但现在的情况，她的反应和意识都极其差。
<br/>　　只是张牙舞爪，发疯一般乱窜，但却无法伤害到任何一人。同时，也挡不住一轮又一轮的箭雨。
<br/>　　今日潜入城主府的，不仅有白衣老尼一个。孟玄青随着白衣老尼过来，只是他目标没有那么显眼，此时还未被发现罢了。
<br/>　　如今，他就躲在大殿的屋脊上，远远看着这一幕。
<br/>　　就见老尼姑像个火球般乱窜，跃到那里，又跃到这里，口中的吼声越来越愤怒，一开始是吼声，但渐渐就是痛苦的惨叫。
<br/>　　她终究是血肉之躯，功夫再高，也没到铜皮铁骨的地步。
<br/>　　看着她此刻的惨状，便是秦蛮手下的蛮字军都觉得她有些可怜了。更何况是与其同出西蜀一脉的孟玄青，此时他将一口老牙咬碎，恨不得当场冲下去……冲下去和老尼姑一起去死。
<br/>　　这……好像也不能解决什么事情。
<br/>　　自己精于毒术，对于功夫虽然也算不错，但毕竟和那白衣老尼有些差距。现在，连老尼姑都不是程大雷的对手，自己下去，也是白白送死。
<br/>　　可若是不下去拼命，能做的似乎就只是躲在这里，眼睁睁看着老尼姑被烧死。
<br/>　　他当然不会看着老尼姑被活活烧死，他选择扭过头不看。
<br/>　　天快亮了，东方露出鱼肚白。在老尼姑被围入火海中后，包围整座城主府的蛮字军，正在安静调动。更多人涌入城主府，这样自然是因为对老尼姑的重视，但同时，也疏忽了外面的防御。
<br/>　　此时不走，更待何时。
<br/>　　孟玄青像一只蝙蝠般起身，神不知鬼不觉的离开城主府，又鬼鬼祟祟潜出城外。直到离城十里，他整个人才颓然倒在地上。
<br/>　　之所以如此，一是因为力疲，二是因为巨大的痛苦。
<br/>　　在山里忍了这么多年，不去理睬人间的是是非非，他已经很习惯当一个缩头乌龟了。可今天老尼姑的惨状，还是唤醒了他，如一根毒针狠狠扎进他心底。
<br/>　　我已经老得快死了，难道还怕死么！
<br/>　　双目不再迷茫，而是有熊熊火焰燃烧起来。就用自己这条快死的老命，再继续做些事吧。
<br/>　　他站起身，冲着蛤蟆城的方向跪下，重重的磕了三个头。
<br/>　　“公主殿下，你一路走好，我会为你报仇的。”
<br/>　　他站起身，掉头离开，然后，一路向东。
<br/>　　城主府内，老尼姑的生命已走到尽头。人力有时而穷，何况在熊熊火海之中，她又能撑多久呢。虽然她口中高喊着『我要杀了你，我要杀了你』，但终究，她谁也没能杀死，最后身体颓然倒地，身上插满羽箭，最后也分辨不清楚，是被烧死的，还是被箭射死的。
<br/>　　秦蛮握紧的拳头缓缓松开，倒此时才舍得放下，嘴角浮现一抹冷冷的笑。
<br/>　　“哼，江湖人。”
<br/>　　如秦蛮这样的武将，自然是对江湖武夫不屑一顾的。任你武功再高强，面对千军万马又能怎样，不过是死路一条罢了。
<br/>　　无非就是多费一些力气，多牺牲几个人而已。
<br/>　　程大雷的想法倒是和他不同，他本身对征战沙场，裂土封侯没多大兴趣。况且现在管理一座城池，对他来说已经很棘手了。他更向往的生活，仍旧是匹夫挂剑，在江湖上闯荡。
<br/>　　人各有志，却也说不上谁对说错。
<br/>　　宰掉老尼姑后，便开始收尾工作，燃起的火自然要灭掉，免得余烬引起更大的火灾。相比杀人，这件事自然更麻烦些，不过这些都有秦蛮的蛮字军处理，倒是用不着程大雷做什么。
<br/>　　等火场的火被扑灭，老尼姑的尸体收拾起来，天光已经大亮。最后清点发现，老尼姑身上竟然插着上百枚箭头，真令后来的人想不通，她临死前究竟经历了什么。
<br/>　　天光大亮的时候，关鱼、高飞豹、张肥、赵子龙几个也到了。出了这么大事，他们总不能不过来看看。
<br/>　　“啧啧，了不得啊。”关鱼忍不住赞道。
<br/>　　秦蛮昂起头，昨夜使了好大力气，为了谁，不就是为了给关鱼这些人看看么。
<br/>　　他装作不在意的样子，道：“无非一个刺客而已，有什么了不得的。”
<br/>　　“哈，因为一个刺客，搞得如此兴师动众，秦将军当真了不得喔。”
<br/>(本章完)
