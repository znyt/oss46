　　这些怨灵的实力，几乎都有着古帝的实力，充满着惊天怨念。
<br/>　　这些怨灵，要将所有进入佛寺之中的人杀死，与他们作伴。
<br/>　　十几名怨灵，同时出手，向着楚风眠偷袭而来，施展而出的武技，基本都是佛门武技。
<br/>　　看来是跟刚刚楚风眠外面听到的一样，这里面的怨灵，应该就是曾经净土宗的弟子。
<br/>　　净土宗的衰败，跟净土佛祖入魔，化为血邪魔帝，有着极大的关系。
<br/>　　后来血邪魔帝，回到净土宗，几乎是将净土宗大半弟子，坑杀在了净土佛寺之中，才化为了如此之多的怨灵。
<br/>　　这也是导致后来净土宗没落，最终消亡的原因。
<br/>　　“始祖第一式！”
<br/>　　面对四面八方而来的，楚风眠冷笑一声，凝聚凝聚在掌心之中，大手一挥，便是化为了一道龙爪，横扫而去。
<br/>　　轰！轰！
<br/>　　所有接近楚风眠的怨灵，纷纷被轰杀殆尽。
<br/>　　他这一次来净土佛寺之中，当务之急是要寻找人皇罐碎片，可懒得跟这些怨灵纠缠。
<br/>　　一招将周围的怨灵，轰杀殆尽后，楚风眠便是突然紧闭双眼，一息之间，再度睁开，便是看向了佛寺之中的一条通路，身形一闪，冲了进去。
<br/>　　在进入到了这净土佛寺之中后，楚风眠便是足以清晰的感觉到人皇罐碎片的方位了。
<br/>　　便是按照人皇罐指引的方向，向着净土佛寺深处冲了进去。
<br/>　　同时楚风眠也感觉到身后，无数魔族强者，也已经是冲入到了净土佛寺之中来。
<br/>　　这些魔族强者冲入其中，也是不断的向着佛寺深处冲去。
<br/>　　不过净土佛寺之中，通道足有上千道之多，几乎每一名进入的通道，都不一样。
<br/>　　楚风眠感觉的出，人皇罐碎片，如今就在这通道的尽头。
<br/>　　净土佛寺之中，倒是没有像是金身佛尊的陵墓之中，处处都是佛门禁制。
<br/>　　看来这其中的禁制，应该都是被血邪魔帝，亲手毁掉了，在里面许多地方，都有着魔意残留的痕迹。
<br/>　　魔意，怨灵，根本看不出，如今楚风眠是在一座佛寺之中去。
<br/>　　与其说是佛寺，不如说这里，更像是大量净土宗弟子的埋葬之地。
<br/>　　没有佛门禁制的威胁，楚风眠便是轻松许多，一路前行，碰到的怨灵，楚风眠全部轰杀。
<br/>　　“不知道那梦烟罗所说的剑气，到底是什么？”
<br/>　　楚风眠一路前行的时候，脑海之中也在思考这一件事。
<br/>　　便是之前梦烟罗说所的，这净土佛寺之中的危险。
<br/>　　这些怨灵，梦烟罗没有说，不过显然是这些怨灵，也难以是对楚风眠这个级别的强者，产生任何的威胁。
<br/>　　这些怨灵的实力，远不如同境界的武者魔族，虽然生前的实力强横，可是死后实力，十不足一。
<br/>　　哪怕是九劫古帝级别的怨灵，楚风眠都足以一招轰杀，所以不算威胁。
<br/>　　梦烟罗所提到的的威胁，却是剑气，楚风眠身为剑修，自然是无比在意这一点。
<br/>　　只是在这净土佛寺之中，楚风眠还没有察觉到有剑气的存在。
<br/>　　就在这思考之际，只见在楚风眠的前方，一道影子向着楚风眠猛冲而来。
<br/>　　“又是怨灵？”
<br/>　　楚风眠还以为是怨灵，凝聚灵力，看向前方，刚刚凝聚灵力，想要出手，突然发现，这却间看到的却是一个人影大小的剑气。
<br/>　　这一道剑气之上，蕴含这极致霸道的力量，似乎是察觉到楚风眠的一刻，一把对着楚风眠斩杀而来。
<br/>　　“这就是梦烟罗所说的剑气？”
<br/>　　楚风眠刚刚还是在想着这剑气的事，没想到这么快就让楚风眠碰到了。
<br/>　　这一道剑气，无情的向着楚风眠斩杀而来。
<br/>　　被这剑气锁定的一刻，楚风眠感觉到，他的身上都有些不寒而栗，仿佛是感觉到了极大的危险。
<br/>　　“剑出！”
<br/>　　祖龙至尊剑，从楚风眠背后天涯剑匣之中，猛然出鞘，被楚风眠握在手中，同样一剑斩杀而去。
<br/>　　楚风眠经历过无数次生死之战，对危险，已经是有着一定的预知。
<br/>　　这剑气，显然是对楚风眠威胁极大，才会让楚风眠感觉到如此危险，如此楚风眠也不敢轻视。
<br/>　　将一身灵力，注入到了剑锋之中，一剑冲天而起，斩杀而去。
<br/>　　“砰！”
<br/>　　剑气碰撞，楚风眠居然是感觉到了一股锋利强横的力量，要将他的身躯硬生生撕裂开来。
<br/>　　楚风眠身躯之中十二万九千六百枚血肉晶体，在这一刻死死的凝聚了起来，化为了一道完美的身躯，承受着剑气锋利的力量。
<br/>　　“这是什么东西？一道剑气，怎么会有着如此可怕的力量？”
<br/>　　楚风眠脸色震惊无比。
<br/>　　他刚刚已经算是催动了大半的力量了，哪怕是四五位九劫古帝，一齐出手，刚刚楚风眠的剑光，都足以将他们的轰击，全部化解。
<br/>　　可是跟这剑气碰撞的一刻，楚风眠居然是都承受了如此巨大的压力，他的剑光都被无情的碾碎。

<br/>　　楚风眠还在震惊之时，那一道剑气，似乎是没有想要放过楚风眠的想法，又是向着楚风眠斩杀而来。

<br/>　　可眼前这一道剑气，却是如同活物一样，居然是懂得剑术，在碰到楚风眠的一刻，施展出剑术，斩杀而来。
<br/>　　施展出剑术来，这一道剑气的威力，便是恐怖了数倍，
<br/>　　铺天盖地的剑气，显然是想要将楚风眠，一举斩杀。
<br/>　　“这剑气，难道是无上剑气？居然懂得剑术，不管如何，这绝对的一剑好宝物，对我有着大用，想将这一道剑气降伏再说！”
