　　在David和Bob的注视下，毒天使走上前去，双手一翻，双手手指间各多了三颗如兵乓球大小的圆球。
<br/>　　兜帽下，毒天使抬起头看向在停车场汽车与汽车间不断做着战术运动的罗昊、李然这些人，嘴角露出勾起一抹冷笑，手一扬，将夹在手指间如兵乓球大小的黑色圆球丢了出去。
<br/>　　嘭！嘭！嘭！嘭！嘭！嘭！
<br/>　　黑色圆球在落地的瞬间，便是爆裂开来，一团团淡蓝色的烟雾升腾而起，而在这淡蓝色烟雾的掩护下，每颗黑色圆球中都是有着一条通体碧蓝带有深蓝色条纹，只有人小手指粗细，长度没有超过十五公分的细蛇贴着地面快速向前方游去。
<br/>　　“杀了我的‘小黑龙’是要付出代价的。”毒天使脸上挂着冷笑，说道：“虽然这些小家伙比不上‘小黑龙’的毒性，但是被咬上一口，如果没有我的解药，也是会死的。”
<br/>　　罗昊虽然不知道这突然被人扔出来的黑色圆球是什么，从黑色圆球中爆出来的蓝色烟雾又是什么东西，但是这淡蓝之色显然是在警告旁人，不要轻易靠近我，不然是要付出惨重代价的。
<br/>　　“退！”
<br/>　　罗昊当即命令周围人立刻后退，远离这些淡蓝色的烟雾；但是罗昊只注意到了被毒天使扔出来的圆球，还有圆球爆裂开来之后飘散出来的淡蓝色烟雾，而没有注意到在圆球爆裂开时候从圆球中爬出来的蓝色小蛇。
<br/>　　其实也不是罗昊大意，而是现场情况如此混乱，加上那些蓝色小蛇细小而且行动迅速，如果不是事先知道的话，基本是不可能在第一时间发现这些蓝色小蛇的。
<br/>　　“看来这次连老天都帮我。”
<br/>　　感觉到风向的变化，毒天使的眼睛变得更加冷冽起来。这些淡蓝色的烟雾虽然不会致命，但在吸入之后却是会让人的神志变得模糊，反应变得迟钝。
<br/>　　“嘶！”
<br/>　　周柏突然感到从自己左小腿上传来一阵尖锐的刺痛，忍不住倒吸一口冷气，低头往脚下看去，发现了自己脚边的蓝色小蛇。周柏顿感不妙，没有丝毫犹豫抬起右脚，将蓝色小蛇的三角形蛇头踩烂，按住耳麦，说道：“毒天使在附近，重复，毒天使在附近，注意地面，有蓝色小蛇！”
<br/>　　这种通体碧蓝带着深蓝色条纹的小蛇毒性远不如“小黑龙”那么猛烈能瞬间致人于死命，但是它的毒性也是不容小觑，仅仅只是片刻间，周柏就觉得自己被咬伤的左脚产生了明显的麻木感。
<br/>　　“糟糕！”
<br/>　　周柏立刻退到一个角落中，坐在地上，撩起自己的裤脚查看伤势。蓝色小蛇在周柏左小腿上留下了两个细小的牙印，在牙印周围一片的皮肤已经发黑，而且这种黑色正在向四周扩散开来。
<br/>　　作为龙隐部队的一员，周柏自然是学过在野外遇到毒蛇咬伤要如何自救处理，从左腿外侧的急救包中拿出一根皮管，在自己膝盖处扎紧，阻止蛇毒进一步扩散，接着拔出身上的CQB格斗刀，深吸一口气在被蛇咬伤的位置划开一道口子，瞬间中毒发黑的毒血便是从周柏自己划开的创口中流了出来。
<br/>　　周柏没有见过这种咬伤自己的蓝色小蛇，不过既然是出自毒天使之手，当前已知的抗蛇毒血清肯定是对这种蛇毒无效的。
<br/>　　随着毒血被排出，周柏很明显感觉到自己左腿的麻木感减弱了不少，可是周柏并没有天真的认为自己这样就能放干净自己体内的蛇毒，如果这么容易就能把蛇毒排干净，那就不是毒天使的手笔了。
<br/>　　有了周柏的提醒，其他人倒是没有再中招，剩余的五条蓝色小蛇被及时发现，然后被杀死。
<br/>　　“哦？看来这样还杀不死他们。”
<br/>　　毒天使轻轻挑了挑眉头，双手一翻又是六颗黑色圆球出现在他手中，冷冷说道：“那么再来一次，你们还能活下来吗？”
<br/>　　六颗黑色圆球再次被毒天使扔了出去。
<br/>　　嘭！嘭！嘭！嘭！嘭！嘭！
<br/>　　六颗黑色圆球爆裂开来，又是六团蓝色烟雾升腾而起，又有着六条跟刚才一样的蓝色小蛇从黑色圆球中爬了出来，迅速的朝着罗昊、李然他们的方向游去。
<br/>　　停车场上的蓝色烟雾越发浓厚起来，尽管罗昊第一时间命令小队后撤，但是奈何他们处于下风口，即便是第一时间后退，可他们还是或多或少的吸入了一些这种淡蓝色烟雾。
<br/>　　六个人都是感觉到了不同程度的晕眩感，这对于六个人来说是个非常不好的消息，不说对面还有黑水安保部队的雇佣兵和毒天使，就是地上这些蓝色小蛇都很有可能要了他们的命。
<br/>　　至于那个被慕容飞抓回来的鸭舌帽男，在刚才交火的过程中就被几颗流弹击中，已经成了一具尸体，不过好在慕容飞已经是从对方嘴里撬出了一些东西来，也不至于完全没有收获。
<br/>　　罗昊架起被蛇咬伤的周柏，继续往后退，可是被风吹来的淡蓝色烟雾还是逐渐逼近他们，而且他们还需要躲开黑水安保部队雇佣兵那边射来的子弹，撤退动作自然也是受到了一些影响。
<br/>　　就在此时，数颗白色圆球从罗昊他们身后飞来，砸落在他们身前。
<br/>　　嘭！嘭！嘭！嘭！嘭！
<br/>　　白色圆球爆裂而开，一团团白色的烟雾升腾而起。
<br/>　　当淡蓝色的烟雾遇到那一团团白色烟雾的时候，就像是酸和碱相遇了一样，瞬间被中和掉了。
<br/>　　停车场上，白色烟雾和淡蓝色烟雾都是消失不见了。
<br/>　　原本志在必得的毒天使，看到停车场上这突兀的一幕，眼睛一眯，咬着牙冷声说道：“阴魂不散的家伙，果然是追到这里来了。”
<br/>　　一身白色装束的巫师出现在罗昊、李然六个人身后，看到被罗昊架着的周柏，扫了眼周柏发黑的左脚，快步上前，对周柏问道：“中毒了？”
<br/>　　周柏此时的嘴唇都是有些发黑发紫了，但意识还能保持清醒，点了点头，说道：“真是倒霉，被蛇咬了一口。”
