        “那就睡一起好了。”傅昼景看着童筱颖脸红的样子，温柔地笑了笑。
<br/>
<br/>        蓦地，童筱颖抬头仰望着傅昼景的眼睛时，不禁惊喜得心跳加速。
<br/>
<br/>        领证后的这些天，他俩一直都是分房睡。
<br/>
<br/>        这一刻，童筱颖心底突然有了一丝期许。
<br/>
<br/>        她期待在他温暖的怀中，搂着他安心入眠。
<br/>
<br/>        搬完家后，傅昼景打扫了一会儿，看了看手腕上的手表上的时间，便将剩下的家务交由童筱颖打理，自己不得不先去公司上班。
<br/>
<br/>        哪怕只是走几步就能抵达家门口，童筱颖仍旧像以前那样，微笑着送傅昼景出门。
<br/>
<br/>        在去公司的路上，有一家资历深厚且开了十年的老当铺。
<br/>
<br/>        傅昼景站在当铺门口，徘徊了一会儿，还是只身走了进去。
<br/>
<br/>        他一边摘下自己手腕上的手表，一边从公文包里掏出了一个长方形的礼盒，放到了柜台上。
<br/>
<br/>        坐在柜台前的老板，拿起他递过来的这两样东西，仔仔细细地验了验后，接着说道：“手表三万，钢笔一万五。”
<br/>
<br/>        “只能典当这么点钱吗？”傅昼景不悦地皱起了眉头。
<br/>
<br/>        “你的手表和钢笔确实是高端品牌，但是，我给你三折的典当价已经算是很高的了。要是换了其他典当店，两折都没有，只有一折给你！”老板一脸“你爱典不典”的样子。
<br/>
<br/>        傅昼景凝眉深思了一会儿。
<br/>
<br/>        下个月起，就要开始还贷了。他每个月的工资，只能全部用来还贷还少了点，他和筱颖总得要花一些生活费。
<br/>
<br/>        为了筱颖，傅昼景还是把手表和钢笔都典当了出去。
<br/>
<br/>        手表是他那个跑路的朋友赚了第一桶金后送给他的，至于钢笔，则是六年前他和季思妍谈恋爱时，季思妍送给他的生日礼物。
<br/>
<br/>        季思妍是个千金大小姐，当年她跟他说，她给他买礼物的钱，是她从小到大攒的压岁钱剩余的。
<br/>
<br/>        她觉得自己把钱花在他身上值，因为她爱他。
<br/>
<br/>        现在想想，傅昼景觉得自己挺窝囊，甚至狼狈到要靠典当初恋送给自己的生日礼物来维持生计。
<br/>
<br/>        他后脚刚离开当铺，不远处，季思妍从拐角走出来，若有所思地注视着他离去的背影。
<br/>
<br/>        阿景去当铺里做什么？
<br/>
<br/>        季思妍见傅昼景走远后，转身便进了刚刚那间当铺，询问傅昼景的事情。
<br/>
<br/>        老板起初还遮遮掩掩的，不肯多说有关于傅昼景的一句话，直到季思妍拿出银行卡出来要买回刚刚傅昼景所典当的东西时，老板这才笑不拢嘴地对季思妍格外殷勤起来。
<br/>
<br/>        “我看那男人啊！是很缺钱用的样子！”老板一边给季思妍刷卡，一边碎碎念叨道。
<br/>
<br/>        季思妍拿着手表和钢笔，黯然神伤地垂下了眼帘。
<br/>
<br/>        这手表是什么来历，她没兴趣，但她在乎的是这支钢笔。
<br/>
<br/>        阿景就这样随便将她曾经送给他的生日礼给典当了出去，所以，在阿景的心里，她已经没有分量了吗？
<br/>
<br/>        不！
<br/>
<br/>        不会的！
<br/>
<br/>        阿景一定还爱着她，他一定是遇上了什么困难，迫不得已才典当她送给他的生日礼物的。
<br/>
<br/>        老板刷完卡后，将卡和单子一并双手呈上，送到了季思妍的手中，并客套地说了一句：“欢迎下次光临！”
<br/>
<br/>        季思妍收好自己的银行卡后，给哥哥季运恒打了通电话。
<br/>
<br/>        电话刚一接通，季思妍便迫不及待地跟哥哥季运恒说道：“哥，你帮我查查，阿景他最近是不是遇上了什么经济困难？”
<br/>
<br/>        “我的亲妹啊！你怎么到现在还想着傅昼景啊！你要把心思花在他大哥傅夜沉身上才对！他傅昼景算个毛？傅家的私生子而已，而且连傅氏集团一毛钱的继承权都没有。”季运恒喋喋不休地对傅昼景的事情说三道四。
<br/>
<br/>        其实，最主要的是，他不看好傅昼景，不希望妹妹把青春浪费在一个没用的男人身上。
<br/>
<br/>        季思妍当然懂哥哥季运恒的心思，自从傅夜沉退役从商的消息不胫而走后，这上流社会的名媛圈里，各家千金名媛都对傅夜沉蠢蠢欲动。
<br/>
<br/>        谁不想成为傅家的大少奶奶，将来和傅夜沉平分傅氏集团？
<br/>
<br/>        先不说傅夜沉的实力背景，就说他那个人，俊逸脱俗、英姿飒爽、帅得能令女人合不拢腿不说，就连傅昼景都无法与他比拟。
<br/>
<br/>        可是，即便如此，季思妍心里仍旧放不下傅昼景。
<br/>
<br/>        这是为什么？
<br/>
<br/>        明明她已经不是当年那个心思单纯的女孩了。
<br/>
<br/>        明明她也想去征服傅夜沉，成为傅家的大少奶奶……
<br/>
<br/>        可她为什么就是对傅昼景心有不甘呢？
<br/>
<br/>        不甘心他就这么把她给忘了……
<br/>
<br/>        不甘心他就这么不在乎她了……
<br/>
<br/>        季思妍深吸了口气，没好气地对哥哥季运恒怒吼：“哥，你若还希望我能吸引傅夜沉的注意，能让冷季两家联姻，你就最好帮我查查阿景的事情。”
<br/>
<br/>        “行了行了，我帮你查就是！噢，对了。季雅姗最近是不是被什么男人包养了？我一朋友跟我说，曾看到她上过一辆价值千万的豪车，但是那辆豪车是外地牌照，我查不到真实车主的信息。”季运恒又忍不住跟季思妍八卦起那个同父异母的妹妹季雅姗来。
<br/>
<br/>        自从那次季雅姗被绑架九死一生地活着回来后，和他们家里的人关系就更加僵了。父亲那次确实没有拿钱出来去赎她的命，是他母亲不让，所以，季雅姗一定恨死他母亲了。
<br/>
<br/>        “我也不太清楚，回来之后，我也就见过她一次，她还是那臭脾气。总而言之，她的事情，我们就不要管了。反正，她就从来不会记得我们俩这哥哥姐姐的好。”季思妍不痛不痒地说。
<br/>
<br/>        她和哥哥也真是的，去瞎操心一个小三的女儿做什么。
<br/>
<br/>        “那行吧！我先挂了，待会查到傅昼景的资料后，我就发给你。”季运恒说完，便将电话给挂了。
<br/>
<br/>        季思妍刚到公司，傅昼景便拿了一叠资料，让她去傅氏集团总部大厦交给他大哥傅夜沉过目签字，然后再带回来。
<br/>
<br/>        正巧，他给了她接近傅夜沉的机会。
