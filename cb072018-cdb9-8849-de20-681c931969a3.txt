　　第265章
<br/>　　青牛山散伙后，蛤蟆寨众兄弟流落四方。高飞虎高飞豹兄弟护送山寨的老弱前往南方，刘关张赵则是浪迹天下，而大部分人则是像熊大熊二一样又拉了一伙弟兄占山为王。
<br/>　　大家都没忘了程大雷，都在用各种各样的手段打听程大雷。直到这次杨龙停被杀，公主和亲的事情传遍天下，诸人才终于知道程大雷的下落。
<br/>　　于是，各山各寨的魔王，浪迹天下的游侠……等等诸人，就像雨后的春笋纷纷冒了出来，杀出关外。
<br/>　　程大雷让秦蛮，徐神机等在峡山，可是二人怎么会安心在峡山等待，于是带刀骑马出关，一路上遇到的人越来越多，都是曾经蛤蟆寨的弟兄，一同喝过酒，斩过敌头，相逢自然有说不完的话。
<br/>　　一到关外，听说大当家把公主抢了。嗯，很好，很好，符合大当家的风格。一听关外草原上群雄汇聚，各方势力都要杀大当家，诸人就像被踩住电门一样跳了起来，兄弟聚在一起，直扑向草原腹地。
<br/>　　恰在此时重逢。
<br/>　　只见一千兄弟骑马奔来，踏破烟尘如龙，直直的杀进乱军的腹心。这些人就像是地狱的魔王，下山的猛虎，没动手就已经杀红了眼。
<br/>　　而这些人口中高呼的都是同一句话。
<br/>　　“大当家在哪里，大当家在哪里！”
<br/>　　宫符大惊：“他们是什么人，他们是什么人！”
<br/>　　若说他们是匪类，可他们进退有据，不输饱经沙场的军人，可他们身上那股凶悍血腥绝不是士兵能有的。
<br/>　　来一杀一，来百杀百，这伙人冲过来，竟无人能挡。
<br/>　　程大雷站在马车上看去，赵子龙顶级武将，高飞虎高飞豹顶级山贼，小白狼顶级山贼，关鱼，张肥，刘悲皆已升到顶级……
<br/>　　战场是最好的教室，死亡是最好的老师，青牛山一战，大家虽然打得很苦，但实力却都得到提升。
<br/>　　现在一眼看过去，顶级人物数十人，至弱也是优秀等级。
<br/>　　一路长驱直入，锐不可当，直接杀进马车周围。包围程大雷的敌人被冲散，大家将马车团团围住。
<br/>　　“大当家，大当家！”众人围着马车高呼。
<br/>　　“好。”程大雷从怀里扯出一物：“秦蛮，扯旗。”
<br/>　　“是！”
<br/>　　秦蛮将程大雷丢过来的东西挂在长枪上，迎风飘展，天蟾吞月图再次升空。
<br/>　　“吼！”
<br/>　　看着再次飘扬的蛤蟆寨大旗，众人心底宛若被点燃一团火焰，发出齐声怒吼。
<br/>　　军威如虎。
<br/>　　宫符被这一幕震惊，他恍然明白一个事实，程大雷本质上其是一个山贼，只要是山贼就有同伙。他不需要一个人代表一支队伍，他身后本就有一支队伍。
<br/>　　“大当家，咱们杀到戎族王城，把他们老窝端了吧。”
<br/>　　高飞豹兴奋道，整个帝国都怕戎族，但蛤蟆寨的山贼是不怕戎族的，毕竟，青牛山杀敌十万，众人砍的脑袋多不胜数。
<br/>　　“你快别扯淡了。”程大雷撇撇嘴，长剑一挥：“收兵。”
<br/>　　这次追击出来的只是戎族的小股部队，人家大军还未出动，靠一千人就想冲人家老巢，这不是脑袋蠢不蠢的问题，这是出门有没有带脑子的问题。
<br/>　　以大旗为号，众人向外冲杀。
<br/>　　宫符和呼延力看到这一幕，脑中冒出同一个想法：绝不能让此贼离开。
<br/>　　“束华，你带人冲过去，别个我不问，我只要程大雷的性命。”
<br/>　　“罗酒，要杀程大雷，不能让他活着离开。”
<br/>　　二人几乎是同时下令。
<br/>　　帝国和戎族很罕见的第一次联手，为的是同一个目的，杀死一个共同的敌人。
<br/>　　罗酒，在鱼龙卫中代号醉汉，在鱼龙卫内他有两件事最为出名，一是好饮，二是力大，酒后力气更大。
<br/>　　束华，使一条镔铁大棍，昨夜戎族的火神节，他就是最后夺魁之人。
<br/>　　二人都是有本事的人，都有一腔傲气，想要在此地诛杀程大雷。
<br/>　　两支队伍从不同的方向奔来，奔向一个共同的目标。
<br/>　　赵子龙接住了束华，而接住罗酒的是张肥。
<br/>　　猛张飞的后人现在已经有了几分先人风范，手中丈八蛇矛抡开，就一个字，砸。
<br/>　　力气越来越大，砸得越来越猛，罗酒完全懵住，根本来不及以招式还击，只是把铁鞭照在头顶格挡。一下，两下……他身上好像顶着一座山，双臂感觉要被震裂一般。
<br/>　　咚！
<br/>　　罗酒被砸落马下，以力大出名的他，败在力气上。
<br/>　　而这个时候，赵子龙已轻取束华性命，一招蹬里藏身，长枪洞穿对方喉咙。
<br/>　　呼延力和宫符窥到这一幕，脸上浮现惊愕的表情，在短时间，他们手下的两员猛将已经分别折了。而此时草原上的马匪以及关内进来的江湖群雄，看情形不对，已经在悄悄往外撤。
<br/>　　“大当家，要不要我们把他们指挥官杀了！”关鱼道。
<br/>　　“咦，你办得到么？”程大雷。
<br/>　　“吾观其插标卖首耳。”关鱼用睥睨的神情道。
<br/>　　宫符和呼延力同时一惊，这莫名其妙的杀气是怎么回事。
<br/>　　“下次，下次，下次有你表现的机会。”程大雷挥挥手中的剑：“弟兄们，撤，撤。”
<br/>　　大军围成一团，向外冲杀，当真是如狼似虎，横扫千军，直直的杀出一条道路。
<br/>　　鱼龙卫以及呼延力率领的戎族骑兵，只能看着他们的背影消失在荡起的烟尘中，竟无一人去追赶。
<br/>　　蛤蟆寨这帮人浩浩荡荡离去，一路急行军，只用了两三日距离，便逼近边关。
<br/>　　这一夜，大家在关外扎营，点燃篝火，吃着随身携带的干粮。
<br/>　　程大雷问明白了诸人的情况，大部分都是重新拉了一伙弟兄，占山为王。
<br/>　　“大当家，咱们接下来去哪儿？”
<br/>　　“不如咱们还回青牛山，聚义占山，把兄弟们聚在一起。”高飞虎。
<br/>　　“青牛山不能去啦，幽州刚经历战火，现在十室九空，没多大发展前途。”程大雷。
<br/>　　“那大当家准备去哪儿，你去哪儿我们跟着一起去。”
<br/>　　“我……”程大雷看着远处：“我还是想往西去的。”
<br/>
