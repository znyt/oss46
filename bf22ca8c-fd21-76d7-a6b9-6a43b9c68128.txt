二十多年前的帝都繁花似锦，柳璎珞冠盖满京华，林水瑶从神秘的地方而来，升级打怪，玩转整个帝都，这两个都是奇女子。
<br/>
<br/>        陆司爵伸手，手指轻轻的抚过了柳璎珞那张清冷绝色的小脸，“林水瑶的女儿一直没有找到。”
<br/>
<br/>        “那是因为老夫人拿走了林水瑶留下的订婚信物，让我们无从查起！司爵，你就是偏心，你只爱璎珞给你生的儿子，我们子羡的婚事你却一点都不放在心上！”柳招娣说出了心里的不满。
<br/>
<br/>        陆司爵抬头，幽沉的眸子落在了柳招娣的脸上，“如果我不爱子羡，我第一件事就应该把你从陆夫人的位置上撤下来，免得你将子羡给养废了，做事情之前先动动脑子，你用这种下作的手段对付夏夕绾，别说你不是夏夕绾的对手，就算你赢了，又将子羡置于何地？”
<br/>
<br/>        “我…我…”柳招娣语结了，她知道陆司爵是不悦了，一个浸淫商场这么多年的掌权者，一旦动怒，那可是雷霆万钧，她吓得一句完整的话都说不出来了。
<br/>
<br/>        陆司爵起身，他最后看了柳招娣一眼，“如果你不想做陆夫人了，告诉我一声，我不缺陆夫人，以后给我安分一点。”
<br/>
<br/>        说完，他拿着那幅画离开了书房。
<br/>
<br/>        柳招娣紧紧的拽着拳，眼眶发红，今天夏夕绾夜闯陆家，让陆司爵将她打回了原型。
<br/>
<br/>        这一招真狠！
<br/>
<br/>        没错，这些年她就是柳璎珞的影子！
<br/>
<br/>        一个歌女生的女儿，如果不是因为她从小患病，柳璎珞根本就进不了柳家，原本柳璎珞只是她的血灌子，是她的影子。
<br/>
<br/>        但是谁知道柳璎珞越长大越风华，直至创下Fly，每一季的珠宝单品一经上市就被横扫一空，惊艳了整个珠宝界，冠盖满京华。
<br/>
<br/>        她再也不是她的影子了，她反而成了她的影子。
<br/>
<br/>        这真是最大的刺激和笑话。
<br/>
<br/>        但是，这些年柳招娣还是甘之如饴，因为她爱陆司爵。
<br/>
<br/>        她无法忘记自己第一次见到陆司爵的画面，那天的陆司爵从商务豪车里下来，他一身手工版的黑色西装，身后跟着一大批挂着蓝牌的公司高层，他信步走在万人中央，璀璨万丈，让人移不开目光。
<br/>
<br/>        三十年前，帝都之子陆司爵，让多少人春心荡漾？
<br/>
<br/>        谁能逃脱得了他的光芒和魅力，她也不能。
<br/>
<br/>        她也对陆司爵一见钟情。
<br/>
<br/>        柳家养在深闺的倾城佳人，一时将心丢失在了这个男人身上。
<br/>
<br/>        这么多年过去了，柳招娣看着陆司爵依旧心动。
<br/>
<br/>        但是，他对她依然薄情冷酷。
<br/>
<br/>        柳招娣突然觉得心口疼，她大声叫道，“来人，快来人！”
<br/>
<br/>        ……
<br/>
<br/>        帝吧里。
<br/>
<br/>        顾夜瑾来到了豪华包厢，他看了一眼此刻正慵懒倚靠在沙发背里的陆寒霆，漫不经心的给自己倒了一杯酒，“听说夏夕绾夜闯了你们陆家，将柳招娣气的心口疼，私人医生已经进去一波又一波了。”
<br/>
<br/>        “那个私人酒庄刚才也被挖土机给推平了，纸醉金迷的人间天堂一下子变成了一杯黄土，寒霆，你爸果然是你爸，他一出手就知道有没有啊。”
