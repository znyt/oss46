　　“我长这么大，从来没有人敢打我，林峰你死定了，我要让我爸爸杀了你！”
<br/>　　啪！

<br/>　　林峰恶狠狠道了句，包括吴倩倩在内每个人都是面色变了一下，都被林峰的样子给吓到了。
<br/>　　林峰之前杀了几个丧尸，渐渐的说话之中有了一些杀气，顿时把他们都给吓住了。
<br/>　　吴倩倩也被吓得一句话都不敢多说。
<br/>　　林峰懒得搭理她，旋即就把苏云给抱进教室。
<br/>　　“老师这是怎么了？”

<br/>　　林峰道。

<br/>　　一个女生拿出了一瓶矿泉水。
<br/>　　“水这个东西，大家以后节约着用，因为我感觉，这一次灾难可能会持续很长时间。”
<br/>　　林峰提醒了大家一句。
<br/>　　说着就给苏云喂了点水。
<br/>　　“林峰，我们接下来怎么办。”
<br/>　　孙畅问道。

<br/>　　“她疯了？这么大力气？”
<br/>　　陈勾皱着眉头走了过去不过很快他面色难看的回来。
<br/>　　“她变成丧尸了，看来被丧尸抓伤的确会变成丧尸。”

<br/>　　林峰道。
<br/>　　孙畅等人都沉默不语。
<br/>　　“可是，怎么清理？就靠我们几个？要知道，我们整个学校将近一千人啊！”
<br/>　　林峰一字一句道：“一层一层的清理！不然我们除非从天台上跳下去，否则永远被困死在这里！！没有食物没有水，根本撑不了多久，另外大家把所有水和食物都拿出来，可能我们要在这里呆一段时间了。”
<br/>　　“嗯。”
<br/>　　每个人都开始行动起来，包括那些死去的同学。
<br/>　　“在这里要想活下去，每个人都要体现自己的价值，不然就是一个废物，而废物，往往都是可以被抛弃的。”
<br/>　　林峰淡然的道了句，让每个人都是面色变了一下，纷纷开始利索的行动起来，吴倩倩也是，她是真的被林峰给吓到了。
<br/>　　林峰冷笑，这群人其中许多人都是太子爷都是小公主，娇生惯养的，饭来张口衣来伸手，之前也就算了，毕竟和林峰没有什么关系，现在不一样了，没有人是什么太子爷，不想死，每个人都必须要发挥自己的能力。
<br/>　　“林峰，你刚才在丧尸脑袋里找什么呢。”
<br/>　　陈勾走来好奇道。
<br/>　　林峰给了他一个眼神，然后把孙畅也给叫到教室外。
<br/>　　林峰扫了他们一眼，目光落在孙畅身上。
<br/>　　“我能不能相信你。”
<br/>　　孙畅略一沉吟，“我是个分得清大局观的人。虽然我之前没和你说几句话，但是你放心，我对你是没有什么偏见的。”
<br/>　　“嗯。”
<br/>　　林峰不再废话，拿出了丧尸脑袋里找到的东西。
<br/>　　“晶石？？”
<br/>　　陈勾愣了。
<br/>　　“这就是你在丧尸脑袋里找到的？”
<br/>　　“没错，每个丧尸脑袋里都有这个东西，这很显然，是这群人变成丧尸后才出现的，算是一个异变吧。”
<br/>　　林峰道。
<br/>　　“不会吧，真有这玩意。”

<br/>　　“你知道这东西？”
<br/>　　林峰愕然的看着他，对方似乎很清楚。
<br/>　　“废话，那么多小说里不都这么写的嘛，这玩意说不定还可以增加我们实力。”说着陈勾就把那个玩意放在嘴巴上。

<br/>　　“谁说我要吃了，看着吧。”
<br/>　　陈勾咔擦一下就捏碎了那个晶石，然后一股翡翠颜色的液体就流入了他嘴巴之中。

<br/>　　陈勾还在那里品味，很快面色一喜。

<br/>　　孙畅和林峰对视一眼，若是之前，他们肯定以为陈勾和他们在聊什么游戏，但现在他们知道，这估计就是事实了。
