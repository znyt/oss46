　　萧逸炼丹的速度相当快。
<br/>　　毕竟他拥有完美的炼药术，不用担心炼药失败。
<br/>　　加之又有四品炼药炉离火金炉，绿色武魂的控火。
<br/>　　不出片刻，九粒金灿灿的丹药便从丹炉中出现。
<br/>　　这便是他渴望了许久的金脉丹。
<br/>　　快速地服下一粒，丹药立刻化为一股神奇的力量，包裹在第一条主脉中。
<br/>　　主脉如同被金箔包裹一般，几乎以肉眼可见的速度变为一条金脉。
<br/>　　萧逸尝试利用第一条主脉催动真气，下一秒，表情变得惊讶至极。
<br/>　　第一条主脉催动真气的速度和强度，竟比以往强了十倍都不止。
<br/>　　如果将真气比作水流，那么主脉就是一条不断有水流淌的小河。
<br/>　　而被金脉丹增强后的主脉，简直就是一条汹涌崩腾的大江大河。
<br/>　　两者根本没有可比性。
<br/>　　一条金脉，比九条主脉加起来还要强悍。
<br/>　　萧逸毫不犹豫将其余八粒金脉丹吞入腹中。
<br/>　　不出片刻，九条主脉全部化为了金脉，变得更加坚韧与宽广。
<br/>　　这也代表着，一条主脉能蕴含的真气将更多。
<br/>　　萧逸再次催动两大武魂，吸收这里的浓郁灵气，而后转化为真气，填充更加宽广的九条主脉。
<br/>　　当九条主脉完全填满后，萧逸明显感觉到，自己的实力发生了质的变化。
<br/>　　如今的自己，就算不使用任何手段和底牌，战力也足以媲美先天五重武者。
<br/>　　一个后天九重，竟然拥有媲美先天五重武者的实力，足以让人惊骇。
<br/>　　这时，萧逸看了眼那光幕，恰好看到铁达大叔被天星荒鳄击飞的场景，顿时脸色一变。
<br/>　　多年前，狂血玄君便是经常坐于这个蒲团上，边修炼，边通过光幕查看宫殿内被囚禁的妖兽情况。
<br/>　　蒲团前那卧榻，有一层禁制，可让狂血玄君自己瞬间出现在洞府中的任何地方。
<br/>　　.....
<br/>　　宫殿上层，浅沫看着那朝自己点来的手指。
<br/>　　手指的速度很慢，但给人的感觉却是更强烈的折磨。
<br/>　　手指上那尖锐的指甲，粗糙而带着疙瘩的皮肤，让人毛骨悚然。
<br/>　　死亡只在一瞬间，但面临死亡的感觉被无限放慢，却会让人抓狂。
<br/>　　浅沫如今便是这样的心情，当想到自己下一秒就可能会被雷电劈成粉末，身体开始发抖。
<br/>　　此时此刻，她忽然想起了某个背影。
<br/>　　她曾躲在这个背影身后，看着背影的主人剑舞飞扬，将一头头荒鳄击杀，冰冷的妖血飞溅，却连一滴都无法沾到她身上。
<br/>　　她在背影身后，始终只有温暖。
<br/>　　她曾趴在这个背影上，看着背影主人那坚毅而俊俏的面庞，肆意游走于无数荒鳄中。
<br/>　　荒鳄无数，凶残狂暴，却无法伤她分毫。
<br/>　　一个个画面在她脑海中飞速掠过。
<br/>　　此刻，她不知哪来的勇气，在那恐怖的手指即将点到她时，猛地大呼了一声，“易霄，救我。”
<br/>　　话音刚落。
<br/>　　一道身影凭空出现，一拳将天星雷鳄轰飞。
<br/>　　在场所有猎妖师尽皆一愣，而后便是瞳孔一缩。
<br/>　　“好强。”李元等人惊呼一声。
<br/>　　人影戴着个面具，披着宽大的袍子，看不清样子，但似乎年龄不大。
<br/>　　人影自然是萧逸。
<br/>　　只不过，自己毕竟和北山慕容家有仇，而且这里猎妖师如此多，人多口杂，还是谨慎些掩盖了样貌为好。
<br/>　　北山慕容家的商行遍布整个北山郡，眼线可不少。
<br/>　　袍子是萧逸乾坤袋里本来就有的干净衣服。
<br/>　　倒是那面具，名唤幽魂面具，乃是狂血玄君的宝物之一。
<br/>　　与那些高品阶的天材地宝一样放在黑盒子中，有诸多妙用，萧逸随手便戴上了。
<br/>　　这时，众猎妖师发现，自己头上的不时凭空降临的雷电竟然消失了。
<br/>　　一看才知道，人影与天星雷鳄酣战，竟逼得天星雷鳄再也分不出心来释放漫天雷电。
<br/>　　“好厉害，天星雷鳄竟被完全压制到下风。”
<br/>　　李元等人面露惊骇，当看到人影手上那火焰缠绕的手套时，更是面露狂热，“灵器，那是灵器。”
<br/>　　另一边，萧逸与天星雷鳄的战斗，无数火刃与漫天雷电不停地碰撞，又在碰撞中互相抵消，谁也奈何不了谁。
<br/>　　“哪来的人类，竟敢坏我大事？”天星雷鳄愤怒地吼着。
<br/>　　萧逸不语。
<br/>　　他早已使用了升龙，配合上烈焰手套，战力达到了先天九重，未能达到洞玄战力。
<br/>　　先天境与洞玄境的差距极大，就算用灵器增幅，也无法跨越这层差距。
<br/>　　但毫无疑问，无论是萧逸还是天星雷鳄，都必然是洞玄境之下的最强战力，无限接近洞玄。
<br/>　　“人类，不要太嚣张了。”天星雷鳄见萧逸不回答，更加愤怒，漫天雷电在他手中猛地凝聚起来，顷刻间成了一道暗黑色的凶雷。
<br/>　　“暗雷破。”天星雷鳄大手一挥，一道比寻常雷电粗了十倍不止的暗黑色雷电劈下。
<br/>　　萧逸眉头一皱，明显从这道凶雷中感受到威胁。
<br/>　　“噬火百刃。”萧逸心中暗喝一声，五十道火刃同时劈向雷电。
<br/>　　噬火百刃，以往萧家的长老练了大半辈子，成功同时凝聚的火刃，大多不过在五六十道之间。
<br/>　　但萧逸如今在灵器的增幅下，也勉强使用了出来。
<br/>　　轰...轰....轰....轰。
<br/>　　火刃与凶雷的碰撞，产生一道道轰鸣。
<br/>　　每一道火刃，都会抵消掉凶雷的一部分力量。
<br/>　　但，直到五十道火刃完全消散，凶雷却仍旧没有消散，直直地打在萧逸身上。
<br/>　　“噗。”萧逸当众吐出一口鲜血，暗道，“被我的噬火百刃消耗了大半威力，劈在我身上却仍旧如此强大，这暗雷怕是很厉害的招式。”
<br/>　　“哼。”天星雷鳄见萧逸受伤，发出了狰狞的笑意，再次喝道，“暗雷破。”
<br/>　　萧逸反应极快，这次，不等它释放，他便立马凝聚了一把火剑，快速朝它劈去。
<br/>　　崩山斩！
<br/>　　“嗯？”天星雷鳄一惊，明显从火剑上感受到一股崩山裂石的厚重之势，立刻覆盖一层雷电在身上以作防御。
<br/>　　嘭的一声。
<br/>　　天星雷鳄身上的雷电顷刻间破碎，而后，身上那狰狞的荒鳄皮肉，更是被劈得血肉绽裂。
<br/>　　“噗。”天星雷鳄大口鲜血喷出，同样受伤。
<br/>　　“怎么可能，竟然能伤我？”天星雷鳄满脸不可置信之色，但很快，它再次恢复了狰狞，一道暗雷打出，击退了萧逸。
<br/>　　无数火刃，再次与漫天雷电对轰起来。
<br/>　　另一边，一众猎妖师们愣愣地看着战斗。
<br/>　　他们之前抵挡雷电，已经受了不轻的伤势，根本帮不上忙，就算勉强一战，也只会是被天星雷鳄秒杀的份。
<br/>　　“那家伙好强，我们破玄城的猎妖师中，何时出现了这样一个强者。”
<br/>　　众人疑惑地说道。
<br/>　　铁达大叔早就回到了林飞等人身边，紧紧地盯着萧逸的背影，喃喃低声道，“那强者虽然戴着面具披着袍子，但看那背影，我怎么感觉有些眼熟阿。”
<br/>　　一旁的林飞和黄源也点了点头，“对，是有些眼熟。好像....好像是易霄兄弟。”
<br/>　　“切。”杜新嗤笑道，“少来了，易霄不过是后天一重的修为。但面前这强者，竟能力敌天星雷鳄，怕是有着先天九重的修为吧。”
<br/>　　杜新继续嗤笑道，“你们别告诉我，易霄那小子才消失了一个时辰不到，修为就跳了差不多两个大境界。”
<br/>　　后天一重到先天九重，确实是差不多两个大境界，这绝对是不可能的事。
<br/>　　但，若他们知道萧逸本身就是后天八重，仅仅是跳了一个大境界，就会觉得并非不可能了。
<br/>　　杜新如此一说，铁达大叔等人也摇了摇头，只觉自己看错了。
<br/>　　“希望这个强者的出现，能救我们的性命，易霄兄弟也能化险为夷吧。”铁达大叔等人希冀着。
<br/>　　杜新鄙夷道，“少来了，这洞府内如此危险，连其他更强的猎妖师都死了，加之雷鸣荒鳄众多，易霄不死才有鬼。”
<br/>　　“唉。”众人叹了口气，面露难过。
<br/>　　“你们认识这位强者？”旁边一个猎妖师见铁达大叔等人议论，不禁问道。
<br/>　　铁达大叔等人摇摇头，似是真地觉得易霄已死，道“这等强者，我们这些小喽啰怎么可能认识呢。”
<br/>　　这猎妖师见状，没有多问。
<br/>　　只有浅沫，一直在盯着萧逸，没有说话，但她很清楚，她没有认错。
<br/>　　那个背影，她绝不会看错。
