　　众人等待了约莫一刻钟左右的时间，祝英和王副尉回来了。
<br/>　　祝英的脸色已然铁青，阴沉的几乎要跌出水来。
<br/>　　众人见此，心中咯噔一声，看来龙脉真的不见了，不然祝英的脸色不会这么难看。
<br/>　　“血瞳族来袭击之时，你们有发现什么可疑之处吗？”祝英来到了众人的前方，冷着脸问道。
<br/>　　“没有！”众人纷纷摇头。
<br/>　　“那在你们大战之时，有没有发现有人暗自潜进矿脉中？”祝英再次问道。
<br/>　　“没有！”众人再次摇了摇头，之前大战如此危险，他们自己的安危都顾不及了，谁能有心顾及矿脉啊！
<br/>　　祝英闻言，沉默了下来，面色不停的变换，越发的难看了起来。
<br/>　　“祝尉将，想必在我们大战之时，血瞳族的人偷偷摸进了矿脉中，将矿脉全部收走了，并且故意惊走了龙脉！”任云都沉声说道。
<br/>　　“对啊！肯定是这样！”
<br/>　　“这帮血瞳族之人，看来是有备而来！”
<br/>　　“这帮血瞳族的混蛋，将我们引出矿脉，居然暗自偷走了整个矿脉！”
<br/>　　众人纷纷开口了，个个面现愤怒之色，因为这是明摆着的事情，定然是血瞳族所为。
<br/>　　苏莫闻言心中暗笑，他都不用开口，其他人就将罪责怪到血瞳族的身上了。
<br/>　　祝英闻听众人之言，面上沉默不语，这不用想也知道，肯定是血瞳族所为。
<br/>　　但是，此事却是不能这样定夺。
<br/>　　他之所以不来增援，就是想要借血瞳族的手，将任云都等人全部除掉。
<br/>　　但现在任云都等人，只是陨落了十数人而已，完全没有达到他的目的。
<br/>　　沉吟了一番，祝英目光扫视任云都、苏莫等一众神武学府的弟子，而后一声冷喝：“将他们都包围起来！”
<br/>　　唰唰唰！！
<br/>　　祝英话音一落，其带来的两千名士兵，立刻身形齐动，将苏莫等神武学府的人包围了起来。
<br/>　　虽然这些士兵不清楚祝英为何如此，但作为铁血军队，他们完全服从命令。
<br/>　　祝英带来的这些士兵，可比镇守在矿脉的士兵强大的太多了，两千余人当中，武帝境修为的人足有四分之一。
<br/>　　不仅如此，其中武帝境七重以上的高阶武帝，都有不下十几人。
<br/>　　“祝英，你想干什么？”任云都见此，顿时怒声问道，其他人也均是脸色狂变。
<br/>　　“想干什么？”
<br/>　　祝英闻言轻笑一声，淡漠的道：“矿脉消失，龙脉也不见了，任何人都脱不了干系，你们也要接受调查！”
<br/>　　“什么？”众人闻言一惊，祝英这不是怪罪他们守护矿脉不利，而是怀疑他们偷走了矿脉！
<br/>　　这区别可就大了，若只是怪罪他们守护不利，最多稍稍惩罚一番，但若是将矿脉的丢失，全部怪在他们的头上，那他们会被直接处死。
<br/>　　“祝英，你放屁！”
<br/>　　任云都彻底怒了，大声怒喝道：“矿脉不见了、龙脉跑了，与我们何干？我们众神武学府的弟子，为了守护矿脉，已经战死了十七人，你还想冤枉我们吗？”
<br/>　　“是啊！矿脉和龙脉不见了，是血瞳族的事，你应该去找血瞳族！”
<br/>　　“你不能冤枉我们！”
<br/>　　“真是岂有此理！”
<br/>　　其它神武学府的弟子，亦是纷纷开口，个个一脸怒气。
<br/>　　他们为了守护矿脉，战斗了这么久，陨落了十七人，更有很多人受了或轻或重的伤。
<br/>　　没有奖励也就罢了，毕竟没有保护好矿脉，现在祝英居然怀疑他们，他们岂能不怒？
<br/>　　“冤枉你们？”
<br/>　　祝英眸中冷光一闪而逝，随即环视众人，微笑摇头道：“大家放心，并不是要冤枉你们，只是带你们回要塞中调查一番而已，只要调查清楚了，一切都好说！”
<br/>　　言罢，他见众人还是一脸愤怒之色，继续道：“毕竟，矿脉和龙脉的消失，此事非同小可，调查清楚之后，祝隆将军也好向皇朝交代不是！”
<br/>　　祝英面色郑重，但心中却是冷笑，只要将众人带到了黑山要塞，到时候就能强行扣上罪名，直接就能处死。
<br/>　　众人闻言面面相觑，个个面现沉吟之色，若只是接受调查，倒也无可厚非。
<br/>　　“尉将大人，我们守护矿脉尽心尽力，带我们回去接受调查，你这是对我们的侮辱！”
<br/>　　冷冷的声音响起，是苏莫开口了，他不得不开口，若是真的会去调查，那他就完了。
<br/>　　祝英闻言，冷漠的目光顿时落在了苏莫的身上，他心中大怒。
<br/>　　刚才他的言语，已经让众人松懈了，不准备抵抗了。
<br/>　　但现在，苏莫的一句话，立刻就能带起节奏。
<br/>　　果然，苏莫话音一落，又有几名神武学府的弟子开口了。
<br/>　　“是啊！这简直是对我们的侮辱！”谢彬怒声道。
<br/>　　“不错！凭什么要调查我们？”
<br/>　　“我们守护不利，你责怪我们也无可厚非，但接受调查，简直是欺人太甚！”
<br/>　　几名神武学府弟子的话语，立刻让得所有神武学府的弟子，都坚定了信念。
<br/>　　是啊！
<br/>　　他们不顾生死，守护矿脉，还要回去接受调查，简直是对他们的侮辱！
<br/>　　“调查一番而已，各位何必说的如此严重？”
<br/>　　祝英心中虽然怒急，但面上却是丝毫不变，耐着性子道：“不仅是你们要接受调查，这里守卫的军士都要接受调查。”
<br/>　　“尉将大人，据我所知，你们祝家一直是三皇子一脉的人，和八皇子殿下关系极差，若是我们被带回了黑山要塞，难免不被冤枉！”苏莫继续说道。
<br/>　　众神武学府的弟子闻言一怔，随即个个眸光闪烁，苏莫的话不无道理，若是祝英和祝隆想冤枉他们的话，那就糟了。
<br/>　　到时候他们身在黑山要塞，只有死路一条。
<br/>　　“混账！”
<br/>　　祝英闻听此言，口中立刻暴喝一声，凌厉的目光如同一把尖刀，狠狠的刺在苏莫的身上：“祝隆将军做事一向公平公正，难道堂堂大将军，还会冤枉你们不成？”
<br/>　　“那不一定！”苏莫面色淡淡，丝毫无惧祝英的目光。
<br/>　　“你……！”祝英闻言大怒，眸中冰冷的杀机爆射，他恨不得一掌镇杀苏莫。
<br/>　　“祝英，我们是不会回去接受调查的！”任云都开口了，眸中充满了坚定，他是众人的领队，他的话语便代表了众人的立场。
<br/>　　“是吗？”祝英闻言眼眸微眯了下来，心中杀机大起。
