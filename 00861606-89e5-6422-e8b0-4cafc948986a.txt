　　不知从何时开始，大唐占着装备之利，在战场上横行无忌，打仗变的越来越容易，敌人好像变的越来越不禁打，以往需要连场大战的日子已经一去不复返。
<br/>　　在先进的装备之下，一顿火炮砸下去敌人的士气就先弱了一半，再一顿排枪，敌人的士气又弱了一半，而等到大军杀过去之后，敌人基本上也就没有多少战斗力了。
<br/>　　尉迟宝琪本以为这次可以好好的过一次海战的瘾头，结果一顿舰炮砸过去，大食水军立马就怂了，毫无反抗的加入了投降大军。
<br/>　　当然，这期间大食人也不是没有反抗过，只不过他们的反击实在太弱，弱到南海水师将所有大食人全部俘虏，清点伤亡的时候才发现，已方竟然只有一个炮手因为太过兴奋，抡大锤的时候砸到了自己的脚。
<br/>　　仗打成这样，再打下去也没啥意思了，这就像一个成年壮汉在和一个三岁顽童决斗一样，完全没有任何意义。
<br/>　　李承乾在战斗结束的当天夜里便收到了消息，对这样的结果他一点都不意外，大唐的钢铁战舰与大食人的木质战船就船只本身来说基本上就差着两到三代，至于武器更是天与地之间的差别，如果这样还能打输，那估计水师也就没有存在的必要了。
<br/>　　不过这话又说回来了，现如今四海升平，每日里李承乾除了处理公务就是处理公务，时间长了也觉得有些无聊，不过他的几个老婆倒是忙的不可开交，苏玫、孔雯、程小四、小武这四个娘们儿相互较劲，谁都不想比其他三人逊色。
<br/>　　首先是苏玫，这娘们儿负责的是还是老项目——医疗！
<br/>　　正巧李慕云前段时间想到要给那些老家伙们每人搞一个医疗小组，每天负责他们的身体健康情况，苏玫索性就主动接手了这项工作，成为首任‘老干部科科长’。
<br/>　　人员选拔，挑选精干力量为那些老干部服务；成立老年活动中心，反正老头子已经去了大明宫，太极宫那里也空出来了，索性把太极宫的御花园直接改了，改成老年活动中心，让一群精力无处发泄的老家伙都去那里撒泼。
<br/>　　其次是孔雯，这娘们儿负责的还是教育，在孔颖达那个老头儿的支持下，带着一群老家伙制定新的教材，从蒙学到太学，从文科到理科，分类归纳，将老头子们支使的团团转。
<br/>　　程小四这丫头比较简单，逮住她老子往死里坑，硬是要在右武卫搞什么大练兵，练精兵、强兵、特种兵……，好好一个右武卫被折腾的乌烟瘴气，数万大头兵累的几乎要拽着狗尾巴上床。
<br/>　　至于小武，这丫头只对钱感兴趣，京畿地区的地方官员被她指使的团团转，今天做调研，明天搞开发，一天到晚两头看不见太阳。
<br/>　　只有林晓晓和杨雨馨两个比较安于现状，没有接任何差事，就那么乖乖的待在宫里。
<br/>　　虽然看着那四个败家娘们儿有些羡慕，但她们两个也是自家人知自家事，知道自己没有强大的靠山，根本不可能有机会来展示自己的能力。
<br/>　　最后要说的就是长乐了，这丫头自从老头子走了之后，便成了后宫一霸，每天带着一群‘狗腿子’横行霸道，今天抢个宫女，明天揍个内侍。
<br/>　　李承乾实在看不下去她孩子王一样带着自己家的几个娃闹腾，索性将长乐叫到了自己的书房。
<br/>　　偏生长乐那丫头这段时间也是玩嗨了，被李承乾叫到书房也不老实，古里古怪的对他做了个鬼脸问道：“哥，找我有事儿？”
<br/>　　李承乾有些无奈的看了一眼长乐，乌黑的头发随意披在身后，一身纯黑色洛丽塔公主裙，脚上穿着精制的高根凉鞋，好好的大唐长公主竟然硬生生把自己打扮的……。
<br/>　　好吧，李承乾也说不出什么了，总之看着长乐深深的叹了口气。
<br/>　　自己这个哥哥当的可真是太不容易了，长乐这个样子真应该给她照下来，等老头子和老太太回来之后，让他们看看，看看到时候这丫头片子的腿能不能被打折喽。
<br/>　　长乐被李承乾盯着看了半天，脸上先是泛起一层红润，接着有些娇嗔的再次问道：“哥，你到底找我有什么事啊？”
<br/>　　李承乾摇摇头，把那些不切实际的想法从脑子里摇出去，然后淡淡说道：“没啥，就是我最想打算办一场华夏杯马球大赛，不知道你有没有兴趣帮我张罗一下。”
<br/>　　“打马球？”长乐有些不确定的问道。
<br/>　　“对，打马球，不过不是咱们自己打，是把一些番邦异族找来，每个国家为一队，最后的优胜者可以获得华夏奖杯！”李承乾稍微解释了一下什么是华夏杯马球赛。
<br/>　　“还要算上异族？不是只有我们自己么？”长乐原本是没什么兴趣的，但是听到以国家为队伍之后，顿时来了精神。
<br/>　　“当然不只有我们自己。”李承乾笑了笑说道：“大食人，拜占庭人，法兰克王朝，萨珊波斯人、斯拉夫人……这些全都可以找来。”
<br/>　　“这么多人？原来除了除了大唐世界上还有这么多国家？可是他们真的会来么？”长乐一根一根弯着手指，数了半天才惊讶的说道。
<br/>　　“不，有些是国家，有些不是国家，只是部落。不过管他呢，我们只是向外界发出邀请，愿意来的我们欢迎，不愿意来的也不强求嘛。”李承乾摊了摊手说道。
<br/>　　“不对吧，哥，你是不是还有什么别的想法啊？要不然没有理由搞这种事情吧？”长乐现在已经不是以前的那个小女孩了，尽管李承乾是一脸无所谓的表情，但还是被她看出来一些端倪。
<br/>　　不过李承乾却并没有多说什么，只是笑了笑：“怎么可能呢，我不过就是看朝臣这段时间都不怎么积极，所以打算给他们找点乐子，刺激他们一下，不过这种事情应该限于民间，朝庭不好过多的参与，所以我才会找你这个长公主来张罗这件事情。”
<br/>　　“我会信你才怪呢！”长乐哼了一声，装模作样的想了想，最后点点头说道：“不过本宫主正好这段时间没什么事情，就勉为其难的帮你一下吧！”
