
<br/>　　<br/>　　    王惊蛰晕了倒在后座上，茅小草停下车打开后车门，伸出两根手指摸到他的脉搏上，脉络有些紊乱呼吸还算均匀，只是脸色白的有些吓人，又看见王惊蛰的长袍下有血迹滴落下来，茅小草皱了皱眉然后掀开了长袍。<br/>　　<br/>　　    映入眼帘的除了王惊蛰浓密的腿毛外，还有右边小腿上一处已经有些溃烂的脓疮，创口的面积大概将近手掌大小，流出来的脓是黄褐色的，还伴随着一股难闻的味道。<br/>　　<br/>　　    茅小草顿时呆住了，这处脓疮看起来非常吓人，里面的血肉都烂掉了，而且明显没有任何复原的征兆。<br/>　　<br/>　　    “死不了就行啊······”茅小草无奈的摇了摇头，关上车门再次启程。<br/>　　<br/>　　    等王惊蛰醒来的时候，车停在了路边，天边露出了鱼肚白，小草姑娘趴在方向盘上睡着了，长长的睫毛搭在眼皮下，看起来睡得特别舒坦。<br/>　　<br/>　　    王惊蛰伸手拉了下长袍，看着腿上那处明显较之先前又恶化了的脓疮，昨日动手又扩散了脓疮的面积，照这么下去的话他这条腿挺不了多久，就得全都烂掉了。<br/>　　<br/>　　    惆怅的叹了口气，王惊蛰推开车门下了车，他的动静似乎吵到了睡着了的茅小草，她揉着眼睛看见王惊蛰醒了，就问道：“你没什么事吧？”<br/>　　<br/>　　    “死不了就行啊”王惊蛰淡淡的回了一句，然后靠在车门上点了根烟，眺望远处发现地平线的那头似乎是个城市。<br/>　　<br/>　　    “我们离开长安多久了？”<br/>　　<br/>　　    “三四百公里左右吧，我开了差不多一晚上，凌晨的时候睡了一会”茅小草抻了抻懒腰，看见他在抽烟就不满的说道：“首先我就不考虑你的什么绅士风度了，难道你就不考虑下自己的伤？”<br/>　　<br/>　　    王惊蛰弹了弹烟灰，说道：“死不死的也不差这一根了，没事”<br/>　　<br/>　　    尬聊了，一阵寂静。<br/>　　<br/>　　    一根烟抽完，王惊蛰看了眼低着脑袋踢着脚尖没有说话的茅小草，寻思了半天后，说道：“前面好像有座小城，你开车给我送过去给我扔在那就行了”<br/>　　<br/>　　    “什么意思？”茅小草头也不抬的问道。<br/>　　<br/>　　    “我得养伤，昨天被偷袭的那一拳头可能让我有点内伤了，不知道是什么程度，我需要静养一段时间不能在奔波了，这里离长安又不太远，我估计可能会被人再追过来，你跟着我有点危险，昨天那种状况可能还会发生的”王惊蛰条理清晰的说道。<br/>　　<br/>　　    “卸磨杀驴么？”茅小草斜了着眼睛说道。<br/>　　<br/>　　    “不是，我是为你考虑啊”<br/>　　<br/>　　    茅小草冷笑着说道：“说好的仗剑走江湖呢？”<br/>　　<br/>　　    “腿都残了，走不了了”<br/>　　<br/>　　    茅小草揉着脑袋，有些头疼的说道：“你真的是给我出了个难题啊，我现在要是把你给扔下了，可能会于心不忍，虽然咱俩也不是很熟·····但我这个人心软，连小猫小狗都见不得受欺负，更何况是个有伤的人了”<br/>　　<br/>　　    王惊蛰无语的说道：“你这个比喻，不亚于又给我捅了一刀”<br/>　　<br/>　　    “反正你也死不了，上车吧”<br/>　　<br/>　　    一个多小时后，清晨时分两人抵达了一个叫良水的小城，城市不大，典型的那种十八线小县城，环境也不错，依山傍水。<br/>　　<br/>　　    两人随便吃了点东西后，王惊蛰跟她说既然要修养一段时间，就不能住在旅馆酒店这种地方了，最好能租一间民宿，寂静一点，不太引人瞩目的那种。<br/>　　<br/>　　    随后，茅小草找了家中介谈了下要求，中介很快就找到了一处合适的地方，在良水河旁有一片老城区，那里人不是很多，几个月之后就要拆迁了比较适合短租。<br/>　　<br/>　　    谈完后，茅小草看着王惊蛰，看了半天之后都有点不耐烦了，他就尴尬的问道：“你这么看着我干什么啊？”<br/>　　<br/>　　    “租房不要钱么？”小草皱眉说道。<br/>　　<br/>　　    王惊蛰顿时羞涩的从口袋里翻出一把毛票，摊开后说道：“这样吧，咱俩凑凑，我这还有十几块钱，剩下的你来？”<br/>　　<br/>　　    “妈蛋，合着我跟你东奔西跑的，最后还得倒贴？”<br/>　　<br/>　　    “好人一生平安啊”王惊蛰认真的说道。<br/>　　<br/>　　    茅小草：“······”<br/>　　<br/>　　    中介随后领着茅小草和王惊蛰去了租住的地方，尽管都是老房子但环境还可以，就靠着河边比较幽静，独门独院，有三间房。<br/>　　<br/>　　    “我出去买点东西，你随意吧，医院离这好像也不太远，你走过去就行了”房子里面有点空，短期生活的话，很多东西都得要置办。<br/>　　<br/>　　    王惊蛰说道：“我这个病看医生没用，你去买东西的时候，顺便帮我买点滋补的药材就行了，剩下的主要就是静养”<br/>　　<br/>　　    “很好，你很阔以的”茅小草磨着牙说道。<br/>　　<br/>　　    王惊蛰又羞涩了，挠着脑袋说道：“反正你都倒贴了，也不差这一点了，是不？”<br/>　　<br/>　　    “好好说话行么？否则，我可能一去就不回来了······”<br/>　　<br/>　　    王惊蛰一本正经的说道：“你走了，我也不怪你”<br/>　　<br/>　　    “保重吧，朋友，再见来不及握手！”<br/>　　<br/>　　    茅小草走了以后，王惊蛰抹了把头上的冷汗，进到屋子里后连忙就躺在了一张硬板床上，不停的喘着粗气。<br/>　　<br/>　　    脱掉身上的长袍，光着膀子，王惊蛰反手摩挲着后背右侧肩膀，昨天对方忽然砸过来的拳头，拳劲全都蔓延至了他的体内，王惊蛰估计五脏六腑肯定都已经受创了，现在就连喘气都有点费劲。<br/>　　<br/>　　    “至少几个月之内都不能大动干戈了，愁人啊”王惊蛰无奈的感叹了一声：“不过，幸亏有个女人，真好”<br/>　　<br/>　　    中午的时候，茅小草带着大大小小几个袋子回来了，里面全是日常用品，宛若居家过日子一样，东西一应俱全。<br/>　　<br/>　　    王惊蛰瞅着她一样一样的往出拿着东西，忽然开口说道：“我们这算是要同居了么·······”
