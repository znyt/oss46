　　雷鸣荒泽，位于破玄城外百里处。
<br/>　　里头处处是沼泽、荒野和瘴气，极为危险。
<br/>　　一般人几乎不愿意前往。
<br/>　　萧逸一行六人，刚天亮便开始赶路，来到雷鸣荒泽的边缘地带。
<br/>　　从外头看，已经可以看到里面一层层雾气，显得极为诡异。
<br/>　　萧逸拿出一袋丹药，道，“里面的雾气怕是不简单，可能还会有瘴气存在。为免意外，还是先服用解毒丹吧。”
<br/>　　铁达大叔则道，“这雷鸣荒泽，我和林飞他们曾经进入过猎杀妖兽。外围的雾气倒没什么，顶多影响视线。”
<br/>　　“前行数十里之后，才会出现瘴气。我们武者身体强悍，不惧这些许瘴气，顶多就是吸多了会有点头晕。”
<br/>　　“丹药珍贵，还是留着需要的时候再用吧。”
<br/>　　萧逸笑笑，道，“这解毒丹不贵，我炼制了许多，够我们用的。”
<br/>　　说罢，萧逸给众人分发丹药，解毒丹、巨力丹、隐气丹等等。
<br/>　　这些丹药都是昨晚提前炼制好的。
<br/>　　众人接过丹药，黄源得意道，“我们还真是幸运，有易霄这个炼药师加入，有大把的丹药吃。”
<br/>　　“可不是嘛。”林飞也道，“丹药太贵了，我们做一次任务都买不上几颗，以前出任务可舍不得吃。”
<br/>　　萧逸笑笑，事实上丹药的材料并不贵，只不过炼药师身份尊贵，炼制出来的丹药也就凭空贵了许多。
<br/>　　众人吞服下丹药，铁达大叔道，“这次我们的目标是狂血玄君的洞府。”
<br/>　　“路途上肯定会遇到不少妖兽，斩杀妖兽所得，我们平分。”
<br/>　　“而到了洞府，是否有收获，就全看自己的机遇了，如何？”
<br/>　　众人点点头，而后彻底进入雷鸣荒泽。
<br/>　　......
<br/>　　雷鸣荒泽内，雾气环绕，众人的视线限制在数十米之内，再远就看不到了。
<br/>　　当然，林飞除外。
<br/>　　嗖嗖，只见林飞凝聚出冲云箭武魂，快速打出几箭。
<br/>　　远处，一阵穿刺的声音响过。
<br/>　　“前面浓雾里有三头荒泽蟒，准备袭击我们呢。”林飞解释道。
<br/>　　众人走前一看，果然，三头身躯庞大的黑蟒被三根羽箭钉在了地面上，但还没死，仍旧凶猛地扭动着身躯。
<br/>　　荒泽蟒，二级妖兽，实力在后天二重。
<br/>　　嗖嗖，铁达大叔数刀劈出，了结了它们的性命，而后大手一抓，将尸体扔进乾坤袋。
<br/>　　“尸体全身都是宝呢。”铁达大叔对萧逸和浅沫道，“除了内丹外，它们的皮毛、爪子、妖血，都是值钱的材料。”
<br/>　　“若你们日后自己出任务，可千万要记住。”
<br/>　　铁达大叔知道两人是新手，教导道。
<br/>　　众人再次前行，不时遇到妖兽，但都被林飞的冲云箭提前钉住，减少了许多麻烦。
<br/>　　这时，浅沫忽然问道，“铁达大叔，为什么这里叫雷鸣荒泽阿，也不见有打雷阿？”
<br/>　　萧逸无奈地看了她一眼。
<br/>　　从走入雷鸣荒泽开始，她就没紧张过，时不时问些奇怪的问题，看看周遭有趣的植物，倒像是来游山玩水的。
<br/>　　不过，铁达大叔是个好说话的人，笑道，“说起这个，除了浅沫和易霄外，恐怕林飞你们这些年轻人都不知道吧？”
<br/>　　说到年轻，萧逸之前看林飞三人的样子，还以为他们已经三十出头了。
<br/>　　昨晚才知道，这三人原来不过是二十五、六岁，只不过他们是猎妖师，需要到处奔波出任务，饱经风霜，显得样子比较成熟而已。
<br/>　　铁达大叔没有卖关子，直接道，“听老一辈的猎妖师说，以前这里曾经是禁地呢。雷电密布，肆虐整个荒泽，寸草不生，万物不长，宛若雷狱。”
<br/>　　“曾经有一个洞玄境武者冒险进入这里，一记雷电就把他劈死了。后来，再无人敢踏足此地。”
<br/>　　“只不过，不知多少年前，这里的雷电忽然消失了，这才慢慢有猎妖师来这边猎杀妖兽。”
<br/>　　铁达大叔说罢，众人面露震惊，没想到以前这里是如此恐怖的险地。
<br/>　　正当众人陷入震惊时，数道魅影从浓雾中飞掠而出。
<br/>　　嗖嗖，铁达大叔眼疾手快，一刀将数道魅影劈开两半。
<br/>　　“是浓雾风妖。”林飞等人吓了一跳，“幸好队长你反应快。”
<br/>　　浓雾风妖，二级妖兽，实力不强，但速度奇快，擅长偷袭。
<br/>　　铁达大叔顿时严肃道，“这里毕竟是雷鸣荒泽，万万不可掉以轻心。易霄和浅沫是新手也就罢了，你们三人瞎愣神个什么劲。”
<br/>　　“是。”三人应答一声。
<br/>　　众人再次前进，渐渐出现了瘴气，使得所有人的视线更加差，勉强能看到二十米内的东西。
<br/>　　嗖嗖，林飞射杀掉几只试图偷袭的妖兽。
<br/>　　“这几只归我捡了。”杜新着急地说着，快步朝前方走去。
<br/>　　事实上，谁捡都一样，反正之后是平分的，不过他怕吃亏罢了。
<br/>　　然而，正当他走到妖兽尸体前，准备捡起来时....
<br/>　　忽然，一头庞然大物凭空而现。
<br/>　　“杜新小心。”林飞一箭射出。
<br/>　　不过，意料中的穿透并没有出现，仅仅是将那妖兽击伤。
<br/>　　嗖的一声，妖兽再次凭空消失。
<br/>　　“那是什么东西，怎么出现的？”王源惊呼一声。
<br/>　　铁达大叔脸色凝重道，“如果我没有看错的话，那是妖兽荒鳄。”
<br/>　　妖兽荒鳄，二级妖兽，相当于后天五重武者的实力。
<br/>　　但妖兽的肉体是远强于人类武者的。
<br/>　　它们生活在山林中，本身的环境远差于人类。
<br/>　　恶劣的环境，本身凶残的性格，不时能吃到山林中的天材地宝，加之其它弱小妖兽的血肉和内丹。
<br/>　　这造就了它们的肉体极为强悍。
<br/>　　若是同境界武者与妖兽战斗，几乎没有任何胜算。
<br/>　　林飞的冲云箭，之前都能穿透其它妖兽。可偏偏射中荒鳄时，却仅仅能射伤它，就是这个原因。
<br/>　　这还是因为林飞乃后天六重修为，比荒鳄高了一级，否则根本伤不了荒鳄。
<br/>　　“阿。”这时，杜新忽然惊呼一声。
<br/>　　众人看去，只见杜新忽然下沉，双脚陷于沼泽之中。
<br/>　　“怎么回事，哪来的泥潭，这里明明是平地。”林飞吃了一惊。
<br/>　　“别废话了，快拉他上来。”铁达大叔吩咐一声。
<br/>　　林飞和王源刚要动，萧逸立马喝道，“慢。”
<br/>　　“人陷于沼泽，越拉越陷进去。”萧逸喝道，“杜新，调动体内真气，附于双脚，靠真气抵消你的重力，自己浮上来。”
<br/>　　“好。”杜新应答一声，快速照办。
<br/>　　但，下一秒杜新惊呼道，“有...有东西，我脚下好像有什么东西...”
<br/>　　嗖..嗖..杜新身旁，忽然两只荒鳄凭空跃出，张开血盆大口，欲要吞食杜新。
<br/>　　“哼。”铁达大叔冷哼一声，一身先天三重的气息澎湃而出。
<br/>　　两道刀光闪过，两只荒鳄应声死亡。
<br/>　　“快上来。”铁达大叔脸色难看道，“该死，我们武者反应灵敏，根本不可能会陷入泥沼，杜新你怎么回事？”
<br/>　　“我...我不知道阿。”杜新也是脸色难看。
<br/>　　这时，萧逸蹲下，伸手捻了些地面的泥土，顿时脸色凝重。
<br/>　　“这泥土有问题。”萧逸道。
<br/>　　“易霄，你发现什么了？”铁达大叔连忙问道。
<br/>　　“哼。”一旁的杜新见自己刚才出丑，嘲讽道，“他一个后天一重，能发现什么。”
<br/>　　萧逸没有管他，沉声道，“泥土很松软，还带着些许粘稠的水分。而且，这些泥土好像很散乱，像是...”
<br/>　　萧逸皱眉道，“像是被某些东西堆放过来的，而并非自己本身存在的。”
<br/>　　萧逸的话语刚落。
<br/>　　铁达大叔林飞瞬间脸色大变。
<br/>　　萧逸也立刻反应过来，“不好，中计了，好狡猾的妖兽。”
<br/>　　“怎...怎么回事阿？”浅沫见他们三人脸色大变，不禁问道。
<br/>　　三人没有回答。
<br/>　　铁达大叔闭上眼睛，细细地感知了一番，而后道，“果然，我们被包围了。”
<br/>　　“易霄，到底怎么回事嘛。”浅沫摇着萧逸的手臂追问道。
<br/>　　在她看来，以她的实力，能欺负的人也就‘最弱’的易霄。
<br/>　　萧逸沉声道，“那些荒鳄将大量泥土覆盖在沼泽表面，这里迷雾重重，能见度极低，便令我们误以为这里是平地，自己落入它们的圈套了。”
<br/>　　“也就是说，我们现在站着的这地，只是一层薄薄的泥土，下面就是生活着成群荒鳄的沼泽？”浅沫惊声道。
<br/>　　难怪之前的两条荒鳄凭空出现，根本就是在沼泽下破土而出。
<br/>　　“别废话了。”萧逸沉声道，“立刻服下隐气丹，我们立马离开。”
<br/>　　然而，就在这时，无数荒鳄纷纷冒头，凶残的血目冷冷地盯着萧逸等人。
<br/>　　荒鳄的出现，驱散了一些迷雾。
<br/>　　众人这才看到，除了自己周围十数米的地方是平地外，其余地方根本就是一片庞大的沼泽。
<br/>　　而自己等人，就站在这庞大沼泽的正中央，如盘中餐一般。
<br/>(本章完)
