　　坐在梧桐树下，陆雪琪心中仍有许多问号，道：“你方才说的张小凡，便是大竹峰的张师弟吧？”
<br/>　　“嗯，确实是他。”
<br/>　　“那刚才所提到的，普智又是何人？”
<br/>　　对于陆雪琪的问题，慕清霄是有问必答，片刻后，她就清楚事情的来龙去脉，眼中闪过一丝笑意，语气温柔，道：“没想到，此事倒是帮了你一个小忙。”
<br/>　　就在这时，法相已经折回，十指合一，道：“慕师兄，我奉师命，来请您前去相见，恩师他老人家在山顶小天音寺禅室之中，师兄尽管自己前去就好。”
<br/>　　闻言，慕清霄心中一动，法相这一番话可是大有深意，语气态度与先前完全不同，似乎已经将他当作了天音寺自己人看待，看来普智对普泓的影响真不小。
<br/>　　不一会，法相告辞转身离去，慕清霄则牵着陆雪琪向山顶走去。
<br/>　　朝阳之下，小天音寺朴实无华的座落在前方，低矮的墙壁，小型的院落，与四周的金碧辉煌格格不入，回首眺望，半山腰上的寺庙中，隐约传来人声，香火繁盛，一派热闹景象。
<br/>　　慕清霄微不可查的摇头，刚踏入小天音寺中，外面的声音就被隔绝，四周看了看，果然有阵法的痕迹，不大的院落中，只剩下两人的脚步声。
<br/>　　走到禅室门口，慕清霄轻轻敲响木门，很快，室内就传来普泓主持平和的声音：“是慕施主吧，快请进来吧。”
<br/>　　慕清霄与陆雪琪相视一眼，轻轻推门走了进去，偌大的屋中，只有普泓主持独自一人盘坐在禅床上，面露微笑望着走进来的两人。
<br/>　　“坐吧。”
<br/>　　话语刚落，便隔空取来两个蒲团，待慕清霄与陆雪琪坐下，方才开口，道：“听法相说，你找老衲有事相谈，还是关于普智师弟的？”
<br/>　　慕清霄道：“准确的来说，我是有求于普泓主持。”
<br/>　　“有求于我，从何说起？”
<br/>　　“老狐狸。”慕清霄暗骂一声，微笑道：“普泓主持的师弟，八年前将草庙村屠戮一空，是也不是？”
<br/>　　话语刚落，屋中静得可怕，气氛也变得有些压抑，陆雪琪情不自禁的握住了天琊神剑的剑柄。
<br/>　　片刻后，普泓叹息一声，道：“慕施主说的不错，不过此事，好像与慕施主并无关系吧？”
<br/>　　慕清霄微微摇头，道：“怎会无关，普泓主持请听我将此事一一道来......”
<br/>　　片刻后，普泓眼中满是不敢置信，失声道：“施主是说，张小凡身上的噬血珠，已经被施主所封印，再也无法重见天日？”
<br/>　　“不错，噬血珠确实已被我用特殊的方法所封印，再也不能为祸人间，当初可是差点要了我的性命！”
<br/>　　慕清霄特地将一件小事说的凶险万分，他自然不可能说，噬血珠已经化作了自己的修为，反正噬血珠已经消失在人世间，他说什么，那就是什么。
<br/>　　“这......这不可能，我师弟普智终其一生，都只能用无边佛法压制住噬血珠的煞气，凭你个小娃娃，怎么可能封印噬血珠！”
<br/>　　慕清霄依旧心平气和，道：“普智大师无法封印，却不代表我无法封印，你作为天音寺主持，不可能不知道，人外有人天外这句话吧？”
<br/>　　“可是......”
<br/>　　见普泓依旧不信，慕清霄继续道：“而且，我不仅封印了噬血珠，且帮你师弟完成了一大遗愿，让张小凡将《太极玄清道》与《大焚般若》两门至高心法结合在了一起。”
<br/>　　闻言，普泓再也坐不住，因为慕清霄所说的话都天衣无缝，每一句都与当初所发生的事情完全符合。
<br/>　　“明人不说暗话，前些日子，主持想必派法相观察过张小凡的情况吧，可有察觉到他身上带有一丝煞气？”
<br/>　　慕清霄端坐在前方，一切尽在掌握中，倒是普泓面色变化不定。
<br/>　　片刻后，普泓盘坐下来，叹息一身，脸色变得与平常无异，道：“慕施主的话，老衲信了，半个月前，确实也派法相前往空桑山，探查张小凡的情况，可却未曾探查出结果。”
<br/>　　“张小凡资质不凡，两门至高心法结合在一起，修行一日千里，特别是在《大焚般若》上，天赋远胜于《太极玄清道》.....”
<br/>　　“普智师弟落得如此下场，虽然乃是自作孽，罪不可恕，但究其根源，还是那大凶之物‘噬血珠’。”
<br/>　　语气顿了顿，普泓继续说道：“普智师弟犯下弥天大错，天音寺责无旁贷，老衲本想尽一切办法弥补张小凡，只是没想到，慕施主已经替天音寺化解仇恨，真是因果循环，冥冥中自有定数。”
<br/>　　感慨一声，普泓目光落在慕清霄身上，道：“难怪我看不透施主，不知那噬血珠被封印在何处？”
<br/>　　“噬血珠封印在一处无人知晓之地，魔教中人也无法触及，所以主持不必担忧。”
<br/>　　听到慕清霄将话堵死，普泓也不再多问，道：“若又朝一日，慕施主压制不住封印，还请将噬血珠带到我天音寺，由我亲自封印。”
<br/>　　“主持放心，这一日永远不会来临。”
<br/>　　“如此甚好，那施主来天音寺，所谓何求？”普泓松了一口气，接着问慕清霄此来目的。
<br/>　　“无字玉璧。”
<br/>　　闻言，普泓瞳孔一缩，沉吟片刻，道：“也罢，慕施主与我天音寺有宿愿在身，让你参悟无字玉璧也无不可，慕施主稍等片刻，老衲让法相为你领路。”
<br/>　　片刻法相来到院中，慕清霄与陆雪琪告辞，望着两人的背影，普泓叹息一声，自言自语道：“去青云门确认一下，慕施主所言是否属实，记住暗中查探确认便是，以免勾起那孩子早年的记忆。”
