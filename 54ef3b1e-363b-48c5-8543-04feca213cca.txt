　　这支由卡拉尼什科夫集团生产的原装SVD狙击步枪旁边，还有一张纸条，上面详细说明了弹匣中所装填的空包弹和实弹的顺序。
<br/>　　慕容飞把纸条上的内容记住后，把纸条撕碎，扔进了雨水排水管中，拿起SVD狙击步枪，装入弹匣，哗啦一声拉动枪栓，把子弹推入枪膛，然后抱着狙击步枪，坐在这栋烂尾楼中，安静地等待着。
<br/>　　周志斌带着两名重案组的警员赶到医院的时候，“司机”刚做完“手术”，还处在“昏迷”中，不能录笔录。
<br/>　　就在这个时候，两名国安局的同志赶到了医院，向周志斌出示了一下自己的工作证后，周志斌申请立刻变得严肃起来。
<br/>　　一个地级市重案组的组长，而对方两人是来自国家安全部，两者在分量上差得可就不是一星半点了。
<br/>　　“国家安全部第九局，舒磊。”
<br/>　　舒磊伸手跟周志斌握了握手，对着周志斌介绍他身边另外一个女子，说道：“她是我的同事，也是第九局的，韩梅梅。”
<br/>　　“你好。”周志斌又是跟韩梅梅握了握手，问道：“不知道两位来这里是为了……？”
<br/>　　尽管在心中已经有了一个初步的猜测，可周志斌还是想要得到一个确切的答案。
<br/>　　“为了iCu里的那个。”舒磊对周志斌解释道：“现在在iCu里待着的那个家伙代号‘司机’，是境外某军火集团在亚洲的代理人，跟多起恐暴案有直接关系，我们盯着他已经有段时间了，本来是想要吊出他背后的负责人，可是没想到他居然被人打成了重伤。第九局会参与到这次案件中，相关手续文件会稍后送到贵局。”
<br/>　　“既然如此，那我们就通力合作。”周志斌又是跟舒磊握了握手。
<br/>　　“通力合作。”
<br/>　　国家安全部第九局要参与，盯着“国家安全部”这块能把人闪瞎眼的招牌，周志斌根本没办法拒绝，别说是他没办法拒绝，就是市局局长都没办法拒绝。人家根本就不是跟你商量，而是命令！
<br/>　　这时“医生”从iCu中走了出来，舒磊立刻上去询问情况。
<br/>　　“病人手术很顺利，等到麻药过去后，应该就能醒过来。”
<br/>　　“医生”一本正经的说道：“等到他苏醒后，你们就能进去问情况，但是时间不要太长。”
<br/>　　舒磊谢过这个“医生”后，对周志斌说道：“周队长，如果你那边还有事情的话，这里就交给我们，我们得到口供后会在第一时间跟你们进行共享。”
<br/>　　周志斌想了想，也就点头答应了，带着两个重案组的警员离开了医院。
<br/>　　看着周志斌他们三人走进电梯离开，韩梅梅立刻对着旁边的舒磊比了个胜利的手势，说道：“我在第九局待了这么长时间，还是第一次这么一本正经的骗人。”
<br/>　　“刺激吗？”舒磊问道。
<br/>　　“那是相当的刺激。”韩梅梅兴奋地说道。
<br/>　　两个人推开iCu的门走了进去，舒磊抬脚就在病床床脚上踢了一下，对还在床上装昏迷的“司机”说道：“行了，别装了，人都走了。”
<br/>　　“妈的，终于走了。”
<br/>　　“司机”立刻从床上坐了起来，对舒磊说道：“刚才我是真怕他们直接进来，你也知道这些重案组的是有多贼，我真怕自己会露馅，幸亏你们来了。”
<br/>　　“我们一直都在外面盯着呢，看着他们进的医院。”韩梅梅对坐在床上的“司机”调侃道：“你不是‘老司机’了么，也会有担心的时候？”
<br/>　　“老司机也有翻车的时候。”
<br/>　　“司机”撇了撇嘴，说道：“而且这次行动可是暗影的大动作，要是在我这边漏了陷，我特么都没脸回去。”
<br/>　　“还真是大动作。”韩梅梅认可的点了点头，说道：“连我们第九局都被要求配合进来了。”
<br/>　　“因为‘暗月’，第二、第七、第八、第十局有多少兄弟姐妹出去了就没有再回来。”舒磊的眼眶微微有些发红，说道：“这次说什么也要从‘暗月’身上讨回点债来！”
<br/>　　“对了，东西都准备好了么？”“司机”对舒磊问道。
<br/>　　舒磊从口袋中拿出一个u盘，说道：“都在这里呢。”
<br/>　　舒磊和韩梅梅在医院待到快要天亮的时候，起身离开，把u盘里的“口供”拿去交给周志斌。
<br/>　　……
<br/>　　东海市，特警支队简报室。
<br/>　　副支队长王荣浩正在讲台前介绍情况，他把周志斌从舒磊手中拿来的u盘插在电脑上，打开里面的文件，说道：“昨天晚上，在‘蓝玫瑰’酒吧背后的小巷中发生一起枪击案，一人重伤。枪击案的嫌疑人叫慕容飞，前国家特种部队队员，根据国家安全部方面传过来的情报，慕容飞是跟这个代号‘司机’的军火贩子在进行军火交易时，发生争执进而发生冲突，慕容飞拔枪射击，‘司机’重伤。目前慕容飞被困在距离‘蓝玫瑰’酒吧三公里的一栋烂尾楼中，市局要求我们立刻赶往现场……”
<br/>　　“你他妈放屁！”
<br/>　　王荣浩的话还没说完，肖天行就满脸怒气冲进了简报室。
<br/>　　“肖天行，你说什么！？”王荣浩瞪着肖天行，大声质问道。
<br/>　　“我说你放屁！”肖天行指着投影布上慕容飞的照片，说道：“他，绝对不可能做出你说得那些事情！”
<br/>　　“不可能？”王荣浩嗤笑一声，说道：“有什么不可能？事实就是事实，现场有弹壳、弹头和人证，还有国家安全部提供的口供，可以说，现在是铁证如山！”
<br/>　　“这不可能，这不可能……”
<br/>　　肖天行失神的说道：“他是无常，他在国旗下宣过誓，他不可能跟军火贩子进行军火交易，更不会故意杀人，他是绝对不会背叛这个国家！”
<br/>　　王荣浩看着肖天行，脸上突然露出恍然大悟的表情，说道：“我差点都忘了，你跟他一样，也是前国家特种部队队员，你们好像还是战友是吧？肖天行，你不会就因为他是你的战友，所以才主观臆断他不会故意杀人吧？”
