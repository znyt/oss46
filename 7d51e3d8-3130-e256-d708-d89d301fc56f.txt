　　“暗影千仞？”萧逸的脸色变得无比凝重。
<br/>　　明显可以感觉到，钟无忧身上正凝聚着一股让人倍感压力的气息。
<br/>　　方圆千米的灵气，瞬间暴走。以他为中心，渐渐形成了一个澎湃的漩涡。
<br/>　　高空中，一片暗黑之幕凭空而现，如同一片数百米之大的‘黑色乌云’。
<br/>　　“好恐怖的压力。”萧逸发现，自己在这暗黑之幕下，仿佛被压得直不起身来，双脚也微微发颤。
<br/>　　这就是地阶武技的恐怖与强大，甚至还未释放出来，形成的压力已经如巍峨高山般沉重。
<br/>　　暗影楼，正是北山郡少数几个拥有地阶武技的大势力。
<br/>　　这暗影千仞，虽然只是地阶低级，比玄阶巅峰武技只高出一个等级，但这是大境界上的差距。
<br/>　　天地玄黄，四个阶级。每个阶级内，低级、中级、高级、巅峰，差距已经极大，彼此差了十倍不止。
<br/>　　更别说阶级之外了，地阶低级，与玄阶巅峰相比，简直是一个天上，一个地下。
<br/>　　这钟无忧拥有中品灵器，又有地阶武技，难怪能斩杀掉一个洞玄境武者。
<br/>　　“易霄。”
<br/>　　钟无忧的脸色变得杀意凛然。
<br/>　　“北山郡年轻一代，除了那几个家伙外，你是第一个逼得本公子全力战斗，连地阶武技都使用出来。”
<br/>　　“哪怕是死了，你也该自豪了。”
<br/>　　钟无忧相信，自己这一招之下，足以将易霄劈成齑粉。
<br/>　　萧逸收起了之前的淡淡神色，沉声道，“钟无忧，谁死，还不一定呢。”
<br/>　　话音刚落，萧逸再次凝聚出那片百丈紫炎光幕。
<br/>　　紫炎光幕挡于自己头顶上方，萧逸身上那股如大山般的压力顷刻间消散。
<br/>　　.....
<br/>　　另一边，数百武者震惊地看着两片巨大光幕。
<br/>　　耀眼的紫炎光幕，骇人的暗黑之幕，如同光与暗之间的对峙。单单散发的气息，就足以让他们感到心悸。
<br/>　　“地阶武技，竟恐怖如斯。”
<br/>　　“如无意外，此战无忧公子必胜。毕竟是暗影楼的少楼主，远非一个籍籍无名的猎妖师能比的。”
<br/>　　“虽然那紫炎也不错，但跟地阶武技相比，还是有差距的。”
<br/>　　“但无论如何，此战过后，若易霄不死，他必然声名大噪。”
<br/>　　“能逼得无忧公子全力应付，在无忧公子手下撑那么久，也算厉害了。”
<br/>　　武者们一边心惊着，一边则似乎已经看到战斗的胜负。
<br/>　　.....
<br/>　　萧逸与钟无忧这边。
<br/>　　暗黑之幕内，无数道暗黑之刃激射而出。
<br/>　　密密麻麻，如天降黑刃。
<br/>　　嘶嘶嘶....黑刃上不知存在什么力量。
<br/>　　才刚刚从暗黑之幕中射出，接触到空气，竟让空气发出一股被侵蚀的声音。
<br/>　　显然，每一把暗黑之刃都强大至极，千把齐出，就算是洞玄境武者都得退避三舍。
<br/>　　地阶武技本就强悍，又是以中品灵器千影剑发出，其威力可想而知。
<br/>　　当然，紫炎也非凡物。
<br/>　　千把黑刃射向萧逸，都率先被紫炎光幕挡下。
<br/>　　噗..噗..噗....密密麻麻的低沉声响起。黑刃落入紫炎光幕中，竟只如小石头落入水中，仅仅溅起了些许水花，便再无踪影。
<br/>　　紫炎焚尽万物的功效，此刻展露无遗。
<br/>　　任你暗黑之刃再强，通通都在通过紫炎光幕时便被焚毁殆尽，休想伤萧逸分毫。
<br/>　　但，紫炎毕竟也是能量之一。
<br/>　　它烧毁了黑刃，却也被黑刃消耗了不少力量。
<br/>　　当千把黑刃完全射出时，虽已被烧毁，却也让紫炎几近消耗一空。
<br/>　　原本百丈大小的紫炎光幕，只剩下不足数丈，仅仅能护与萧逸头顶的小一片位置。
<br/>　　这时，有数把黑刃由于没有紫炎的阻挡，径直落下，打在了萧逸身旁数米之外的地面上。
<br/>　　顿时，嘶的一声....
<br/>　　地面一大片的泥土瞬间被侵蚀殆尽，化成了虚无，最终形成了一个大坑。
<br/>　　“嘶。”萧逸不禁倒吸一口凉气，“地阶武技，果然不凡，这些黑刃，竟有如此强的侵蚀能力。”
<br/>　　他甚至可以想到，哪怕是一把黑刃，打在自己身上，就足以将自己侵蚀得渣都不剩。
<br/>　　别说自己媲美先天九重妖兽的肉体了，就算是一个半步洞玄武者，用尽全身真气去抵挡，也会瞬间被黑刃侵蚀为虚无。
<br/>　　想到这点可怕的后果，萧逸当即催动烈焰手套的力量，将紫炎光幕再次化为百丈大小，将所有暗黑之刃挡下。
<br/>　　中品灵器施展下的地阶武技，绝非洞玄境以下，乃至一般洞玄境武者能够抵挡的。
<br/>　　另一边，钟无忧明显注意到萧逸的情况，冷笑道，“易霄，虽然我不知你的紫炎是何物，但也不过如此。”
<br/>　　“虽然你挡下了一次暗影千仞，却也让你的紫炎消耗殆尽。”
<br/>　　“以我的修为和体内真气，足以再释放两次地阶武技。”
<br/>　　“你呢？如果我没有猜错的话，你的真实修为，根本没有半步洞玄，顶多是先天五重左右，靠着灵器的力量才施展出这片光幕罢了。”
<br/>　　他没有猜错，萧逸只有先天三重的修为。
<br/>　　“哼，就算我体内真气耗光。我也仍旧能借助千影剑的力量，继续释放武技。”
<br/>　　“恐怕，你的灵器力量耗尽之后，我的千影剑仍旧力量十足。此战，你必败。杀你，轻而易举。”
<br/>　　萧逸听着他的冷笑，心头也是一阵着急。他说得没错，自己的下品灵器，可比不过中品灵器内的力量。
<br/>　　最终很有可能是自己率先耗尽灵器力量。
<br/>　　一旦灵器力量耗光，无法释放紫炎，随便一道暗黑之刃便能将自己击杀。
<br/>　　但，他仍旧没有害怕。
<br/>　　他仍旧有着自己的把握。
<br/>　　“钟无忧，你今天说的大话够多的了。”萧逸冷笑道，“你已经不止一次说能必杀我，可结果，却是你无可奈何。”
<br/>　　“这次也一样，落败的会是你。”
<br/>　　话音刚落，紫炎光幕瞬间从高空中消失，而后凝聚到自己身边，成为一片紫炎火海。
<br/>　　远远看去，萧逸就像置身于火海中的神祗，强大至极。
<br/>　　他这样做，只是不想紫炎的力量做无谓消耗。有些本来就不会打中自己，而是打到附近的黑刃便不用去抵挡了。
<br/>　　以此来最大化利用灵器的力量和紫炎。
<br/>　　“哼。”钟无忧冷哼一声，不屑道，“不过是垂死挣扎罢了。”
<br/>　　说罢，又是一剑劈出，喝道，“暗影千仞。”
<br/>　　噗...噗....
<br/>　　情况与之前一样，千把暗黑之刃打完，紫炎火海被消耗大半。
<br/>　　两次之后，钟无忧体内真气已经耗尽。
<br/>　　萧逸仍旧在支撑着。
<br/>　　“哼，易霄，我要使用千影剑内的力量了，我看你还如何抵挡。”
<br/>　　钟无忧说着，千影剑高高举起，剑内力量凭空连接了高空中的暗黑之幕。
<br/>　　顿时，暗黑之幕上的诡异黑芒更甚几分。
<br/>　　发出的暗黑之刃也更为强大，每一把都比之前强了数倍都不止。
<br/>　　只不过，这次的暗黑之刃，似乎不大受控制，有一些竟然脱离了打向萧逸的轨迹，反而往四面八方打去。
<br/>　　.......
<br/>　　嗖...嗖...嗖....
<br/>　　千米范围内，凡是被黑刃达到的东西，尽皆被侵蚀得渣都不剩。
<br/>　　远处，观战的武者瞬间大惊。
<br/>　　“不好，原本无忧公子用体内真气释放地阶武技，以千影剑辅之，其威力已经达到洞玄境的层次了。”
<br/>　　“如今，无忧公子直接调动千影剑内的力量释放地阶武技，威力强了数倍都不止。这个层次的攻击，已经不是他能完美掌控的了。”
<br/>　　“千把暗黑之刃，起码有百把以上会不受控制四处激射，若是打到我们这边来，我们必死无疑。”
<br/>　　数百武者中，也有几个半步洞玄的武者，一眼就看出了情况，顿时大惊失色。
<br/>　　果然，他们的话音才刚落，数十把黑刃不受控制地打向他们这边。
<br/>　　“糟糕，快退。”
<br/>　　数百武者顿时乱作一团。
<br/>　　这些黑刃，足以将他们数百人秒杀。
<br/>　　恰在此时，一道流光快速从柳絮城中飞出。
<br/>　　“凝。”流光之中，一道人影大喝一声。
<br/>　　一道坚实的灵气屏障凭空而现，挡在数百武者面前。
<br/>　　嘭嘭嘭....数十黑刃打在屏障上，让屏障一阵颤动，幸好屏障没有破碎掉。
<br/>　　“呼...”这时，数百武者才松了口气。
<br/>　　当他们看到来人时，立刻恭敬地行了一礼，“多谢柳城主救命之恩。”
