　　　　由于这个山洞的关系，涂小安只是显化了十米的长度，可就算如此，也足够的可怕，能摧毁一切。
<br/>　　猛兽般的猫精极尽疯狂，妖邪的眸光将它的理智全部占据，看到这条十米显化的巨蟒，不但不吃惊，反而通灵慑厉的扑了上来。
<br/>　　这一扑，风卷残云，腾涌跳荡间，如狂狼暴虎，飙奔而下，朝着涂小安迅猛袭来。
<br/>　　涂小安蛇眸一扫，山洞中的温度顿时下降三分，一条紫金色的蛇尾骤然翘起，对着从天而降的猫精就是轻轻一拍。
<br/>　　轰...
<br/>　　猫精直接被拍入了地表之中。
<br/>　　可仅仅下一秒，猫精腾跃而起，浑身毛发炸立，根根尖锐，它辗转腾挪，灵巧的身姿倏忽在左，倏忽在右，想出其不意的将青白獠牙咬在这条巨蟒的七寸之上。
<br/>　　涂小安的蛇眸在收缩，紫金色蛇尾再次出手，看似对着空气拍打，但转眼，倒飞出去的却是猫精。
<br/>　　这次，涂小安不等猫精起来，蛇尾席卷，狂风暴雨一般的抽打过去。
<br/>　　一鞭子一鞭子的抽在猫精的身上，将它重新抽入地表中，然后再给卷出来，再次的暴抽。
<br/>　　不得不说这猫精疯狂之后，好像不惧痛疼，生命力还异常的旺盛，只闻洞中它的哀嚎声冲天而起。
<br/>　　但是戾气却更为的可怕。
<br/>　　特殊的蛇类气息愈演愈烈，当最后一鞭子将猫精抽在岩石墙壁上的时候。
<br/>　　它既然还能稳稳的落地，一落地，体型再次暴涨一倍，红目妖艳，血光漫天。
<br/>　　“嗷...”
<br/>　　猫精一声狂啸，乍听之下，凄若游魂，厉似鬼魅，声声入耳，震得涂小安心头不由得一颤。
<br/>　　“不好，它的气息不断的在变强”
<br/>　　恍然间，涂小安甚至以为这猫精要突破三阶生灵了。
<br/>　　就在这时，一道红光，惊鸿一瞥的从猫精的左眼中骤射出来。
<br/>　　一个眨眼的工夫都没有，涂小安低头一看，难以置信，自己刀枪不入，无比坚硬的蛇鳞片破了，出现一个伤口，鲜血淋淋。
<br/>　　眸光化为实质？
<br/>　　涂小安吃惊不已，这是非常可怕的攻击手段，让人根本来不及反应。
<br/>　　这叫什么，典型的用眼睛来杀人。
<br/>　　要知道涂小安的蛇躯防御力是何等的惊人，就算是导弹轰来，他都有底气不退，用身体硬吃一枚导弹。
<br/>　　可现在，一道眸光，就让他的蛇躯出现伤口。
<br/>　　“好呀，这蛇王石还真是一件宝物”
<br/>　　毫无疑问，这猫精越来越强，是体内的蛇王石在作妖...
<br/>　　涂小安不敢在大意了，必须全力以赴才行。
<br/>　　这猫精本身等级就达到二十九级，虽然战斗力说不上特别特别的强悍，但是此时已经今非昔比。
<br/>　　逼的涂小安不得不认真对待。
<br/>　　倏然，猫精身形一动宛如奔雷，带着滔天威势冲身而至，速度之快，涂小安从未见过。
<br/>　　紧接着，他感觉自己的身体一痛，剧痛无比，猫精的锋利的青白獠牙已经咬在了他的蛇皮上，蛇鳞片防御力骤破。
<br/>　　“该死，痛死我了”
<br/>　　涂小安无名火涌了起来，火焰异象直接显化，炙热无比，空气中出现滋滋的声音，有股白烟在沸腾。
<br/>　　猫精甚至想一口将这条十米巨蟒咬成两截，但不可思议的是，这条巨蟒周身熊熊烈火的燃烧了起来。
<br/>　　逼的它不得不松开獠牙后退，因为根本难以承受这种温度。
<br/>　　“雷霆之怒，三倍...”
<br/>　　涂小安体内的雷电之力顷刻间去了三分之二，眼前出现了一片惊涛骇浪的雷海，将猛兽般的猫精顿时笼罩其中。
<br/>　　雷海下，还有不断的惊雷在翻腾，将山洞照耀的如同白昼。
<br/>　　涂小安十米的体型不断的缩小，缩小到了一公分，蛇躯上有好几处的伤口，鲜血还在不断的流。
<br/>　　没想到，到了他这种阶段，还能受这样的伤。
<br/>　　涂小安意念一动，一下子出现五颗蛇莓，一口吞了进去。
<br/>　　这蛇莓有霸道的能量，可以让体型暴涨，但是也可以迅速的恢复自己的伤势。
<br/>　　果不其然，蛇莓一吃，涂小安蛇躯上的伤口凝固，不在流血，随后，伤口已肉眼可见的速度痊愈。
<br/>　　这时，雷海消失，猛兽般的猫精身上一片焦黑，但却还屹立不倒。
<br/>　　“海棠，你还不清醒”
<br/>　　涂小安都生怕打死它，所以才不想动用霸道的火焰之力攻击。
<br/>　　怕火焰力量过猛，一下子将这只猫精给火化了。
<br/>　　通体焦黑的猫精掀开了眼，没有红光妖艳，涂小安一喜，清醒了。
<br/>　　可还没等高兴，那双眼眸一下子殷红起来，特殊的蛇类气息再次席卷出来。
<br/>　　“怎么可能？”
<br/>　　自己三倍的雷霆之力，威力何等之大，几千个人类站在一起，都能给一起劈死。
<br/>　　猫精还能有战斗力？
<br/>　　涂小安一阵蹙眉，看来只能动用火焰之力了。
<br/>　　烧死了活该，怨不得涂小安了。
<br/>　　可忽然，猫精的身体开始发光发亮了起来，涂小安蛇眸收缩，既然诡异的看到了它体内一颗光芒四射的石头。
<br/>　　这是蛇王石？
<br/>　　此时特殊的蛇类气息对涂小安产生了一种无比吸引的感觉。
<br/>　　像是吸力一样的，让涂小安想接近，那种感觉，就好像要投入了妈妈的怀抱。
<br/>　　“这蛇王石在召唤自己？”
<br/>　　涂小安一声笃定，旋即冷哼一声，劳资才是蛇王，要召唤，也是劳资召唤你。
<br/>　　霎时间，只见涂小安的身上爆涌出一股无比恐怖的蛇王气息，君临天下，睥睨一切。
<br/>　　整个山洞的温度下降到了极点，与此同时，两股无比强大的蛇类气息相互的碰撞到了一起去。
<br/>　　空气中激烈的震动，空间在放肆的波动扭曲，层层爆响不绝于耳。
<br/>　　“给我镇压”
<br/>　　涂小安的蛇王气息充满了威压，他就是要压过蛇王石散发出来的特殊蛇类气息，只要压制住了，就不定是谁召唤谁了。
<br/>　　两股至强的蛇类气息好比在对比力量，看看谁的力量够大。
<br/>　　到是猫精仿佛是陷入了沉睡，身体没了掌控权。
<br/>　　这种蛇王石，邪恶又霸道，被别的生物吞进去，不见得是好事，除非你能完全的控制跟掌握。
<br/>　　显然，猫精做不到这点，被体内的蛇王石牵着鼻子走。
