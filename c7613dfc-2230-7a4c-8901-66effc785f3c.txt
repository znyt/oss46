　　“六成弟子，应该都死在了三大宗门之手。”
<br/>　　寒月漓眼神之中也露出几分伤感，这剑道门弟子，许多都是她曾经寒雪阁的弟子，在这一次劫难之中，都有不少人陨落了。
<br/>　　楚风眠目光也是冰冷无比，这一次三大宗门围攻剑道门，对剑道门的损失可以说是无比严重的。
<br/>　　这还是楚风眠敢来及时，要是楚风眠再晚一天到来，这个损失还会更大。
<br/>　　不过唯一庆幸的是，现在剑道门这一方小世界，还没有受到任何的损失，只是被打破了门户。
<br/>　　这剑道门门户，还有上面的护山大阵，都是依靠剑道门之内的灵泉所建立的，虽然是被打破，但是也足以缓缓修复。
<br/>　　而且楚风眠本来回到剑道门，也是有着计划重新布置护山大阵的打算。
<br/>　　这一次前去归墟天之旅，也让楚风眠得到了不少的宝物，更加重要的，是现在楚风眠的实力，已经是跟上一次离去的时候，截然不同了。
<br/>　　正好借助这个机会，重新布置剑道门的防御。
<br/>　　楚风眠看了一眼剑道门破碎的门户，身形一闪，突然飞了过去，一股灵力在楚风眠的掌心之中凝聚。
<br/>　　随着楚风眠掌心灵力猛然催动，这破碎的剑道门门户，开始缓缓凝聚起来。
<br/>　　同时这重塑的剑道门门户之上，一道道纹路，形成了坚固无比的阵法，笼罩在剑道门门户之上。
<br/>　　楚风眠心神一动，那一把把位于剑道门之中的黑玄灵剑也飞了出来，这些黑玄灵剑之上，都出现了丝丝裂痕。
<br/>　　在之前这黑玄剑阵，为了抵挡盘石等人的联手轰击，也已经是将力量催动到了极致，最终被打出了裂痕。
<br/>　　楚风眠看了一眼那些黑玄灵剑，打开空戒，从其中足足数百件天级灵器，飞了出来。
<br/>　　这些天级灵器，都是楚风眠从归墟天之中的收获，楚风眠在归墟天之中，不知道是斩杀了多少的强者。
<br/>　　他们空戒之中，拥有这大量的天级灵器，这些天级灵器，楚风眠留着也没有任何的用，倒不如来凝炼这黑玄剑阵。
<br/>　　一件件天级灵器飞了出来，在楚风眠的灵力催动之下，突然在空中破碎，化为了一道道光芒。
<br/>　　这一道道光芒，顿时涌入到了黑玄剑阵之中，跟其中的每一把黑玄灵剑，凝聚在一起。
<br/>　　一千把黑玄灵剑在，在这一刻同时蜕变，这些黑玄剑阵，本就是勉勉强强达到了天级灵器的层次。
<br/>　　这一次凝炼，更是让这些黑玄剑阵的威力，又是足足提升了数倍，整个黑玄剑阵的威力，要比起之前，强横了十倍不止。
<br/>　　黑玄剑阵，再度笼罩在了剑道门的门户之上。
<br/>　　这一次，哪怕是盘石等人的联手轰击，数个时辰，都不可能打破这黑玄剑阵。
<br/>　　现在这黑玄剑阵的威力，如果全然催动起来，九劫古帝深陷其中，都要陨落。
<br/>　　不止是将黑玄剑阵，楚风眠更是将护山大阵，也给重新凝炼了一遍。
<br/>　　随着楚风眠实力的提升，他所凝炼布置出来的阵法，也要比起之前剑道门之中的阵法强横的多。
<br/>　　瞬息之间，现在这剑道门的防御，便是要比起之前要强横无数倍了，哪怕是十几名九劫古帝同时来攻打剑道门，想要再度打破这剑道门门户，都无比困难了。
<br/>　　同时楚风眠也是毫不吝啬，从他的身躯之中，两道光芒飞了出来，正是那法理之门，还有太古铜门。
<br/>　　这两道门户，本是镇压在楚风眠的身躯之中，为稳固楚风眠的身躯之用。
<br/>　　但是随着楚风眠身躯越发的强横，这两道门户，对楚风眠也没有太大的意义了，楚风眠便是要将他们留在了这剑道门之中。
<br/>　　这两道门户，可不是那种普通的天级灵器，都是古老无比的灵器，在天级灵器之中，品级都接近极品了。
<br/>　　这两道门户被楚风眠布置出来，镇压在剑道门之中，整个剑道门的防御，都是提升了一个等级。
<br/>　　楚风眠看到布置出的这一切，眼神之中才露出了几分笑意。
<br/>　　这样的防御，这剑道门才配得上是楚风眠的老巢。
<br/>　　这一下子哪怕是楚风眠一时半会赶不回来，七大宗门的人想要来攻打剑道门，一时半会也攻打不进来了。
<br/>　　楚风眠将一切布置完后，身形这才化为了一道遁光离开，下一刻进入到了云羽天宫之中。
<br/>　　在云羽天宫之中，如今只有云老，寒月漓，洛汐三人。
<br/>　　“这一次三大宗门来攻打剑道门，只是一次试探。”
<br/>　　楚风眠来到云羽天宫之中，看向这三人，不由的眉头一皱。
<br/>　　在刚刚布置剑道门阵法的时候，楚风眠在脑海之中，也在观察这那屠人间的记忆。
<br/>　　可看得越多，楚风眠也就感觉越是棘手了，这七大宗门，这一次攻打剑道门，其实只是一次试探。
<br/>　　七大宗门如今内部，也不是铁板一块，对楚风眠的身份，也其实只是在猜测。
<br/>　　所以这一次，不管是三大宗门来攻打剑道门，还是天枢殿，荒古门来围攻楚风眠，其实都没有动用全力。
<br/>　　起码古月峡谷，药王谷这两方势力，都还没有出手，而且其他出手的几方势力，也没有动用全力。
<br/>　　在他们的计划之中，如果说剑道门是如此弱小，不堪一击的话，那他们也就会这样，顺势灭了剑道门跟楚风眠。
<br/>　　可如果说剑道门跟楚风眠的实力，超出了他们的试探的话，只怕是下一次，等待楚风眠的，就是真正七大宗门的联手围攻了。
<br/>　　在这七大宗门之中，曾经万年之前的老古董，可不止东宫绝，盘石这几个。
<br/>　　这些老古董的实力，才是无比的可怕，而六道魔门的崛起，也是因为九魔城之中，许多老古董，都是叛变了九魔城，加入到了六道魔门之中。
<br/>　　导致了现在六道魔门的实力，一下子强横了不少，继承了九魔城的底蕴，在加上曾经六道魔门玄妙的魔功，让现在六道魔门的势力，瞬间位于七大宗门之首了。
<br/>　　这一次楚风眠杀了攻打剑道门所有三大宗门的强者，让七大宗门知晓了现在楚风眠的实力之后。
<br/>　　下一次要的，只怕是七大宗门联手的攻打了。
