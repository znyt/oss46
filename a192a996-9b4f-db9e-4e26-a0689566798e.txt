        苏漫雪将童筱颖送到大门口，又转身小跑着去迎接傅夜沉。
<br/>
<br/>        “夜沉，你回来了！我很想你噢！你也很想我，对不对？”苏漫雪大步流星地跑到傅夜沉的跟前，一把挽住了他的臂弯，亲昵又热情地将他的手臂在自己身上蹭了蹭。
<br/>
<br/>        童筱颖此时已经进了屋，傅夜沉也毫不客气地推开苏漫雪，二话不说，径直往前走去。
<br/>
<br/>        苏漫雪瘪了瘪嘴，悻悻地跟在了傅夜沉的身后。
<br/>
<br/>        晚餐的时候，刘婶按照傅夜沉的吩咐做了一桌子的菜，重点是，全是童筱颖爱吃的菜。
<br/>
<br/>        看着这一桌子丰盛的菜，苏漫雪心里便气得直发痒。
<br/>
<br/>        童筱颖也在这桌菜上看出了点猫腻，就比如说那坛瓦罐乌鸡松茸汤，就是苏漫雪最不爱吃的。
<br/>
<br/>        而此时，苏漫雪却拿起汤勺，替自己舀了两勺乌鸡松茸汤放在了自己的小汤碗中，嗲声爹气地跟傅夜沉撒娇道：“夜沉，这乌鸡和松茸都是野生空运过来的，为了给我补身子，难为你费心了。”
<br/>
<br/>        傅夜沉不吭声，只是自己默默地夹菜吃。
<br/>
<br/>        童筱颖端着饭碗，微笑道：“大嫂，你真幸福。”
<br/>
<br/>        “那当然，你大哥，真的很宠我噢！”苏漫雪得意洋洋地扬了扬眉梢。
<br/>
<br/>        刘婶看在眼里，想反驳，却又不好意思道出实情。
<br/>
<br/>        这一桌子菜，可是大少爷特意吩咐她为筱颖小姐做的，跟她苏漫雪没点关系。
<br/>
<br/>        但是这话若是说穿了，只会让筱颖小姐和大少爷觉得面子上挂不住。
<br/>
<br/>        童筱颖其实心里只是觉得，就他们四个人吃饭，而刘婶做这一大桌子的菜，实在是太浪费了。
<br/>
<br/>        晚饭过后，童筱颖给刘婶帮忙收拾碗筷，却被刘婶给拒绝了。
<br/>
<br/>        苏漫雪见傅夜沉已经上楼回房去了后，立马就像变天一样变了张脸色，不禁教训起童筱颖来：“筱颖，你这都是当二少奶奶的人了，这些家务活就让佣人去做，你就不用插手了。不然，夜沉花钱雇她，岂不是白花了。”
<br/>
<br/>        一听苏漫雪这话，刘婶心里就很不是滋味。
<br/>
<br/>        童筱颖不顾苏漫雪的嚼舌根，继续帮着刘婶，并安慰道：“刘婶，没关系的。我看你这两天腰不大好，今晚又这么多碗筷，我帮你收拾到洗碗机里去。不打紧，你别在意。”
<br/>
<br/>        “噢，刘婶，你腰不好使啊！那明天，我让夜沉再雇一个保姆进来帮你好了。毕竟，我家里这人口多了嘛！你的家务量就大了些。”苏漫雪说话带刺，拐着弯，间接地说她童筱颖现在是“寄人篱下”。
<br/>
<br/>        童筱颖心里十分清楚苏漫雪这话的另外一层意思，只是默默地忍气吞声，毕竟，她现在确实是寄人篱下。
<br/>
<br/>        刘婶气恼地白了苏漫雪一眼，接着转脸温和地对童筱颖说道：“大小姐，待会我给您做点小点心吃，是我家乡那边特好吃的一种小点心。”
<br/>
<br/>        “好呀！还好刚刚，我没把肚子吃得饱饱的！我现在又有口福了！”童筱颖微笑着应道，然后一边帮刘婶收拾着碗筷，一边和刘婶谈笑风生地进了厨房。
<br/>
<br/>        两人完全把苏漫雪晾到了一旁，让她自己在那儿唱独角戏。
<br/>
<br/>        苏漫雪愤愤地瞪着童筱颖的背影，不禁在心底诅咒童筱颖，活该她童筱颖没有当豪门大少奶奶的命。
<br/>
<br/>        在一个下人面前，还无事献殷勤，连主仆身份都不分。
<br/>
<br/>        眼里心里嘲讽完童筱颖，苏漫雪扶着楼梯扶手，抬头挺胸又扭着翘臀，往楼上走去。
<br/>
<br/>        苏漫雪上楼后在房间里未见到傅夜沉，随后又找去了健身房，直到在射击房门口，她正巧撞见傅夜沉手握气枪，威风凛凛，连续十发，有九发正中活动靶的红色中心。
<br/>
<br/>        好帅！
<br/>
<br/>        这到底是当过兵的人，枪法不错。
<br/>
<br/>        苏漫雪惊赞地拍着手，朝傅夜沉走了过去。
<br/>
<br/>        傅夜沉见苏漫雪过来，放下了手中的气枪，摘掉耳朵上的耳麦后，冷淡地问道：“你进来做什么？”
<br/>
<br/>        “夜沉，人家想你了嘛！”苏漫雪咬了咬红唇，学着童筱颖的音调，撒娇道。
<br/>
<br/>        “用你原来的声音跟我说话。”傅夜沉不悦地皱了下眉。
<br/>
<br/>        苏漫雪却不以为然地回答道：“我的声音，本来就是这样的啊！”
<br/>
<br/>        “苏漫雪，不要试图挑战我的耐性！”傅夜沉呵斥道。
<br/>
<br/>        苏漫雪不禁得意地笑了笑：“夜沉，别生气嘛！你这若是把我惹不开心了，万一哪天我说漏了嘴，在弟弟昼景面前说，其实夜沉喜欢的人是筱颖了，该如何是好？！”
<br/>
<br/>        “呵，你可以去试试，看看阿景会不会信你的话。”傅夜沉嗤之以鼻，淡定自若。
<br/>
<br/>        苏漫雪靠近傅夜沉，踮起脚尖，抬起双手勾住傅夜沉的脖子，将自己那对大大的美胸，直接撞到傅夜沉的胸膛上，还撞出了隐隐的波动。
<br/>
<br/>        “夜沉，你怎么就这么死脑筋呢？反正筱颖已经是你的弟妹了，你和筱颖是不可能的了。而我，我不仅拥有一张和筱颖一模一样的脸，而且，我的身材也比筱颖好。你为何就不试试我呢？”苏漫雪目送秋波，柳眉微挑，嗲声嗲气地问。
<br/>
<br/>        傅夜沉剑眉微蹙，刚想推开苏漫雪的时候，门外传来了童筱颖的声音。
<br/>
<br/>        “大哥，大嫂，你们两在射击房里吗？刘婶做了饭后茶点，说是大哥喜欢的，让我端上来给大哥尝尝。”童筱颖试探性地问，她刚刚找了几个房间，都未找着傅夜沉和苏漫雪。
<br/>
<br/>        苏漫雪见状，忽然“嘶啦”一声，撕开了自己的裙摆和上衣的领口，还拉下了胸衣的吊带，露出大片美胸。
<br/>
<br/>        “你——”傅夜沉的话刚说到第一个字。
<br/>
<br/>        童筱颖听闻屋内有动静，便端着茶点走了进去。
<br/>
<br/>        苏漫雪趁机往傅夜沉身上跳，双脚勾住了傅夜沉的健硕腰肢，双手紧紧地抱着他的脖子不放，故意叫出那种很享受的“啊啊”声，姿势十分暧昧。
<br/>
<br/>        童筱颖看到这一幕，“唰”地一下，脸红了个通透，连忙将手里的盘子放到一旁矮柜上，并用手捂住了眼睛，讷讷地说道：“那个……我把茶点放这里了……你们继续……”
<br/>
<br/>        从小到大，除了在电视上看过男女主角的吻戏，现实生活中，她还是第一次撞见这种真人版的香艳画面。
<br/>
<br/>        童筱颖说完，转身便跑开了。
<br/>
<br/>        刚刚那个一瞬间，傅夜沉看着童筱颖的样子有点儿哭笑不得，她娇俏的脸蛋浮泛着红润，那样子很是可爱。
<br/>
<br/>        再回眸看着面前这个女人，除了一身骚气，就是一肚子的心机。
<br/>
<br/>        “她已经走了，你也可以从我身上下来了。”傅夜沉淡定道。
<br/>
<br/>        苏漫雪一脸震惊地瞪着傅夜沉。
<br/>
<br/>        她的身子可是跟他贴得紧得不能再紧了，然而，他腹下却对她没有任何反应！
<br/>
<br/>        苏漫雪气恼地咬了咬下唇，“嗖”地一下，放开了傅夜沉并从他的身上跳了下来。
<br/>
<br/>        “你不去追她吗？不去跟她解释？”苏漫雪反问。
<br/>
<br/>        傅夜沉却冷哼：“我跟她有什么好解释的，反倒是你，你若是饥渴难耐，我可以考虑找几个男人过来伺候你！”
<br/>
<br/>        “你——”苏漫雪顿时气得咬牙切齿。
<br/>
<br/>        他这是在给她下马威。
<br/>
<br/>        “滚！”傅夜沉眸光一敛，冷冰冰地呵斥。
<br/>
<br/>        苏漫雪自讨没趣，却在洋洋得意地碎碎念叨道：“看样子，你这是破罐子破摔了！呵，不接受我也没关系，反正你也得不到筱颖。比起我的一厢情愿，我看你才是最痛苦的那个吧！自己喜欢的女人，成了自己的弟妹！我看着就觉得爽了！每天你对着筱颖，爱而不能说，膈应都快被膈应死了吧？”
