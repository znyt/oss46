　　九劫古帝施展出自爆的威力，足以令天地震动，毁灭一方大陆，都不成问题。
<br/>　　碧螺老祖的每一滴精血，如今都在疯狂的燃烧膨胀着，要跟楚风眠鱼死网破，同归于尽。
<br/>　　“想要跟我同归于尽？你还没有这个资格，你的精血，都会成为我的补品！统统给我吞噬！镇！”
<br/>　　一个镇字一出。
<br/>　　只见一尊古老的铜鼎，从楚风眠的身上飞了出来。
<br/>　　直直的镇压在了这碧螺老祖的身躯之上。
<br/>　　这一尊铜鼎，有着九座龙头，每一座都惟妙惟肖，如同真正的龙头一般。
<br/>　　九龙神苍鼎
<br/>　　在天下百鼎之中，位列第十一的神鼎。
<br/>　　这九龙神苍鼎，是一尊药鼎，但更是有着镇压一切的力量。
<br/>　　这一件镇压神器，被楚风眠施展出来，镇压在了这碧螺老祖的身上。
<br/>　　轰！
<br/>　　这碧螺老祖刚刚还在疯狂燃烧的精血，在这镇压之下，其中的疯狂被全部熄灭。
<br/>　　随即吞天秘术转动，万千精血，统统被吞噬进去，化为了精粹的灵力。
<br/>　　顿时楚风眠便是感觉得到，一股精粹的灵力涌入到了楚风眠的身躯之中。
<br/>　　“不！”
<br/>　　碧螺老祖疯狂的咆哮着。
<br/>　　但是他如今在疯狂的咆哮，也没有任何的意义，吞天秘术碾碎了他的意识，随即这些精血，便是不在反抗，都被尽数吞噬，化为了楚风眠力量。
<br/>　　“好，一尊九劫古帝的精血，果然精粹！”
<br/>　　九劫古帝的精血，比起许多古圣药，古圣丹之中蕴含的灵力，都要精粹，楚风眠利用吞天秘术所吞噬。
<br/>　　他身上的气息，更是又强横了几分。
<br/>　　这是这碧螺老祖的空戒！
<br/>　　楚风眠看到这吞天秘术之中，碧螺老祖的一切都被吞噬，只有一枚空戒留了下来，他打开一看，琳琅满目的无数宝物，都要晃花了楚风眠的眼睛。
<br/>　　古圣丹，古圣药，天级灵器，无数材料，甚至是许多已经是消失在这个时代之中的诸天奇物，都在这空戒之中，堆积的满满的。
<br/>　　单单是其中的古圣丹，就有数百枚之多，简直是一大笔的财富，不亚于一座宝藏。
<br/>　　这碧螺老祖，看来是在九域之中不知道是得到过多少的好处，积累下来的无数宝藏，都在空戒之中。
<br/>　　如今这碧螺老祖死在了楚风眠的手中，他的一切积攒，也都便宜了楚风眠。
<br/>　　“好，在多杀几个碧螺老祖，我就可以准备在将一些远古战龙精血，转化为龙皇精血！”
<br/>　　楚风眠心中无比兴奋。
<br/>　　远古战龙精血，想要转化成龙皇精血，需要大量圣丹，圣药才行。
<br/>　　像是那一瓶武祖精血，也只是仅仅只能转化其中的一千滴，想要转化一滴都需要耗费不少的古圣药才行。
<br/>　　要是让楚风眠一个人去积攒，简直是杯水车薪，现在楚风眠杀了这碧螺老祖，得到了他的空戒，几乎是相当于发了一笔横财。
<br/>　　比起楚风眠自己慢慢的去寻找宝藏，要快得多。
<br/>　　而且不止是这空戒之中所蕴含的宝藏，那碧螺老祖的精血，也都是无比精粹的灵力。
<br/>　　吞天秘术之中，这碧螺老祖的精血，已经是被彻底的炼化干净，再也没有碧螺老祖的气息，都没有血腥味，而是成为了最为精粹的灵力结晶。
<br/>　　化为了一枚枚如同灵石一般的存在，但是要比起圣品灵石的灵力，还要精粹。
<br/>　　楚风眠一把将其中的所有灵力结晶，全部吞噬，同时运转起血脉之力，远古战龙虚影，浮现在楚风眠的背后。
<br/>　　现在楚风眠想要提升实力，转化龙皇精血，正是需要大量的灵力。
<br/>　　这源源不断的灵力被楚风眠吞噬之后，便是纷纷被一滴滴远古战龙精血所吞噬。
<br/>　　旋即一滴金色的血液飞了出来，其中蕴含无数法阵，正是一滴龙皇精血。
<br/>　　第一滴，第二滴，第三滴！
<br/>　　在充足的灵力之下，一滴滴龙皇精血被凝聚了出来，在凝聚了五十滴龙皇精血后，这碧螺老祖精血的力量，都被完全吞噬。
<br/>　　五十滴，虽然不多，但是也足以提升楚风眠不少的实力了。

<br/>　　五行刀祖看向楚风眠，急忙的开口道。
<br/>　　“只要你放了我，今日的事我可以帮你瞒住，不会被三大势力的人知道。”“你觉得我会相信你吗？”
<br/>　　楚风眠看向五行刀祖的表情，如同看向一个白痴。
<br/>　　别说这五行刀祖的话是真是假，楚风眠本来也没有放过敌人的打算。
<br/>　　与他为敌者，只有一条路，便是死。

<br/>　　楚风眠大手一挥，吞天秘术的漩涡顿时笼罩在了五行刀祖的身上。
